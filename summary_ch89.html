第8-9章 醇酚醚 梳理

第 31 页

第九章：醇、酮、醛、酯

(1) 结构

醇的结构

根据与羟基相连的碳原子的级数不同，醇分为：

  • 伯醇 (1°)：−CH2OH — 1个碳取代
  • 仲醇 (2°)：>CHOH — 2个碳取代
  • 叔醇 (3°)：>C(OH) — 3个碳取代

注意：C−OH 不稳定（烯醇式不稳定，会重排为羰基形式）。

(2) 命名

1) 含−OH的最长碳链作为主链，C−OH 不在母体的碳链上。

2) 伯醇和仲醇属于同一类R和H组的顺反异构。

3) 如: $\text{CH}_3\text{CH(OH)CH}_3$ — 2-丙醇

多种官能团同时存在，按优先级确定主官能团名字。

醇的命名：
类别IUPAC名结构
伯醇1-丙醇CH3CH2CH2OH
仲醇2-丙醇CH3CH(OH)CH3
叔醇2-甲基-2-丙醇(CH3)3COH

例：HO−CH2−CH2−OH — 1,2-乙二醇（甲基→乙二醇）

(3) 物理性质

  • 甲/乙/丙/丁醇与水混溶，1-戊醇以上微溶，有氢键
  • 沸点比相应烃高，氢键：沸点高
  • 低级醇与 MgCl2、CaCl2 可以保持以致形成加合物，如 MgCl2·6H2O
  • CaCl2·4C2H5OH 等，一般使用 CaO 干燥

高键极性 / 构型

$sp^3$ 杂化的醇碳

$\ce{R-OH}$ 中氧为 $sp^3$ 杂化，构成四面体构型，接近 $109.5°$ 键角。

$\ce{C-O}$ 键极性较强，构型保留有助于判断反应机理（$\ce{S_N1}$ 消旋 vs $\ce{S_N2}$ 反转）。

(4) 化学性质概述

C−O 键和 O−H 键均可断裂：

  • O−H 断裂：→ 酸性反应、酯化反应、氧化反应
  • C−O 断裂：→ 取代反应（被卤素取代）、脱水反应

1. 醇的酸性 (pKa)

醇的酸性

$\ce{H-OH}$：$pK_a \approx 15.7$

$\ce{R-OH}$：$pK_a \approx 16\text{--}18$

醇的酸性比水弱（碳为供电子基团，使 O−H 键中的氢更难解离）。

  • 上面的碳越多越难解离
  • 酸性顺序：$\ce{CH_3OH} > 1° > 2° > 3°$

$\{X_a \in [7, \Delta]\}$ — 取代基电子效应影响

酸性排序 / 配对

酸性顺序：甲醇 > 伯醇 > 仲醇 > 叔醇

注意：醇 + NaOH 不反应（醇酸性太弱）；醇 + Na(金属钠) → 放出 H2 + RONa

$\ce{2ROH + 2Na -> 2RONa + H2 ^}$

$\ce{RONa + H2O -> ROH + NaOH}$

$\ce{HC#CH + ROH ->[\text{催化}] HC#C- + NH3 -> RO^-}$

$pK_a$：$\ce{H2O}$ 15.7   $\ce{ROH}$ 16~18   $\ce{HC#CH}$ 25   $\ce{NH3}$ ~35

第 32 页

2. 醇的取代反应：卤代反应

与 HX 反应机理

总反应：$\ce{R-O-H ->[\ce{H+}] R-\overset{+}{O}H_2 ->[X^-] R-X + H2O}$

先质子化再离去，醇的取代总是先质子化（因为 OH− 是差的离去基）

1) HX 与 R−OH

  • 伯醇：$\ce{S_N2}$ 机理
  • 仲醇：视条件，$\ce{S_N1}$ 或 $\ce{S_N2}$（伯仲交叉/边界 $\ce{S_N2}$ 为主）
  • 叔醇：$\ce{S_N1}$ 机理 → 生成碳正离子中间体 $(R_3C^+)$

HX 活性：$\ce{HI > HBr > HCl}$（与酸性/亲核性相关）

醇的活性：叔醇 > 仲醇 > 伯醇 > 甲醇（碳正离子稳定性）

SN1 机理（叔醇）

$\ce{(CH3)3C-OH + H+ -> (CH3)3C-OH2+}$ （质子化）

$\ce{(CH3)3C-OH2+ -> (CH3)3C+ + H2O}$ （生成碳正离子，决速步）

$\ce{(CH3)3C+ + X- -> (CH3)3C-X}$ （亲核进攻）

碳正离子重排：碳正离子中间体可能发生重排（如甲基迁移、氢迁移），生成更稳定的碳正离子。
SN2 机理（伯醇）

$\ce{R-CH2-OH + H+ -> R-CH2-OH2+}$

$\ce{X- + R-CH2-OH2+ -> R-CH2-X + H2O}$ （背面进攻，一步）

构型：Walden 反转

Lucas 试验（区分伯、仲、叔醇）

Lucas 试剂：浓 HCl + 无水 ZnCl2
醇反应速度现象
叔醇 (3°ROH)立即反应立即浑浊/分层
仲醇 (2°ROH)约 5 分钟缓慢浑浊
伯醇 (1°ROH)长时间不反应需加热才浑浊
Lucas 试验只适用于 6 个碳以下的可溶性醇。
浑浊/分层原因：生成的卤代烃不溶于水。

Lucas 试剂的原理

$\ce{ROH + ZnCl2 -> R-O-ZnCl + HCl}$

$\ce{ZnCl2}$ 作为 Lewis 酸配合醇的氧，使其成为更好的离去基团。

活性差别取决于碳正离子稳定性：$3° > 2° > 1°$

地位及适用性：C ≤ 6 的可溶醇，$5° > 2° > 1°$（反应速度排列），烯丙醇和苄醇速度与叔醇相当。

PX3 与 ROH 反应

与 PBr3 / PCl3 反应 [SN2]

$\ce{R-OH + PBr3 -> R-Br + P(OH)3}$ 或 $\ce{R-X + P(OH)3}$

$\ce{3R-CH(OH)CH3 + PX3 -> 3R-CH(X)CH3 + P(OH)3}$

注意：反应中产生强酸性环境，但碳骨架不发生重排（因为经 SN2 机理，无碳正离子中间体），因此与 HX 反应生成构型翻转（Walden 反转）。
拓展：PBr3 的反应机理

$\ce{3ROH + PBr3 -> (RO)3P + 3HBr -> 3RBr + P(OH)3}$

反应中生成亚磷酸酯中间体，然后被 HBr 亲核取代（SN2），因此无碳正离子中间体，无重排。

3) SOCl2 与 R−OH 反应

亚硫酰氯法

$\ce{R-OH + SOCl2 -> R-Cl + SO2 ^ + HCl ^}$

副产物 SO2 和 HCl 都是气体，有利于正反应进行（无需分离）。

$\ce{R-Cl + SO2 ^ + HCl ^}$

构型转化（SOCl2）：
  • 不加吡啶（pyridine）：构型保持（SNi，分子内前面亲核取代）
  • 加吡啶：构型翻转（SN2，吡啶捕获中间体 Cl− 进行背面进攻）

4) 磺酰氯酯 与 R−OH (SN2)

对甲苯磺酰氯（TsCl）与醇反应：

$\ce{R-OH + TsCl ->[\text{吡啶}] R-OTs + HCl}$

OTs 是优良的离去基团，后续可进行 SN2 反应（构型翻转）。

第二步置换与卤素的反应：SN2 → 翻转构型。

三、无机酸酯 与 R−OH

硫酸酯

硫酸酯的生成

$\ce{CH3OH + HOSO3H -> CH3OSO3H + H2O}$ （硫酸氢甲酯）

$\ce{2CH3OSO3H ->[\text{减压蒸馏}] CH3OSO2OCH3 + H2O}$ （硫酸二甲酯）

硫酸二甲酯是强烈的甲基化试剂，剧毒！

硝酸酯与亚硝酸酯

$\ce{C2H5OH + HNO3 -> C2H5ONO2 + H2O}$（硝酸乙酯）

甘油与浓硝酸 → 三硝酸甘油酯（硝化甘油，炸药）

$\ce{CH2(OH)-CH(OH)-CH2(OH) + 3HNO3 -> CH2(ONO2)-CH(ONO2)-CH2(ONO2) + 3H2O}$

2. 脱水反应

分子内脱水（消除反应）

醇的脱水

$\ce{R-CH2-CHR'-OH ->[\ce{H2SO4}, \Delta] R-CH=CHR' + H2O}$

遵循 Zaitsev 规则（扎依采夫规则）：生成取代基最多的烯烃（即最稳定的烯烃）为主要产物。

脱水难易顺序：叔醇 > 仲醇 > 伯醇（碳正离子稳定性：$3° > 2° > 1°$）

伯醇脱水条件苛刻，如乙醇需要 170°C 浓 H2SO4 才能得到乙烯。
• 脱水与酯的关系：双分子间缩合得到醚 ⇐ 低温条件（如乙醇 140°C → 乙醚，170°C → 乙烯）

分子间脱水

$\ce{2ROH ->[\ce{H2SO4}, 140°C] R-O-R + H2O}$（生成醚）

频哪醇重排

频哪醇重排

邻二醇脱水得到酮（1,2-二醇 → 酮 + 水）：

$\ce{R2C(OH)-C(OH)R2 ->[\ce{H+}] R2C(OH)-CR2+ ->[\text{迁移}] R3C-CO-R}$ + H2O

机理要点：先质子化一个 −OH 脱水形成碳正离子 → 邻位基团迁移（1,2-迁移） → 生成更稳定的碳正离子（氧稳定化） → 去质子得到酮。

3. 酯化反应

酯化反应 (Fischer 酯化)

$\ce{RCOOH + R'OH <=>[\ce{H+}][\Delta] RCOOR' + H2O}$

1° 醇：酰氧键断裂（醇的 O−H 断裂，酸的 C−OH 断裂）

3° 醇：烷氧键断裂（醇的 C−O 断裂）

可逆反应，需移除水或过量一种反应物以推动平衡。

4. 氧化反应

铬酸氧化（$\ce{CrO3}$ / Jones 试剂）为经典的醇氧化方法：

$\ce{RCH2OH ->[\ce{CrO3}/\ce{H2SO4}] RCHO ->[\text{继续氧化}] RCOOH}$

$\ce{R2CHOH ->[\ce{CrO3}/\ce{H2SO4}] R2C=O}$

1° 醇 → 醛 → 羧酸；2° 醇 → 酮；3° 醇 → 不反应（无 α-H 可脱）

Oppenauer 氧化

Oppenauer 氧化

用 $\ce{Al(O\text{-}iPr)_3}$ + 丙酮（$\ce{(CH3)2CO}$）进行氧化，条件温和，醛酮不被继续氧化。

适用于不能耐受强氧化条件的底物（铬酸法较为激烈）。

5. 碘仿反应（卤仿反应）

碘仿反应

含有 $\ce{CH3CO-}$（甲基酮）或 $\ce{CH3CH(OH)-}$（α-甲基仲醇）结构的化合物可发生碘仿反应：

$\ce{RCOCH3 + 3I2 + 4NaOH -> RCOONa + CHI3 v + 3NaI + 3H2O}$

加 $\ce{NaOH}$、$\ce{I2}$，观察是否产生黄色沉淀（$\ce{CHI3}$），判断反应是否发生。

6. 通过酯化反应确定醇的构型

通过酰基化测定醇的构型

方法：

  • 利用已知构型的酰基化试剂与未知构型的醇反应
  • 四步法：① 原醇 → ② 酰化(构型保持) → ③ 水解(构型翻转) → ④ 比旋光度
  • 目的在于确定碳上基团的绝对构型（R/S 构型）

注释：酯化只断 O−H 键（构型保持），而水解则根据机理可能翻转或保持。

第 34 页

迁移基团和离去基团

迁移基团与离去基团的关系

迁移之离子在碳骨架上有两个不同的位置，发生不同反应时，迁移基团/离去基团在反式位置时，速率快。

反应位置关系：迁移基团与离去基团呈反式(anti-periplanar)关系时最有利。

频哪醇重排的应用

1) 环状化合物的频哪醇重排 → 扩环反应：

环戊二醇 →[H+] 环己酮（5元环 → 6元环扩环）

碳正离子中间体中，邻位碳发生1,2-迁移导致扩环。

2) 半频哪醇重排

半频哪醇重排

$\ce{R-C(-OH)-C(=O)-R' ->[\ce{-H2O}] }$ 酮类产物

即一个 −OH、一个 C=O 的 β-羟基酮，脱水时发生迁移重排。

3) 二苯基乙二醇

$\ce{C6H5C(OH)=C(OH)C6H5 ->[\text{重排}] }$ 二苯甲酮类产物

$\ce{RCH(OH)CH=O ->[\ce{H2O}] RCH(OH)CH(OH)2}$

五、醛的制备

1) 醇的脱氢/氧化

$\ce{RCH2OH ->[\ce{-2[H]}][\text{Cu}, 300°C] RCHO + H2}$（催化脱氢）

工业制法：甲醇 → 甲醛，用铜催化 300°C 脱氢。

2) 烯烃的臭氧化

$\ce{R-CH=CH-R' ->[\ce{O3}][\ce{Zn/H2O}] RCHO + R'CHO}$

3) Rosenmund 还原

Rosenmund 还原

$\ce{RCOCl + H2 ->[\ce{Pd/BaSO4}][\text{喹啉}] RCHO + HCl}$

$\ce{Pd/BaSO4}$ 催化剂活性被喹啉毒化降低，只还原酰氯到醛，不会过度还原到醇。（$\ce{S_N2}$ 反应）

利用聚合态的 Cr(VI) 的选择性

$\ce{RCH2OH ->[\ce{PCC}] RCHO}$（PCC = 氯铬酸吡啶鎓，温和的选择性氧化剂）

伯醇氧化到醛停止，不继续到酸。

4) 格氏反应

$\ce{R-MgBr + HCHO -> RCH2OMgBr ->[\ce{H3O+}] RCH2OH}$（甲醛 → 伯醇）

$\ce{R-MgBr + R'CHO -> RR'CHOMgBr ->[\ce{H3O+}] RR'CHOH}$（醛 → 仲醇）

酚 结构与性质

1) 结构

酚的结构

苯环直接连羟基，氧的孤对电子与苯环 $p$-$\pi$ 共轭（$\pi$ 离域）。

C−O 键有部分双键性质，键长比醇短。

2) 命名

苯酚（phenol）为母体。

优先级：$\ce{-COOH > -SO3H > -COOR > -COX > -CONH2 > -CHO > -CO- > -OH > -NH2 > -OR > -X > -NO2}$

(3) 物理性质

  • 弱酸性固体/低熔点
  • 溶于热水及有机溶剂（乙醇、乙醚）
  • 氢键：3维

(4) 化学性质

酚的化学性质分两大类：

  • 酸性反应（酚羟基上的 O−H 键断裂）
  • 芳环反应（酚羟基是强活化基，邻/对位定位）
第 35 页

1. 酸性

酚的酸性

$\ce{Ar-OH <=> Ar-O^- + H+}$

酚的酸性比碳酸弱但比醇强（酚氧负离子与苯环共轭稳定化）。

影响酸性的因素：

  • 吸电子基（如 −NO2）增强酸性 → 稳定酚氧负离子
  • 供电子基（如 −CH3、−OCH3）减弱酸性
  • 一般来说酚的酸性: 受吸电子基团（特别是硝基）影响明显
酸性强弱比较：

$\ce{PhOH + CH3O^- -> PhO^- + CH3OH}$ ✓（酚酸性 > 醇酸性）

$\ce{PhOH + HCO3^- -> PhO^- + CO2 ^ + H2O}$ ✗（酚酸性 < 碳酸 $\ce{H2CO3}$）

$\ce{PhCOOH + HCO3^- -> PhCOO^- + CO2 + H2O}$ ✗（这里 PhCOOH 是苯甲酸不是酚）

即酚能与 NaOH 反应，但不能与 NaHCO3 反应 → 可借此区分酚和羧酸。

2. 与 FeCl3 显色

FeCl3 显色反应

凡含有烯醇式结构（$\ce{>C=C(-OH)}$）或酚羟基的化合物，均可与 $\ce{FeCl3}$ 显色。

$\ce{6ArOH + FeCl3 -> [Fe(OAr)6]^{3-} + 6H+ + 6Cl-}$

蓝紫色（酚的特征显色反应）

3. 芳环上的反应 — 亲电取代（邻/对位定位）

(1) 卤代反应

酚的 −OH 是强活化基团，亲电取代活性非常高：

$\ce{C6H5OH + 3Br2(aq) -> C6H2Br3OH v + 3HBr}$（2,4,6-三溴苯酚白色沉淀）

此反应可用于酚的定性检测和定量分析。

如需要单取代，需在非极性溶剂（如 CS2）中用等量 Br2 反应 → 对溴苯酚为主。

(2) 合成反应

$\ce{ArOH ->[\ce{HNO3}/\ce{H2SO4}] }$ 邻/对位硝基酚

稀 HNO3：混合物（邻/对）；若用 NaNO2 + 稀 H2SO4 则可选择性。

冰浴 → 亚硝基酚 → 继续得到对硝基酚。

(3) 磺化

低温（常温）→ 邻位磺化产物（动力学控制产物）

高温（100°C）→ 对位磺化产物（热力学控制产物）

(4) F-C 反应 (Friedel-Crafts)

Friedel-Crafts 反应 → 对位优先，限次取代。

$\ce{PhOH + CH3Cl ->[\ce{AlCl3}] }$ 对甲基苯酚（对甲酚）

叔丁基化：2,6-二叔丁基-4-甲基苯酚（BHT，抗氧化剂）

(5) 氧化反应

4-叔丁基-2-二叔丁基苯酚（BHT）抗氧化剂的应用

(6) 柯尔贝 (Kolbe) 反应

Kolbe-Schmitt 反应

$\ce{C6H5ONa + CO2 ->[\text{125-150°C}][\text{0.5MPa}] }$ 邻羟基苯甲酸钠 $\ce{->[\ce{H+}]}$ 水杨酸

酚钠在加压、加温下与 CO2 反应 → 邻位羧基化（水杨酸）。

水杨酸（邻羟基苯甲酸）可进一步合成阿司匹林。
第 36 页

4. 酚的制备方法

1. 异丙苯氧化（工业法）

异丙苯法制苯酚

$\ce{C6H5CH(CH3)2 ->[\ce{O2}][\text{催化}] C6H5C(CH3)2OOH ->[\ce{H+}] C6H5OH + CH3COCH3}$

异丙苯（cumene）先氧化为过氧化异丙苯，再在酸催化下分解为苯酚和丙酮。

工业上最重要的苯酚制备方法，同时得到丙酮副产物。

2. Ar−Cl 水解法

$\ce{Ar-Cl ->[\ce{NaOH(aq)}][\text{高温高压}] Ar-OH}$

芳环上的 C−Cl 键很稳定（p-π 共轭），需要高温高压条件下水解。

3. 磺酸钠碱熔法

碱熔法

$\ce{Ar-SO3H ->[\ce{NaOH}] Ar-SO3Na ->[\ce{NaOH}][\text{300-350°C 碱熔}] Ar-ONa ->[\ce{H+}] Ar-OH}$

磺化 → 磺酸钠 → 碱熔（与 NaOH 固体在 300–350°C 熔融） → 酚钠 → 酸化得酚。

4. 重氮盐水解

重氮盐法

$\ce{Ar-NH2 ->[\ce{NaNO2}/\ce{H2SO4}][\text{0-5°C}] Ar-N2+ HSO4- ->[\ce{H2O}][\Delta] Ar-OH + N2 ^ + H2SO4}$

芳胺重氮化后水解，适合制备特定取代酚。

重氮盐的制备需在低温（0–5°C）进行以防止分解，然后加热水解。
补充：含卤芳烃的 Br 可先用 NaNO2、H2SO4 进行重氮化处理：

$\ce{ArNH2 + NaNO2 + H2SO4 -> ArN2+HSO4- ->[\ce{H3PO2}] ArH}$（还原脱氨基）

$\ce{ArN2+HSO4- + Br- -> ArBr + N2}$（Sandmeyer 反应）

$\ce{ArN2+HSO4- + H2O -> ArOH + N2}$（水解制酚）

醚的结构与性质

1) 结构

醚的结构

通式：$\ce{R-O-R'}$，氧为 $sp^3$ 杂化，碳氧碳键角 ~112°。

2) 命名与分类

$\ce{R-O-R'}$：$R = R'$ 时为简单醚（对称醚）；例：$\ce{CH3OCH3}$（二甲醚/甲醚）

$R \ne R'$ 时为混合醚（不对称醚）；例：$\ce{CH3OC2H5}$（甲乙醚）

  • 甲醚（二甲醚）
  • 混合醚：大基团为母体，小基团为取代基（"甲氧基"、"乙氧基"）
  • 环醚：环氧乙烷（oxirane），四氢呋喃（THF），1,4-二氧六环等

3) 对称醚的鉴别

对比结构：醚 $\ce{C_{n}H_{2n+2}O}$ 与 醇 $\ce{C_{n}H_{2n+2}O}$ 互为同分异构体。

醚对碱、对稀酸（常温下）均不反应 → 化学性质"惰性"（相对醇而言）。

7) 物理性质

  • 分子间无氢键，沸点比醇低
  • DI 型分子间力：沸点与 $\ce{H2O}$ 交叉溶解性下降
  • 化学惰性：常用作反应溶剂

(四) 醚的化学性质

1. 氧鎓盐的生成 -- 溶液质酸

醚的碱性 -- 形成氧鎓盐

醚中的氧原子有孤对电子，可以接受质子形成氧鎓盐：

$$R\text{-}O\text{-}R + H^+ \longrightarrow R\text{-}\overset{+}{O}H\text{-}R$$

因此醚可以溶解在浓酸中：

$$R\text{-}\overset{+}{O}H\text{-}R \longrightarrow R\text{-}F + R'\text{-}OH \quad (\text{或 } R' \text{-}X)$$

2. 与路易斯酸形成络合物

醚可以与路易斯酸（如 $BF_3$、$AlCl_3$）形成络合物：

$$R_2O + BF_3 \longrightarrow R_2O \cdot BF_3$$

$BF_3 \cdot Et_2O$（三氟化硼乙醚络合物）是常用的路易斯酸催化剂。

3. 醚键的断裂（需要较强的酸，如 HX，X = I, Br）

醚键断裂机理
$$CH_3\text{-}O\text{-}CH_2CH_3 + HI \longrightarrow CH_3\text{-}\overset{+}{O}H\text{-}CH_2CH_3 + I^-$$ $$\longrightarrow CH_3I + C_2H_5OH$$

或反过来：

$$C_2H_5I + CH_3OH$$
反应规律：在含不同烃基的醚中，$I^-$ 攻击 $C\text{-}O$ 键时遵循以下规律：
• 当为 SN2 机理时，$I^-$ 攻击较小的烃基（位阻小）；断裂后生成较小烃基的碘代烃
• 当含有叔碳时，倾向于 SN1 机理
示例 1（SN2）：
$CH_3\text{-}O\text{-}CH_3 + HI \longrightarrow CH_3I + CH_3OH$
（$I^-$ 攻击甲基）
示例 2（芳基醚）：
$\varphi\text{-}O\text{-}CH_3 + HI \longrightarrow \varphi\text{-}OH + CH_3I$
（芳基碳 $sp^2$ 不能发生 SN2，所以 $I^-$ 只能攻击甲基）
叔丁基醚在与 HX 反应时倾向于在叔碳位上断裂（SN1 机理）。
混合醚断裂举例
$$CH_3\text{-}O\text{-}C(CH_3)_3 + H^+ \longrightarrow CH_3\text{-}\overset{+}{O}H\text{-}C(CH_3)_3$$

然后：叔碳正离子离去（SN1），生成 $(CH_3)_3C^+$，再与 $X^-$ 结合。

$$\longrightarrow CH_3OH + (CH_3)_3CX$$

4. 过氧化物的生成

$$CH_3\text{-}O\text{-}C_2H_5 \xrightarrow{O_2} CH_3\text{-}O\text{-}\overset{O\text{-}O\text{-}H}{|}CH\text{-}CH_3 \quad (\text{含 }\color{red}{-O\text{-}O-}\text{ 键})$$

醚中 $\alpha$-H 容易被氧化生成过氧化物（含 $-O-O-$ 键）。

检测方法：加入 $FeCl_3$ 和 $KSCN$，若有过氧化物：
• 防止：加入抗氧剂（对苯二酚）
• 检测：加入还原剂（$Fe^{2+}$）
$$Fe^{2+} \xrightarrow{H_2O_2} Fe^{3+} \xrightarrow{SCN^-} \text{血红色}$$

5. 环氧化物的性质

a) 合成

环氧化物的制备（Williamson 合成法）

由邻卤醇在碱的作用下关环：

$$CH_2\text{-}CH_2 \xrightarrow[\text{(1) } Cl_2/H_2O]{\text{(2) } NaOH} \quad \text{环氧乙烷}$$

卤醇法（$Cl_2/H_2O$）：

$$CH_2\!=\!CH_2 \xrightarrow{Cl_2, H_2O} ClCH_2CH_2OH \xrightarrow{NaOH} \text{环氧乙烷}$$

其他方法：

  • 过氧酸氧化：$RCH\!=\!CHR' \xrightarrow{mCPBA} \text{环氧化物}$ （mCPBA = 间氯过氧苯甲酸）
  • 直接工业法：$CH_2\!=\!CH_2 \xrightarrow{O_2, Ag} \text{环氧乙烷}$
HOCl 加成法：先 $Cl_2/H_2O$ 得卤醇，再用 $NaOH$ 闭环（Markovnikov 方向添加 OH，anti 加成）。

b) 开环反应

环氧化物开环反应总结

酸催化开环

酸催化条件下，亲核试剂进攻取代基较多的碳（类似 SN1，Markovnikov 方向）：

$$\text{环氧化物} \xrightarrow{H^+/ROH} \text{反式-二醇或醚醇}$$
  • $H^+/H_2O$ → 反式 1,2-二醇
  • $H^+/ROH$ → 醚醇
  • $H^+/HX$ → 卤醇

碱催化开环 / 亲核试剂开环

碱性条件下（SN2），亲核试剂进攻取代基较少的碳（anti-Markovnikov，位阻控制）：

$$\text{环氧丙烷} \xrightarrow{CH_3O^-} CH_3O\text{-}CH_2CH(OH)CH_3$$

常见亲核试剂开环：

  • $Grignard$ 试剂（$RMgX$）→ 增长碳链
  • $CH_3COO^-$ → 酯
  • $HCN$ → 氰基化合物
  • $NH_3$ → 氨基醇
  • $LiAlH_4$ → 醇（还原开环）
开环方向判断口诀：
• 酸催化：亲核试剂攻击取代基多的碳 → Markovnikov
• 碱催化：亲核试剂攻击取代基少的碳 → anti-Markovnikov（SN2 位阻控制）

c) 酸催化开环（续：环氧化物的开环选择性与立体化学）

醚键断裂的 SN2 与 SN1 竞争

环氧化合物的酸催化开环，既有 SN2 也有 SN1 的特征：

$$CH_3\text{-}CH\text{-}CH_2 \xrightarrow{H^+} CH_3\text{-}\overset{+}{C}H\text{-}CH_2 \longrightarrow \text{开环产物}$$

酸催化开环先质子化再开环，然后 $Nu$ 进攻碳正离子（有 SN1 特征）。

醚键断裂的构型保持与翻转

醚键断裂的机理：取决于底物结构：

  • 若断裂处为伯碳或仲碳：SN2 机理 → 构型翻转（Walden翻转）
  • 若断裂处为叔碳：SN1 机理 → 构型难以预测（外消旋化）
如果进攻的亲核试剂是带负电的：按照 SN2 处理；若亲核试剂较弱（中性），酸催化，按照碳正离子稳定性判断位置。
记忆：如果是酸催化的醚的裂解，本质是 均裂 + 异裂。如果速度快的醚键的裂解是有利的，按照碳正离子稳定性判断。
2-甲基环氧丙烷的开环：
  • 酸催化：$H^+/CH_3OH$ → 亲核试剂攻击取代基多的碳（2号碳）→ 2-甲氧基-1-丙醇
  • 碱催化：$CH_3O^-$ → 攻击取代基少的碳（1号碳）→ 1-甲氧基-2-丙醇

d) 碱性条件下的开环反应（SN2 反应）

碱性开环的区域选择性
$$\text{环氧丙烷} + CH_3O^- \xrightarrow{S_N2} CH_3O\text{-}CH_2\text{-}CHOH\text{-}CH_3$$

SN2 机理：亲核试剂进攻位阻小的碳（取代基少的碳）。

构型翻转：产物构型与原料相反（Walden 翻转）。

(5) 聚醚反应

环氧乙烷的聚合
$$HOCH_2CH_2OH + n \underset{\text{环氧乙烷}}{\overset{O}{\frown}} \xrightarrow{} HO(CH_2CH_2O)_nCH_2CH_2OH$$

环氧乙烷开环聚合：生成聚乙二醇（PEG）

$$CH_3(CH_2)_nCH_2OH + n \underset{\text{环氧乙烷}}{\overset{O}{\frown}} \xrightarrow{OH^-} CH_3(CH_2)_nCH_2(OCH_2CH_2)_nOH$$

此即非离子型表面活性剂的合成方法。

缩水甘油缩合：
$$HOCH_2 - \underset{\text{环氧}}{\overset{O}{\frown}} - CH_2 \longrightarrow \text{聚合物}$$

酸催化SN1 特征取代基多的碳构型保持/混合

Claisen 重排示例（烯丙基芳基醚的重排）

脂肪族 Claisen 重排

烯丙基乙烯基醚在加热条件下发生 [3,3]-$\sigma$ 迁移：

$$\text{烯丙基乙烯基醚} \xrightarrow{200\,°C} \text{不饱和醛/酮}$$

产物为 $\gamma,\delta$-不饱和羰基化合物。

芳香族 Claisen 重排

苯基烯丙基醚的 Claisen 重排分两步：

  1. 邻位 Claisen 重排：烯丙基先迁移到邻位（生成环己二烯酮中间体），然后互变异构得到邻烯丙基酚。
  2. 对位 Claisen 重排：若邻位被占据，烯丙基经 Cope 重排 迁移到对位。
$$\text{苯基烯丙基醚} \xrightarrow{200\,°C} \text{邻-烯丙基酚} \quad (\text{邻位})$$ $$\text{邻位被封} \xrightarrow{200\,°C} \text{对-烯丙基酚} \quad (\text{对位})$$
注意：Claisen 重排后的产物中，烯丙基发生了翻转（即原来连接氧的碳变成了远端碳）。

应用

Claisen 重排的合成价值：可以用于在芳环上特定位置引入碳链（邻位或对位烯丙基化）。这是一种 $C\text{-}C$ 键形成的重要方法。

(3) Williamson 醚合成法

Williamson 合成
$$RX + NaOR' \longrightarrow ROR' + NaX$$

SN2 反应，卤代烃与醇钠反应生成醚。

副反应：

  • $RX$ 与 $NaOR'$：$RX + R'O^-Na^+ \longrightarrow R\text{-}O\text{-}R' + NaX$（期望产物）
  • 若 $RX$ 为叔卤代烃：以消除反应（E2）为主，而非取代。
Williamson 合成的适用范围：
• 卤代烃最好是伯卤代烃（SN2 易进行，副反应少）
• 仲卤代烃也可以但产率降低
• 叔卤代烃不能用（主要发生 E2 消除）
合成不对称醚的策略：
$$CH_3CH_2\text{-}O\text{-}CH=CH_2 \quad (\text{乙基乙烯基醚})$$ 可以由 $CH_3CH_2ONa + CH_2\!=\!CHCl$ 得到，或反过来 $CH_3CH_2Cl + CH_2\!=\!CHONa$ 也可。
但要注意选择不会发生消除的组合。
醚的其他制备方法
  • $CH_3CH_2ONa + CH_3I \longrightarrow CH_3CH_2OCH_3 + NaI$ （甲基碘是好的 SN2 底物）
  • $CH_3CH=CHCH_2Br + C_2H_5ONa \longrightarrow CH_3CH=CHCH_2OC_2H_5$ （烯丙基溴也可以用）

冠醚

冠醚 (Crown Ethers)
  1. 命名：X-冠-Y（其中 X 为主环中的原子总数，Y 为氧原子数）
  2. 性质：根据空穴大小不同，对金属离子的结合有选择性：
    • 18-冠-6：对 $K^+$ 离子有最合作用
    • 15-冠-5：对 $Na^+$ 离子有最合作用
  3. 应用：相转移催化剂（Phase Transfer Catalysis, PTC）
    • 内圈氧原子亲水，外圈碳氢链亲有机溶剂
冠醚作为相转移催化剂的应用：
$$CH_2\!=\!CH\text{-}CH\!=\!CH_2 \xrightarrow{KMnO_4, \text{水}} \text{-}COOH$$ $$CH_3(CH_2)_4CH_2Br + KF \xrightarrow{\text{18-冠-6}} CH_3(CH_2)_4CH_2F + KBr$$ 冠醚将 $K^+$ 包裹，使 $F^-$（裸露的 $F^-$）在有机溶剂 $CH_3CN$ 中成为强亲核试剂。

  • $PCC$ 也可以

酚的酸性

酚的酸性比醇强，但比碳酸弱：

$$PhOH + Na_2CO_3 \longrightarrow NaOH + PhONa$$

即酚可以与 $Na_2CO_3$（碳酸钠）反应，但不能与 $NaHCO_3$ 反应。（注意：酚不能使石蕊变色。）

(2) FeCl3 显色

$FeCl_3$ 与酚反应显蓝紫色。

这是酚类化合物的特征鉴定反应。

(3) 芳环上的亲电取代反应

酚的芳环反应

酚羟基（$-OH$）是强活化基团和邻对位定位基：

卤代反应

  • 溴代：
    • $PhOH + 3Br_2 \longrightarrow 2,4,6\text{-}三溴酚 \downarrow + 3HBr$（白色沉淀，定量反应）
    • 这是酚的鉴定反应之一

硝化反应

  • 稀 $HNO_3$（室温）→ 邻硝基酚 + 对硝基酚
  • 浓 $HNO_3/H_2SO_4$ → 2,4,6-三硝基酚（苦味酸）

磺化反应

  • 低温（RT）→ 邻位产物为主
  • 高温 → 对位产物为主（热力学控制）

Friedel-Crafts 反应

  • 酚 + $RX \xrightarrow{AlCl_3}$ 烷基化（但要注意多烷基化）
  • 酚 + $RCOCl \xrightarrow{AlCl_3}$ 酰基化

Kolbe-Schmitt 反应

  • $PhONa + CO_2 \xrightarrow{高温高压} \text{水杨酸（邻羟基苯甲酸）}$

Reimer-Tiemann 反应

  • $PhOH + CHCl_3 \xrightarrow{NaOH} \text{邻羟基苯甲醛（水杨醛）}$

与甲醛缩合

  • 酚 + $HCHO$ → 酚醛树脂（高分子材料）

(4) 氧化反应

酚容易被氧化：

$$\text{PhOH} \xrightarrow{[O]} \text{对苯醌} \quad (p\text{-benzoquinone})$$

这就是酚在空气中变色发黑的原因。

(5) 酯化（酚入酰氯反应）

酚的酯化

酚 + 酰氯（或酸酐）：

$$PhOH + R\text{-}CH\!=\!CH_2 \xrightarrow{酸催化} PhO\text{-}CHR\text{-}CH_3 \quad (C^0\text{烷基化为主})$$

酚 + 酰氯：

$$PhOH + RCOCl \longrightarrow PhOCOR + HCl$$

Fries 重排：酚酯在 $AlCl_3$ 催化下重排为羟基芳酮：

  • 低温 → 对位产物
  • 高温 → 邻位产物

酚的制备

酚的制备方法
  1. 异丙苯氧化法（工业法）： $$\text{异丙苯} \xrightarrow{O_2} \text{异丙苯过氧化氢} \xrightarrow{H^+} PhOH + (CH_3)_2CO$$
  2. 磺化碱熔法： $$PhH \xrightarrow{H_2SO_4} PhSO_3H \xrightarrow{NaOH, 熔融} PhONa \xrightarrow{H^+} PhOH$$
  3. 卤代芳烃水解： $$PhCl \xrightarrow{NaOH, 高温高压} PhONa \xrightarrow{H^+} PhOH$$
  4. 重氮盐水解： $$ArN_2^+ \xrightarrow{H_2O, \Delta} ArOH + N_2 + H^+$$

$SO_3H$ 到 $OH$ 的转化：

$$R\text{-}SO_3H \xrightarrow{NaOH, 熔融} R\text{-}OH$$ $$R\text{-}SO_3H \xrightarrow{PCl_5} R\text{-}Cl \xrightarrow{} \cdots$$

醚键断裂反应

(2) 醚键的制备

醚的制备方法总结

a) 醇脱水：

$$2\;R\text{-}OH \xrightarrow{H_2SO_4} R\text{-}O\text{-}R + H_2O$$

适用于伯醇制对称醚（仅限 $1°$ 醇，$2°/3°$ 醇易消除）。

b) Williamson 合成：

$$R\text{-}OM \xrightarrow{\Delta} R\text{-}O\text{-}R' \quad [\text{回扣}]$$
制备方法
$$RX + NaOR' \longrightarrow ROR' + NaX$$

（仲卤代烃会产生更多的 E2 副反应消除产物。）

过氧化物

醚在空气中放置容易生成过氧化物：

$$R\text{-}O\text{-}CHR' + O_2 \longrightarrow R\text{-}O\text{-}C(OOH)\text{-}R'$$

过氧化物受热易爆炸，使用前需要检测。

$$R\text{-}OH \longrightarrow R\text{-}\overset{O}{\frown} \longrightarrow R\text{-}(OCH_2CH_2)_n\text{-}OH$$

醇与环氧乙烷反应，可以制备聚乙二醇醚。

(3) Claisen 重排 & Cope 重排

Claisen 重排

烯丙基芳基醚的 [3,3]-$\sigma$ 迁移：

$$\text{烯丙基苯基醚} \xrightarrow{加热} \text{邻-烯丙基酚}$$

若邻位被封闭，则迁移到对位。

Cope 重排

1,5-己二烯型底物的 [3,3]-$\sigma$ 迁移：

$$\text{1,5-己二烯} \xrightarrow{\Delta} \text{异构化的1,5-己二烯}$$

Williamson 合成补充

Williamson 合成中的 SN2 vs E2
$$R\text{-}OM \xrightarrow{\Delta / RX} R\text{-}O\text{-}R' \quad [\text{回扣}]$$ $$RX + NaOR' \longrightarrow ROR' + NaX$$
SN2 兼有 SN1 的投射：
• 伯卤代烃：以 SN2 为主 → 好的 Williamson 底物
• 仲卤代烃：SN2 和 E2 竞争
• 叔卤代烃：以 E2 为主 → 不适合 Williamson 合成
