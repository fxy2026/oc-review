第一章 有机化学与有机化合物

一、有机化学与有机化合物

定义

有机化合物 = 含碳化合物；有机物的特性：

  • 结构：碳骨架
  • 性质：易燃、熔点低、非电解质
  • 反应：反应慢、副反应多、可逆反应
研究发展

经验 → 理论（结构决定性质）

概念说明
同分异构分子式相同，结构不同
$\sigma$ 键 / $\pi$ 键头碰头 / 肩并肩
$-\!\!\!$（单键）自由旋转
$=\!\!\!$（双键）不能自由旋转
价线式价键表示法，简写为折线式

二、有机物中的化学键

共价键类型
  • 离子键：电子转移
  • 共价键
    • 极性共价键
    • 非极性共价键
  • 配位键
轨道杂化理论
杂化类型几何构型键角实例
$sp^3$四面体$109.5°$$\text{CH}_4$
$sp^2$平面三角形$120°$$\text{CH}_2\!=\!\text{CH}_2$
$sp$直线形$180°$$\text{HC}\!\equiv\!\text{CH}$
分子轨道理论（HOMO / LUMO）

成键轨道 & 反键轨道：

  • HOMO（最高占据分子轨道）
  • LUMO（最低未占据分子轨道）
键的属性
  • 键能：共价键的强度
  • 键长：两原子核间距
  • 键角
  • 偶极矩：偏移量
诱导效应 vs 共轭效应
  • 诱导效应：沿 $\sigma$ 键传递，随距离递减 → 影响酸碱性
  • 共轭效应：$p$-$\pi$ 共轭、$\pi$-$\pi$ 共轭
共价键的断裂方式
  • 均裂（同裂）：A:B → A· + B· → 产生自由基（链锁反应/代替反应）
  • 异裂（不对称断裂）：A:B → A⁺ + B⁻ → 产生离子（碳正离子、碳负离子）
共价键断裂的反应类型
  • 均裂 → 自由基反应：引发 → 增长 → 终止
  • 异裂 → 碳正离子/碳负离子 → 亲电/亲核反应

三、有机酸碱理论

Brønsted 酸碱理论

Acid：质子（$\text{H}^+$）的给予者；Base：质子的接受者

共轭酸碱对

Lewis 酸碱理论
  • Lewis 酸（Acid）：接受电子对 / 缺电子物种，如 $\text{BF}_3$, $\text{AlCl}_3$
  • Lewis 碱（Base）：提供电子对，如 $\text{NH}_3$, $\text{H}_2\text{O}$
酸碱强度与 $\text{p}K_\text{a}$ 的关系：$\text{p}K_\text{a}$ 越小，酸性越强；$\text{p}K_\text{a} + \text{p}K_\text{b} = 14$
比较酸碱性时应注意：
  • 把一些 有机反应归纳为酸碱反应 来理解
  • $\text{p}K_\text{a}$ 值越小 → 酸性越强、其共轭碱越弱
  • 可以判断反应方向：强酸 + 强碱 → 弱酸 + 弱碱
