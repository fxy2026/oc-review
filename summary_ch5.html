第 13 页

第五章 芳烃

一、结构

苯的结构特点
  1. 碳原子均为 $sp^2$ 杂化，构成正六边形骨架
  2. 所有原子共平面，键角均为 $120°$
  3. 碳碳键长 $0.140\;\text{nm}$（介于 $\text{C-C}$ 单键 $0.154\;\text{nm}$ 和 $\text{C=C}$ 双键 $0.134\;\text{nm}$ 之间），是平均化的"离域键"
  4. 闭合共轭体系：6个 $p$ 轨道垂直于平面、肩并肩重叠，形成环形大 $\pi$ 键（$\pi_6^6$）
化学键长

苯的 $\text{C-C}$ 键长 $= 0.140\;\text{nm}$，介于单键与双键之间，说明苯环的键是完全均等的离域键。

二、同分异构

  • 一般叫法：烷基苯结构
  • 取代时产生异构体：苯环上的取代基位置不同（邻位 ortho、间位 meta、对位 para）
  • 各组同分异构体个数与取代基种类和数目有关

三、命名

芳烃命名规则
  1. 苯环作母体的命名：以苯为母体，烷基为取代基，如 $\text{1,2-}$二甲苯（邻二甲苯）
  2. 烷基作母体的命名：当侧链较长或含有官能团时，苯基($\text{C}_6\text{H}_5\text{-}$)作为取代基，命名为"苯基某烷"
  3. 常用俗名：甲苯(toluene)、二甲苯(xylene)、苯乙烯(styrene)、萘(naphthalene)等
  4. 多取代时，编号遵循最低系列原则，编号之和最小
二甲苯有三种异构体：邻二甲苯（$o\text{-}$xylene）、间二甲苯（$m\text{-}$xylene）、对二甲苯（$p\text{-}$xylene）。

四、物理性质

  • 芳烃通常无色液体，有特殊芳香气味
  • 不溶于水，溶于有机溶剂（乙醚、$\text{CCl}_4$、石油醚）
  • 一般芳烃的比水轻，$d < 1$
  • 沸点随相对分子量增大而升高；液态芳烃的沸点与相对分子质量有关，还与分子结构/对称性有关。邻/间/对二甲苯中，对二甲苯对称性最高，熔点最高；而沸点差距不大
  • $^{1}\text{H NMR}$ 化学位移：芳环质子 $\delta \approx 6.5\text{-}8.5\;\text{ppm}$（去屏蔽效应，低场移动）
芳环质子 NMR 化学位移范围：$\delta \approx 6.5\text{-}8.5\;\text{ppm}$（芳环质子因环电流去屏蔽效应，化学位移出现在低场区域）

五、化学反应

（一）一般的亲电取代反应

亲电取代反应概述

芳环上的亲电取代反应通式：

$$\text{ArH} + \text{E}^+ \longrightarrow \text{ArE} + \text{H}^+$$

其中 $\text{E}^+$ 为亲电试剂。苯环提供 $\pi$ 电子作为亲核体，与亲电试剂结合。

亲电取代机理
  1. $\pi$ 络合物（$\pi$ complex）：亲电试剂 $\text{E}^+$ 先与苯环 $\pi$ 电子形成松散的 $\pi$ 络合物（快速、可逆）
  2. $\sigma$ 络合物（慢，决速步）：$\text{E}^+$ 与苯环一个碳形成 $\sigma$ 键，生成 $\sigma$ 络合物（Wheland中间体/碳正离子中间体），芳香性被破坏
  3. 失去质子（快）：$\sigma$ 络合物失去一个 $\text{H}^+$，恢复芳香性，生成取代产物
$$\text{ArH} + \text{E}^+ \rightleftharpoons [\pi\text{络合物}] \xrightarrow{\text{慢}} [\text{ArHE}]^+(\sigma\text{络合物}) \xrightarrow{-\text{H}^+,\;\text{快}} \text{ArE}$$
$\sigma$ 络合物（又叫 Wheland 中间体、碳正离子中间体）：苯环芳香性被破坏，一个碳变为 $sp^3$ 杂化，正电荷在环上离域（可在邻位和对位碳上画出共振式）。$\pi$ 络合物 $\rightarrow$ $\sigma$ 络合物这一步是速控步骤。

（二）卤代反应

苯的卤代

苯与卤素在Lewis酸催化下发生亲电取代：

$$\text{C}_6\text{H}_6 + \text{X}_2 \xrightarrow{\text{FeX}_3} \text{C}_6\text{H}_5\text{X} + \text{HX} \quad (\text{X}=\text{Cl, Br})$$

催化剂：$\text{FeBr}_3$（溴代）、$\text{FeCl}_3$（氯代）、$\text{AlCl}_3$ 等Lewis酸

机理：

  1. $\text{X}_2 + \text{FeX}_3 \longrightarrow \text{X}^+\text{[FeX}_4]^-$（产生亲电试剂 $\text{X}^+$）
  2. $\text{X}^+$ 进攻苯环 $\rightarrow$ $\sigma$络合物
  3. $\sigma$络合物失去 $\text{H}^+$ $\rightarrow$ 取代产物 + $\text{HX}$ + $\text{FeX}_3$（催化剂再生）

（三）硝化反应

苯的硝化
$$\text{C}_6\text{H}_6 + \text{HO-NO}_2 \xrightarrow{\text{H}_2\text{SO}_4} \text{C}_6\text{H}_5\text{NO}_2 + \text{H}_2\text{O}$$

亲电试剂：$\text{NO}_2^+$（硝酰正离子 / nitronium ion），由混酸产生：

$$\text{HNO}_3 + 2\text{H}_2\text{SO}_4 \longrightarrow \text{NO}_2^+ + \text{H}_3\text{O}^+ + 2\text{HSO}_4^-$$

$\text{HNO}_3$ 提供 $\text{NO}_2^+$，$\text{H}_2\text{SO}_4$ 起催化/脱水作用（质子化 $\text{HNO}_3$，促进 $\text{NO}_2^+$ 生成）。

要点归纳：反应是取代而非加成，因为取代产物恢复了芳香性（保留 $4n+2$ 个 $\pi$ 电子的环状共轭体系），在热力学上更加稳定。

说明：供电子取代基（如 $\text{-CH}_3$）使芳环活化，吸电子取代基（如 $\text{-Cl}$）使芳环钝化。

（二）卤代反应（续） — 甲苯的卤代

甲苯的氯代
$$\text{C}_6\text{H}_5\text{CH}_3 + \text{X}_2 \xrightarrow{\text{FeX}_3} \text{C}_6\text{H}_4(\text{CH}_3)\text{X} + \text{HX} \quad (\text{X}=\text{Cl, Br})$$

产物为邻位和对位取代产物（邻对位混合物），因为 $\text{-CH}_3$ 是邻对位定位基。

苯环上的取代 vs 侧链上的取代

甲苯氯代可在不同位置发生：

  • 苯环上取代（离子型，亲电取代）：需要Lewis酸催化（如 $\text{FeCl}_3$），得到氯甲苯（邻/对位）
  • 侧链取代（自由基取代）：光照或加热条件下进行，得到苄基氯 $\text{C}_6\text{H}_5\text{CH}_2\text{Cl}$ 等

卤代反应的选择性

环上取代：$\text{FeX}_3$ / Lewis酸催化 → 亲电取代（离子机理）
侧链取代：$h\nu$ / 加热 / 过氧化物 → 自由基取代

（四）磺化反应

磺化反应
$$\text{C}_6\text{H}_6 + \text{H}_2\text{SO}_4(\text{浓}) \xrightarrow{\Delta} \text{C}_6\text{H}_5\text{SO}_3\text{H} + \text{H}_2\text{O}$$

也可用发烟硫酸（含 $\text{SO}_3$）。亲电试剂为 $\text{SO}_3$ 或 $^+\text{SO}_3\text{H}$。

特点：磺化反应是可逆反应。

  • 稀酸加热可发生脱磺（逆反应）
  • 磺酸基团可作为占位基（先引入后脱去）
磺化的应用：磺酸基 $\text{-SO}_3\text{H}$ 可作为占位基团，利用其可逆性进行定位辅助合成。先磺化占住某一位置，在其它位置引入所需基团后，再脱磺去掉磺酸基。
磺化反应要点
  • 亲电试剂：$\text{SO}_3$（发烟 $\text{H}_2\text{SO}_4$ 中）或 $^+\text{SO}_3\text{H}$
  • 磺化反应可用稀硫酸/加热/水蒸气蒸馏使之逆转（脱磺反应）
硫酸与苯的反应补充：$\text{C}_6\text{H}_6 + \text{H}_2\text{SO}_4 \longrightarrow \text{C}_6\text{H}_5\text{SO}_3\text{H}$，伴有水生成。

（五）亲电取代活性

亲电取代活性
  • 芳烃亲电取代反应的机理——芳烃提供电子（充当亲核体），另给出电子的一方为亲电试剂
  • 芳环上连有给电子基团的芳烃反应速度较快；连有吸电子基团的芳烃反应速度较慢
  • 含有给电子基的苯环活化，反应更容易进行
  • 各类亲电试剂的活性顺序不同

（六）Friedel-Crafts 反应（傅-克反应）

超长线超纲重要反应！！！

(1) 烷基化反应

F-C 烷基化
$$\text{C}_6\text{H}_6 + \text{RCl} \xrightarrow{\text{AlCl}_3} \text{C}_6\text{H}_5\text{R} + \text{HCl}$$

催化剂：$\text{AlCl}_3$（Lewis酸）

机理：$\text{RCl} + \text{AlCl}_3 \longrightarrow \text{R}^+ + [\text{AlCl}_4]^-$

产生碳正离子 $\text{R}^+$ 作为亲电试剂进攻苯环。

烷基化试剂：卤代烃($\text{RX}$)、烯烃和醇均可作为 F-C 烷基化的试剂。
  • $\text{RCl} + \text{AlCl}_3 \longrightarrow \text{R}^+ + [\text{AlCl}_4]^-$（卤代烃路线）
  • 烯烃在酸催化下生成碳正离子
  • 醇在酸催化下脱水生成碳正离子

(1) 烷基化反应（续）

碳正离子来源

方法一：$\text{AlCl}_3 + \text{H}_2\text{O} \longrightarrow \text{Al(OH)}_3\cdot\text{(in)}^- + \text{H}^+$

$\text{CH}_3\text{CH=CH}_2 + \text{H}^+ \longrightarrow \text{CH}_3\text{CH}^+\text{CH}_3$（烯烃质子化得到碳正离子）

方法二：$\text{BF}_3 + \text{H}_2\text{O} \longrightarrow \text{BF}_3(\text{OH})^- + \text{H}^+$（BF$_3$ 也可作Lewis酸催化剂）

F-C 反应的问题

F-C 烷基化的四大问题（非常重要！考试重点）
(1) 多烷基化（连续烷基化）

烷基是给电子基团（供电子）→ 引入第一个烷基后芳环活化 → 更容易发生第二次、第三次烷基化 → 难以停留在一取代。

例：苯 + 过量 $\text{CH}_3\text{CH}_2\text{Cl}$ → 不仅得到乙苯，还得到二乙苯、三乙苯…

控制方法：使用大量过量的苯（$50:1$ 等比例），稀释烷基化试剂。

(2) 异构化（碳正离子重排）

碳正离子会发生重排：伯碳正离子 → 仲碳正离子 → 叔碳正离子（更稳定）

$$\text{CH}_3\text{CH}_2\text{CH}_2\text{-Cl} + \text{AlCl}_3 \longrightarrow \text{CH}_3\text{CH}_2\text{CH}_2^+ \xrightarrow{\text{重排}} \text{CH}_3\text{CH}^+\text{CH}_3$$

结果：正丙基苯和异丙基苯同时生成，甚至以异丙基苯为主要产物。

(3) 烷基迁移（歧化反应）

傅-克烷基化后的烷基在AlCl$_3$存在下可从一个苯环迁移到另一个苯环上，最终达到热力学平衡。

$\text{AlCl}_3$ 催化下，多烷基苯可与苯发生歧化反应。

(4) 对吸电子基团取代苯的限制

傅-克烷基化反应不适用于强吸电子取代基存在的芳环（如 $\text{-NO}_2$, $\text{-COR}$, $\text{-COOH}$ 等）。

这些基团使苯环钝化严重，活性过低，反应往往不能进行。

(2) 酰基化反应

F-C 酰基化
$$\text{C}_6\text{H}_6 + \text{RCOCl} \xrightarrow{\text{AlCl}_3} \text{C}_6\text{H}_5\text{COR} + \text{HCl}$$

产物为芳酮。

酰基正离子（$\text{RCO}^+$，也称酰基阳离子 acylium ion）是亲电试剂：

$$\text{RCOCl} + \text{AlCl}_3 \longrightarrow \text{RCO}^+ + [\text{AlCl}_4]^-$$ $$\text{AlCl}_3^- + \text{H}^+ = \text{AlCl}_3 + \text{HCl}$$
酰基阳离子的结构：$\text{R-C}^+=\text{O} \leftrightarrow \text{R-C}\equiv\text{O}^+$，因共振稳定，不会发生重排。

歧化反应

歧化（disproportionation）

在 $\text{AlCl}_3$ 等Lewis酸催化下，可发生分子间烷基转移：

例如：甲苯 + 甲苯 $\xrightarrow{\text{AlCl}_3}$ 苯 + 二甲苯

F-C 酰基化反应特点

酰基化 vs 烷基化的优势
  • 酰基是吸电子基，引入后钝化苯环 → 不会发生连续酰基化（不会多取代）
  • $\text{RCO}^+$ 因共振而稳定，不会发生碳正离子重排
  • 另外，傅-克酰基化反应也不可逆，不会发生歧化
  • 产物酮可通过还原得到烷基苯：$\text{ArCOR} \xrightarrow{\text{Zn-Hg / HCl}} \text{ArCH}_2\text{R}$（Clemmensen还原）
酰基化 + 还原 = 间接烷基化：先酰基化再还原，可以避免碳正离子重排和多烷基化问题。这是获得直链烷基苯的重要方法。 $$\text{ArH} \xrightarrow[\text{AlCl}_3]{\text{RCOCl}} \text{ArCOR} \xrightarrow{\text{Zn-Hg / HCl}} \text{ArCH}_2\text{R}$$
注意：F-C 酰基化需要超过1当量的 $\text{AlCl}_3$（$\text{AlCl}_3 > 2.0\;\text{equiv}$），因为产物酮会与 $\text{AlCl}_3$ 配位形成络合物，消耗催化剂。
Clemmensen 还原：$\text{Zn-Hg / HCl}$，将芳酮的羰基还原为亚甲基 $\text{-CH}_2\text{-}$，可以得到直链烷基苯。
$$\text{C}_6\text{H}_5\text{COCH}_3 \xrightarrow{\text{Zn-Hg / HCl}} \text{C}_6\text{H}_5\text{CH}_2\text{CH}_3$$

取代基定位规则与芳环侧链反应

一、一取代苯的定位规则

邻对位定位基（第一类定位基）

供电子基 —— 使苯环活化，新的取代基优先进入邻位和对位：

$\text{-O}^-$, $\text{-NR}_2$, $\text{-NHR}$, $\text{-NH}_2$, $\text{-OH}$, $\text{-OR}$, $\text{-NHCOR}$, $\text{-OCOCH}_3$,

$\text{-R}$（$\text{-CH}_3$等烷基）, $\text{-Ar}$（芳基）

特例：卤素（$\text{-F}$, $\text{-Cl}$, $\text{-Br}$, $\text{-I}$）是邻对位定位基但使苯环钝化（吸电子诱导效应 > 供电子共轭效应，总效果为吸电子，但共轭效应决定了定位方向为邻对位）。
间位定位基（第二类定位基）

吸电子基 —— 使苯环钝化，新的取代基优先进入间位：

$\text{-N}^+\text{R}_3$ / $\text{-N}^+\text{H}_3$, $\text{-CF}_3$, $\text{-CCl}_3$, $\text{-CN}$, $\text{-SO}_3\text{H}$, $^+\text{CHO}$,

$\text{-COR}$, $\text{-COOH}$, $\text{-COOR}$, $\text{-CONHR}$, $\text{-CONH}_2$,

$\text{-NO}_2$（最强钝化基之一）

Warning! 卤素虽是邻对位定位基（定位效应由共轭效应决定），但它们使苯环钝化（总体为吸电子效应），这一点容易出错！

二、二取代苯的定位规则

二取代苯上第三个取代基的位置

当苯环上已有两个取代基时，判断第三个取代基进入位置的规则：

  1. 两个定位基效应一致（协同定位）：取代基进入二者共同指定的位置
  2. 强活化基团的定位效应 > 弱活化基团
  3. 两个定位基效应矛盾时：以强定位基为主，并且一般不进入两个取代基之间（空间位阻）
二取代定位示例
底物已有取代基第三取代基位置
4-硝基甲苯$\text{-CH}_3$（邻对位）+ $\text{-NO}_2$（间位）以 $\text{-CH}_3$ 为主，进入 $\text{-CH}_3$ 的邻对位
3-硝基苯甲酸$\text{-COOH}$（间位）+ $\text{-NO}_2$（间位）协同：二者均指向间位

芳环侧链反应

苯环侧链氧化

侧链氧化

不论侧链多长，$\text{KMnO}_4$ / $\text{K}_2\text{Cr}_2\text{O}_7$（强氧化剂）氧化烷基苯时，均氧化为苯甲酸：

$$\text{C}_6\text{H}_5\text{-R} \xrightarrow{\text{KMnO}_4 / \text{H}^+} \text{C}_6\text{H}_5\text{COOH}$$

前提：苯环相邻碳（$\alpha$-碳）上必须有氢原子（$\alpha\text{-H}$）。若 $\alpha$-碳上无氢（如叔丁基苯），不被氧化。

苯环的催化加氢

苯环加氢

苯环在高温高压和催化剂（$\text{Ni}$, $\text{Pt}$, $\text{Pd}$ 等）条件下可被完全加氢：

$$\text{C}_6\text{H}_6 + 3\text{H}_2 \xrightarrow{\text{Ni},\;175°\text{C},\;180\;\text{atm}} \text{C}_6\text{H}_{12}\;\text{（环己烷）}$$

苯乙烯等含侧链不饱和键的芳烃，侧链双键先被加氢，苯环在更苛刻条件下才加氢：

$$\text{C}_6\text{H}_5\text{CH=CH}_2 \xrightarrow{\text{H}_2,\;\text{Ni}} \text{C}_6\text{H}_5\text{CH}_2\text{CH}_3 \xrightarrow{\text{高温高压}} \text{C}_6\text{H}_{11}\text{CH}_2\text{CH}_3$$
苄位（$\alpha$-碳）的特殊反应性：
  • R（苄位有 $\alpha\text{-H}$）：可氧化，被 $\text{KMnO}_4$ 氧化为 $\text{-COOH}$
  • R（苄位无 $\alpha\text{-H}$）：如叔丁基（$\text{-C(CH}_3)_3$），不被氧化

苯环侧链的卤代反应（自由基反应）

侧链卤代（自由基机理）
$$\text{C}_6\text{H}_5\text{CH}_3 \xrightarrow[h\nu]{\text{Cl}_2} \text{C}_6\text{H}_5\text{CH}_2\text{Cl} + \text{HCl}$$

在光照（$h\nu$）或过氧化物引发下，甲苯侧链发生自由基取代反应：

  • 链引发：$\text{Cl}_2 \xrightarrow{h\nu} 2\text{Cl}\cdot$
  • 链传递：$\text{Cl}\cdot + \text{C}_6\text{H}_5\text{CH}_3 \longrightarrow \text{C}_6\text{H}_5\text{CH}_2\cdot + \text{HCl}$
  • $\text{C}_6\text{H}_5\text{CH}_2\cdot + \text{Cl}_2 \longrightarrow \text{C}_6\text{H}_5\text{CH}_2\text{Cl} + \text{Cl}\cdot$
第 17 页

（七）F-C 反应判定点阵图

哪些底物不能发生 F-C 反应？

强吸电子基团取代的芳环不能发生 F-C 反应：

  • $\text{-NO}_2$（硝基苯）
  • $\text{-COR}$, $\text{-COOH}$（苯甲酰基/苯甲酸）
  • $\text{-SO}_3\text{H}$
  • 多卤代苯（环钝化严重）

不是所有芳环都能发生 F-C 反应！杂环（如吡啶）也不行。

F-C 反应的局限性总结
  • 傅-克反应需要加入催化剂（Lewis酸），如 $\text{AlCl}_3$, $\text{FeCl}_3$ 等
  • 傅-克烷基化反应的烷基化试剂可以用卤代烃、烯烃和醇等
  • 傅-克烷基化反应易发生重排反应，反应往往不能停在一元取代（多烷基化）
  • 傅-克酰基化反应不会生成多元取代的混合物

（八）氯甲基化反应

氯甲基化（Blanc反应）
$$\text{C}_6\text{H}_6 + \text{HCHO} + \text{HCl} \xrightarrow{\text{ZnCl}_2} \text{C}_6\text{H}_5\text{CH}_2\text{Cl} + \text{H}_2\text{O}$$

即用甲醛和盐酸在Lewis酸（$\text{ZnCl}_2$）催化下，在苯环上引入 $\text{-CH}_2\text{Cl}$（氯甲基）。

氯甲基化机理

$\text{HCHO} + \text{HCl} \longrightarrow \text{HOCH}_2\text{Cl}$（先生成氯甲醇）

$\text{H}^+ + \text{HOCH}_2\text{Cl} \longrightarrow ^+\text{CH}_2\text{Cl} + \text{H}_2\text{O}$（在酸催化下生成亲电试剂 $^+\text{CH}_2\text{Cl}$，或经 $\text{HOCH}_2^+$ 中间体）

然后亲电试剂进攻苯环，完成取代。

取代基定位效应总结

取代基对反应活性的影响

Factors 影响反应活性和选择性
  1. 电子效应（主要因素）
    • 诱导效应（I效应）：沿 $\sigma$ 键传递，与距离有关
    • 共轭效应（C效应）：通过 $\pi$ 体系传递，影响整个共轭体系
  2. 空间效应（位阻）：取代基体积较大时，邻位产物减少
指导方针
  • 供电子诱导效应（$+I$）：供电子，活化环
  • 吸电子诱导效应（$-I$）：吸电子，钝化环
  • 供电子共轭效应（$+C$）：供电子入环，增大电子密度
  • 吸电子共轭效应（$-C$）：吸电子出环，降低电子密度

两类定位基完整列表

1. 两类定位基

(1) 第一类定位基 — 邻、对位定位基：（除卤素外，均活化苯环）

$\text{-O}^-$, $\text{-NR}_2$, $\text{-NHR}$, $\text{-NH}_2$, $\text{-OH}$, $\text{-NHCOR}$, $\text{-OCOCH}_3$,

$\text{-CH}_3$（$\text{-R}$）, $\text{-C}_6\text{H}_5$, $\text{-Cl}$, $\text{-Br}$, $\text{-I}$, $\text{-F}$

与苯环直接相连的原子上一般有孤电子对，且多数有给电子共轭效应，使邻对位电子密度增大。

(2) 第二类定位基 — 间位定位基：（均为吸电子基，钝化苯环）

$\text{-NR}_3^+$, $\text{-NO}_2$, $\text{-CF}_3$, $\text{-CCl}_3$, $\text{-CN}$, $\text{-SO}_3\text{H}$, $^+\text{CHO}$,

$\text{-COR}$, $\text{-COOH}$, $\text{-COOR}$, $\text{-CONHR}$, $\text{-CONH}_2$

与苯环直接相连的原子上一般有重键或正电荷（$\text{C=O}$, $\text{N=O}$, $\text{S=O}$, $\text{C}\equiv\text{N}$, $\text{N}^+$），具有吸电子诱导和吸电子共轭双重效应。

记忆口诀：邻对位定位基 → 连接原子上有孤对电子（可给电子入环）；间位定位基 → 连接原子上有重键或正电荷（吸电子出环）。
$\text{-SO}_3\text{H}$（磺酸基）是间位定位基。

合成应用与稠环芳烃

以上反应在有机合成中的应用

利用定位效应的多步合成策略

合成目标产物时，关键在于选择正确的反应顺序，利用定位效应引导取代基进入期望的位置。

合成实例

目标：合成对硝基苯甲酸

路线：

  1. 甲苯 $\xrightarrow[\text{AlCl}_3]{\text{CH}_3\text{COCl}}$ 对甲基苯乙酮 — $\text{-CH}_3$ 是邻对位定位基（F-C酰基化）
  2. $\xrightarrow{\text{KMnO}_4}$ 对甲基苯甲酸（氧化 $\text{-COCH}_3 \rightarrow \text{-COOH}$）
  3. 先氧化，再进行硝化（或调整顺序以得到正确的对位产物）
合成思路举例

例1：由甲苯合成对溴苯甲酸

  1. 甲苯 $\xrightarrow[\text{FeBr}_3]{\text{Br}_2}$ 对溴甲苯（$\text{-CH}_3$ 邻对位定位，取对位产物）
  2. 对溴甲苯 $\xrightarrow{\text{KMnO}_4 / \text{H}^+}$ 对溴苯甲酸（侧链氧化）

例2：由甲苯合成间溴苯甲酸

  1. 甲苯 $\xrightarrow{\text{KMnO}_4}$ 苯甲酸（先氧化）
  2. 苯甲酸 $\xrightarrow[\text{FeBr}_3]{\text{Br}_2}$ 间溴苯甲酸（$\text{-COOH}$ 是间位定位基）

例3：由苯合成对硝基苯甲酸

  1. 苯 $\xrightarrow[\text{AlCl}_3]{\text{CH}_3\text{Cl}}$ 甲苯
  2. 甲苯 $\xrightarrow[\text{混酸}]{\text{HNO}_3/\text{H}_2\text{SO}_4}$ 对硝基甲苯（$\text{-CH}_3$ 是邻对位定位基）
  3. 对硝基甲苯 $\xrightarrow{\text{KMnO}_4}$ 对硝基苯甲酸

关键原则：

  • 先引入邻对位定位基，再引入间位定位基 → 得到对位产物
  • 先氧化为 $\text{-COOH}$（间位定位基），再引入其他基团 → 得到间位产物
  • 注意：先硝化后氧化；先氧化后溴化；顺序不同→产物不同
合成中的注意事项
  • 侧链氧化（$\text{KMnO}_4$）将 $\text{-CH}_3$ 变为 $\text{-COOH}$：定位效应从邻对位定位变为间位定位
  • 利用磺酸基占位（磺化→占位→其他反应→脱磺）的策略
  • 注意 $\text{-NH}_2$ 的保护：先酰化保护为 $\text{-NHCOCH}_3$（仍为邻对位定位基，但降低活性，避免多取代和氧化），反应后再水解脱保护

稠环芳烃

稠环芳烃的定义

两个或多个苯环通过共用边（共用两个碳原子）稠合在一起的多环芳烃。

常见稠环芳烃：萘（naphthalene）、蒽（anthracene）、菲（phenanthrene）

1. 萘 $\text{C}_{10}\text{H}_8$（两个苯环稠合）

(1) 萘的结构：共平面

萘的四种键长：

  • $a = 0.136\;\text{nm}$
  • $b = 0.140\;\text{nm}$
  • $c = 0.142\;\text{nm}$
  • $d = 0.139\;\text{nm}$

10个碳原子均为 $sp^2$ 杂化，全部共平面。键长不完全相等，说明萘的键长平均化程度不如苯。

$\alpha$ 位的键 $a$ 较短（$0.136\;\text{nm}$），接近双键键长 → 这解释了萘的化学活性比苯高。

(2) 萘的位置编号

萘有两种不等价的氢原子位置：

  • $\alpha$ 位（C-1, C-4, C-5, C-8）：与稠合键相邻
  • $\beta$ 位（C-2, C-3, C-6, C-7）：与稠合键不直接相邻
a) 萘的取代反应

萘的亲电取代反应比苯容易得多（因为萘的键长平均化不如苯，芳香性稍弱，$\pi$ 电子更容易提供）。

亲电取代优先进入 $\alpha$ 位（动力学控制）：

  • $\alpha$ 位比 $\beta$ 位反应活性更高
  • 进攻 $\alpha$ 位形成的 $\sigma$ 络合物中间体更稳定（可画出更多不破坏另一个苯环芳香性的共振式）
亲电取代进攻 $\alpha$ 位 vs $\beta$ 位

进攻 $\alpha$ 位：$\sigma$ 络合物中间体可画出2个不破坏另一苯环芳香性的共振结构

进攻 $\beta$ 位：$\sigma$ 络合物中间体只能画出1个不破坏另一苯环芳香性的共振结构

→ $\alpha$ 位中间体更稳定，反应速率更快，$\alpha$ 取代为主

萘的取代规律总结
  • 动力学控制（低温）：亲电取代主要在 $\alpha$ 位
  • 热力学控制（高温）：如磺化在高温下，$\beta$ 位产物（$\beta$-萘磺酸）更稳定，为主要产物
  • 硝化反应：优先得到 $\alpha$-硝基萘（1-硝基萘）
磺化的温度效应：
  • 低温（$80°\text{C}$）磺化 → 主要得到 $\alpha$-萘磺酸（动力学产物）
  • 高温（$160°\text{C}$）磺化 → 主要得到 $\beta$-萘磺酸（热力学产物，$\beta$ 位产物空间位阻小，更稳定）
萘的氧化

萘比苯更容易被氧化：

  • 催化氧化：$\text{萘} \xrightarrow[\text{V}_2\text{O}_5]{\text{O}_2} \text{邻苯二甲酸酐}$（工业上重要反应）
  • $\text{CrO}_3$ 氧化 → 1,4-萘醌
蒽与菲：三个苯环的稠合方式不同。蒽为直线型稠合（linear），菲为角型稠合（angular）。菲的 9,10-位（K区）反应活性最高。
编号规则：从稠合键旁边的碳原子开始编号。
第 19 页

b) 萘的还原反应

Birch 还原

萘在 $\text{Na / ROH}$（液氨中）条件下发生 Birch 还原：

$$\text{萘} \xrightarrow{\text{Na},\;\text{ROH}} \text{1,4-二氢萘} \xrightarrow{\text{继续}} \text{1,2,3,4-四氢萘}$$

注：Na/ROH 条件下先得到 1,4-二氢萘（非共轭的1,4-加氢产物）。

2. 蒽 (anthracene)

蒽的结构与反应

蒽由三个苯环线性稠合构成。环的编号为 1-10，其中 9,10 位活性最高。

芳香性比苯及萘差，取代、氧化、还原都优先发生在 9,10-位。

蒽的 9,10-位被称为meso 位。蒽醌是重要的染料中间体。

3. 菲 (phenanthrene)

菲的结构与反应

菲由三个苯环角状稠合构成。

芳香性比苯及萘差，取代、氧化、还原都优先发生在 9,10-位（K 区）。

菲的 9,10-键长最短（$0.134\;\text{nm}$），双键特征最强，化学活性最高。

稠环芳烃反应活性总结：芳香性比苯及萘差，取代、氧化、还原都发生在 9,10-位。

芳香性

Huckel 规则

Huckel 规则 — 判定芳香性的三个条件（必须全部满足）
  1. 单环共轭多烯，环上骨架共平面（平面结构）
  2. 环上每个原子都有 $p$ 轨道（参与共轭）
  3. 离域 $\pi$ 电子数为 $4n+2$（$n=0,1,2,\ldots$），即 $2, 6, 10, 14, 18, \ldots$

芳香性判据

(1) 亲电取代反应（首选判据）

能发生亲电取代反应（而非加成反应）是芳香性的最直接证据：

$$+ \text{E}^+ \longrightarrow \text{取代产物} + \text{H}^+$$

$\text{E}^+ = \text{NO}_2^+,\;\text{X}^+,\;\text{R}^+,\;^+\text{SO}_3\text{H},\;\text{RCO}^+$ 等

(2) 磁环流（NMR 判据）
  • 芳香化合物在外磁场中产生环电流（$\pi$ 电子环流）
  • 芳香性：环外质子受去屏蔽效应，化学位移偏向低场（$\delta$ 值大）；环内质子（如[18]轮烯内部H）化学位移偏向高场（$\delta$ 值小）
  • 反芳香性：环电流方向相反，效果正好相反
(3) 热力学稳定性
  • 比较氢化热 $\Delta H$：实测氢化热低于理论计算值，差值即为共振能/离域能（额外稳定化能）
  • 共振能越大，芳香性越强

非苯芳烃

(4) 轮烯 (annulene)

轮烯是全共轭的单环多烯。命名：[$n$]轮烯（$n$ 为环上碳原子数）。

  • 有芳香性：[6]轮烯（苯）、[10]轮烯、[14]轮烯、[18]轮烯 — $\pi$ 电子数满足 $4n+2$
  • 反芳香性：[4]轮烯（环丁二烯）、[8]轮烯（环辛四烯）— $\pi$ 电子数为 $4n$

注：[10]轮烯因氢原子空间排斥，实际上并非严格平面，芳香性较弱。[18]轮烯几乎为平面，有明显芳香性。

芳香离子与杂环

(5) 具有芳香性的离子
  • 环丙烯正离子 $\text{C}_3\text{H}_3^+$：$2$ 个 $\pi$ 电子（$4 \times 0 + 2 = 2$），具有芳香性
  • 环戊二烯负离子 $\text{C}_5\text{H}_5^-$：$6$ 个 $\pi$ 电子（$4 \times 1 + 2 = 6$），具有芳香性
  • 环庚三烯正离子（$\text{C}_7\text{H}_7^+$，对应"[七]卓鎓离子"）：$6$ 个 $\pi$ 电子，具有芳香性
芳香性判断步骤：
  1. 确认是否为平面环状共轭体系（环上每个原子都有 $p$ 轨道）
  2. 数 $\pi$ 电子数（注意孤电子对是否参与共轭）
  3. 判断是否满足 $4n+2$：满足 → 芳香性；$4n$ → 反芳香性
(6) 薁（azulene，甘菊环）

薁 = 五元环 + 七元环稠合的非苯芳烃。

  • $10$ 个 $\pi$ 电子（$4 \times 2 + 2 = 10$），满足 Huckel 规则，具有芳香性
  • 偶极矩较大：五元环端带部分负电（类似环戊二烯负离子），七元环端带部分正电（类似环庚三烯正离子）
(7) 稠环芳烃的芳香性判断
  • 对于稠环芳烃（如薁、苊、荧蒽等），分析外周 $\pi$ 电子是否满足 $4n+2$
  • 共用边上的电子不重复计数
