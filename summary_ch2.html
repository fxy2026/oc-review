第二章 烷烃和环烷烃

一、烷烃

链状烷烃
  • 通式：$\text{C}_n\text{H}_{2n+2}$
  • 特点：$\sigma$ 键、$sp^3$ 杂化、$109.5°$ 键角

二、命名

A. 开链烷烃（系统命名法）
  1. 选主链：最长碳链（长 > 多 > 支 → 多）
  2. 编号：近（近取代基端）、低（编号之和最小）
  3. 写名称：① 写名 ② 加取代 ③ 写明主链副位次 ④ 标顺序
  4. 书写格式：2,05-三甲基-7-乙基庚烷
B. 单环烷烃
  1. 编号：$\alpha$ 取代基的位置要尽量标记小的编号
  2. 先尝试从各方向编号
    • ② 先产生最低系列的 ③ 与 ④
    • 例：编号 1,2- ... 优于 1,3- ...
C. 桥环烷烃
  1. 基础：二环（入门）/ 三环（复杂），只需掌握二环
  2. 由桥环连接，各桥按碳数从大到小排列
  3. 命名格式：二环[X.Y.Z]烷，括号内标注各桥碳原子数（降序排列，如 [2.2.1]）
  4. 编号起点：从桥头碳开始，沿最长桥 → 次长桥 → 最短桥
  5. 例：二环[2.2.1]庚烷；1,7,7-三甲基二环[2.2.1]庚烷-2-酮（即樟脑）
多环环烷烃
  • 螺环化合物：两环共用一个碳原子（螺原子），命名为 螺[X.Y]烷
  • 桥环化合物：两环共用两个碳原子（桥头碳），即多个 C 组成的桥
椅式构象

三、结构

碳原子的分类

C 按上面所连的其他碳原子数分为：

  • 伯碳（1°C）：连 1 个 C
  • 仲碳（2°C）：连 2 个 C
  • 叔碳（3°C）：连 3 个 C
  • 季碳（4°C）：连 4 个 C

碳上的 H 同理分为伯 H、仲 H、叔 H

环烷烃的结构
  • 小环（3~4元环）：角张力大，不稳定
  • 常见环（5~6元环）：最稳定（接近 $109.5°$）
  • 特别注意：三元环：角张力 → 弯曲键
环烷烃的构象
  • 环丙烷、环丁烷的角张力分析
  • 环己烷最重要的构象：
    • 椅式构象 → 扭曲程度小（对位交叉），最稳定
    • 船式构象 → 不稳定

四、构象

乙烷构象
  • 交叉式：通过 Newman 投影，最稳定 / 扭转能最低（$H$ 之间距离最大）
  • 重叠式：扭转能最高，最不稳定
丁烷构象
  • 全交叉式（anti）：最稳定
  • 邻交叉式（gauche）：$\text{CH}_3$ 之间 gauche 相互作用
  • 全重叠式：最不稳定
  • 稳定性：对位交叉 > 邻位交叉 > 部分重叠 > 全重叠
环己烷构象
  • 椅式构象有直键(a键)和平键(e键)
  • 取代基在 e 键（平键）上更稳定（1,3-二轴向相互作用更小）
  • 翻环：椅式翻转，所有 a 键变 e 键、e 键变 a 键
  • 大取代基优先占 e 键！
构象分析口诀：
  • 取代基越大 → 越要占据 e 键（平伏键）
  • 交叉 > 重叠（扭转能低 → 稳定）
  • 大取代基避免 1,3-二轴向作用
注意：顺反异构与构象异构不同！
  • 构象异构：通过键的旋转互变，不需断键
  • 顺反异构：需要断键才能互变

五、物理性质

物理性质规律
  • 沸点：支链 < 直链（"如同样碳数"）
  • 熔点：对称性好的高、偶数碳比奇数碳熔点高
  • 溶解性：相似相溶

六、化学性质

1. 卤代反应（自由基反应）

总反应：

$$\text{R-H} + \text{X}_2 \xrightarrow{h\nu \text{ 或 }\Delta} \text{R-X} + \text{HX}$$

（$\text{X} = \text{Cl}, \text{Br}$）反应活性 $\text{F}_2 > \text{Cl}_2 > \text{Br}_2 > \text{I}_2$（碘太慢不反应）

自由基链式反应机理：

  1. 链引发：$\text{Cl}_2 \xrightarrow{h\nu} 2\text{Cl}\cdot$
  2. 链增长：
    • $\text{Cl}\cdot + \text{CH}_4 \to \text{CH}_3\cdot + \text{HCl}$
    • $\text{CH}_3\cdot + \text{Cl}_2 \to \text{CH}_3\text{Cl} + \text{Cl}\cdot$
  3. 链终止：自由基相互结合
卤代反应的选择性：
  • 活性（H的种类）：$3°\text{C-H} > 2°\text{C-H} > 1°\text{C-H}$
  • 判断卤代产物时须同时考虑：H 的种类 + H 的个数
  • 选择性：$\text{Br}_2$ 选择性 > $\text{Cl}_2$（$\text{Br}_2$ 几乎只取代 3°H）
自由基稳定性顺序：
$$3°\text{C}\cdot > 2°\text{C}\cdot > 1°\text{C}\cdot > \text{CH}_3\cdot$$

原因：含 C° 越多越稳定，超共轭效应（$\alpha$-H 越多）越强

2. 加成反应（环烷烃）

小环烷烃的开环加成（环丙烷特有）

环丙烷的弯曲键使其像烯烃一样可发生加成反应：

(1) 加 $\text{H}_2$（催化加氢）：

$$\triangle + \text{H}_2 \xrightarrow{\text{Ni}, \Delta} \text{CH}_3\text{CH}_2\text{CH}_3$$

(2) 加 $\text{Cl}_2$ / $\text{Br}_2$：

$$\triangle + \text{Cl}_2 \xrightarrow{\text{FeCl}_3} \text{ClCH}_2\text{CH}_2\text{CH}_2\text{Cl}$$ $$\triangle + \text{Br}_2 \to \text{BrCH}_2\text{CH}_2\text{CH}_2\text{Br}$$

(3) 加 HBr：

$$\triangle + \text{HBr} \to \text{CH}_3\text{CH}_2\text{CH}_2\text{Br}$$
取代环丙烷加 HX：
  • 遵循 Markovnikov 规则
  • H 加在含 H 较多的碳上（类似烯烃加成）
  • 开环断裂的是取代最多的 C-C 键

3. 氧化反应

氧化反应

(1) 燃烧（完全氧化）

(2) 受控氧化：

$$\text{RCHCOR'} + \text{O}_2 \xrightarrow{\text{Mn}^{2+}/\text{Co}^{2+}} \text{RCOOH} + \text{R'COOH} + \text{羧酸/酮}$$

烷烃在催化条件下可被氧化为羧酸（工业用途）
