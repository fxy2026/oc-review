第六章 对映异构体

概念

手性：物质的实体与镜像不能完全重叠的性质。[相当于左手和右手的关系]

方向性与条件：①没有对称面、对称中心、$S_4$ 交替轴 → 才有手性；②要有不对称碳（$C^*$），即连有4个不同基团的碳

R/S 构型

由 CIP 优先次序规则判断：将最小基团朝离观察者最远方向（向后），剩余三个基团按优先次序排列：

  • 顺时针 → R 构型
  • 逆时针 → S 构型
有旋光性时，每台旋光的标记为 $R/S$

ee% 光学纯度

对映体过量百分率
$$ee\% = \frac{[R]-[S]}{[R]+[S]} \times 100\%$$

也可表示为：

$$ee\% = \frac{[\alpha]_{\text{obs}}}{[\alpha]_{\max}} \times 100\%$$

外消旋体的 $ee\% = 0$，纯对映体的 $ee\% = 100\%$

对映体 vs 非对映体

  • 对映体（enantiomers）：互为镜像关系的立体异构体，所有手性碳构型全部相反
  • 非对映体（diastereomers）：不是镜像关系的立体异构体，有些手性碳相同/有些相反
两个 $E/Z$ 构型化合物不能算对映体和非对映体！指的是手性碳的构型关系。

指甲利对映体：$R$和$S$之间是对映关系，$RR$和$SS$之间也是对映关系

外消旋体：$R/S$ 等量混合物（等量+旋光相互抵消），旋光度 = 0

$M$ 构型：| 以 $\dot{C}$ 为中心，$4$个不同基团按CIP优先序从 $1$ 到 $4$ 排列后的左旋/右旋判断排列方向 |

$(+)$：右旋（dextrorotatory）   $(-)$：左旋（levorotatory）  （与 $R/S$ 无必然对应关系，需实验测定）

构型的表示方法

1. 透视式（楔形式）

用楔形键、虚线键表示空间方向：

  • 实楔形：伸向观察者（前方）
  • 虚线楔形：远离观察者（后方）
  • 要标清楚氢的位置，可直观判断 $R/S$

2. Fischer 投影式

"横前竖后"：横线上的取代基朝向观察者（前方），竖线上的取代基远离观察者（后方）

Fischer 投影式规则
  • ①投影后为"十"字，手性碳C在交叉点
  • ②竖直方向固定（碳链纵向），最高氧化态碳在上方
  • ③旋转 $180{^\circ}$ 后，构型不变（同一化合物）；旋转 $90{^\circ}$ 后，构型改变（变为对映体）
  • ④不能离开纸面翻转（翻转 = 构型改变），不能随意交换两个基团的位置；交换奇数次 → 变为对映体；交换偶数次 → 同一化合物
相互转化：Fischer投影式、Sawhorse（锯架式）、Newman投影式之间可以相互转化，但要注意规则。

比旋光度

比旋光度公式：

$$[\alpha]_D^{t} = \frac{\alpha}{l \cdot c}$$

大写希腊 $\alpha$ = 实测旋光度，$l$ = 光程长度（dm），$c$ = 浓度（g/mL），$t$ = 温度，$D$ = 钠光波长（589 nm）

构型标记方法：D-L 构型标记法

D-L 构型标记法：糖、氨基酸等化合物的标记中使用最广。

D-L 标记规则
  • 以甘油醛为标准进行标记（Fischer投影式）。实质是指与标准物的构型关系，右为 D，左为 L。
  • 以甘油醛为参考标准，看不改变手性碳构型的条件下，能否从D-甘油醛或L-甘油醛出发合成该化合物 → 即定为 $D$ 或 $L$。

D-甘油醛（Fischer投影式中 $-\text{OH}$ 在右）  →   L-甘油醛（$-\text{OH}$ 在左）

D-(+)-甘油醛     L-(-)-甘油醛     D-丙氨酸     L-丙氨酸

人为规定 D-甘油醛为 (+) 右旋；实验测定旋光方向。

D, L 表示构型，(+) 为右旋，(-) 为左旋。D 不一定是右旋，L 不一定是左旋。需由实验测定旋光方向。

对映与构型的关系 与 非对映的关系

非对映对称面

对称面互变关系
对映体有镜面全互换
非对映体无镜面非全互换

概念

对比项对映体 vs 非对映体内消旋 vs 外消旋
定义对映体：互为镜像的立体异构体
非对映体：非镜像关系的立体异构体		内消旋：分子内对称，整体无旋光性
外消旋：等量对映体混合物
性质差异对映体：物理性质相同（除旋光方向）
非对映体：物理性质不同		内消旋：纯物质，无法拆分
外消旋：混合物，可拆分
旋光性对映体：旋光度等值反号
非对映体：旋光度各不相同		两者旋光度均 = 0
  • 左旋体对映体 → 右旋体对映体（配对关系），右旋体 + 左旋体 = 外消旋体
  • 两对相似取代基在同侧 → 赤式(erythro)，异侧 → 苏式(threo)；两者都是非对映体关系
每一个手性碳 = 最多 $2^n$ 个立体异构体

$n$ 个手性碳 → 最多 $2^n$ 个立体异构体（含内消旋后可能减少）

环状化合物的对映异构体

环状化合物中如存在手性碳，也有对映异构体问题。

  • 以二取代环己烷、环戊烷为例
  • 顺式：两个取代基在环平面同侧
  • 反式：两个取代基在环平面两侧

例：反式-1,2-二甲基环戊烷 → 有一对对映体；顺式-1,2-二甲基环戊烷 → 有内消旋面，是 $meso$ 化合物

含多个手性碳的化合物 (meso)

meso 化合物 = 内消旋体

$\text{meso}$ = 特殊情况：含偶数个手性碳，分子内有对称面（或对称中心），整体无旋光性。

$A\overset{a}{-}\overset{}{C}\overset{b}{=}\overset{}{C}\overset{a'}{-}A'$：如果 $a = a', b = b'$，则可能有 $\text{meso}$ 构型

每一个C达到2个相同的基团就记得考虑 $\text{meso}$（内消旋体）是否存在！
