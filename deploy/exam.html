1. 桥环烷烃结构 → 命名
答案

7-甲基二环[2.2.1]庚烷

2. 含Cl的烯烃 → 命名
答案

(E)-3-氯-2-苯基戊-2-烯

3. 含苯基的环己烯 → 命名
答案

4-苯基-1-环己烯

4. Fischer投影式 → 命名（含R/S）
答案

(2R,3R)-戊-2,3-二醇

需要先判断每个手性碳的R/S构型，再命名。
5. 8-硝基萘-2-酚 → 画结构
答案

6. 烯丙基乙炔基醚 → 画结构
答案

二、选择题（18分，12题×1.5分）

1. 没有手性的化合物 → B
解析

B选项分子有对称面，无手性。考查对称面判断。

2. 烯烃加氢最快 → B（$\text{H}_2\text{C=CH}_2$）
解析

取代基越多→烯烃越稳定→在催化剂表面吸附越难→加氢越慢。乙烯取代基最少，加氢最快。

3. 能与亚铜氨溶液反应 → C（$\text{C}_2\text{H}_5\text{C≡CH}$，末端炔）
解析

只有端炔（末端炔烃）能与银氨/铜氨溶液反应生成沉淀。

4. 有芳香性的 → A（甲基环庚三烯正离子，6π电子）
解析

环庚三烯正离子（tropylium）有6个π电子，满足$4n+2$（$n=1$），共平面，有芳香性。

5. 顺-2-丁烯→反-2-丁烯的方法 → C
解析

顺 → $\text{Br}_2/\text{CCl}_4$（反式加成）→ 二溴代物 → $\text{NaNH}_2$（消除）→ 炔烃 → $\text{Na}/\text{液NH}_3$（反式加氢）→ 反式烯烃

关键：Na/液$\text{NH}_3$还原炔烃得E-烯烃（反式）；Lindlar催化剂得Z-烯烃（顺式）。
6. 与$\text{HIO}_4$生成两种产物 → D
解析

$\text{HIO}_4$只断裂邻二醇的C-C键。D选项的邻二醇断裂后生成两种不同的羰基化合物。

7. 硝化最快 → A（甲氧基苯 $-\text{OCH}_3$）
解析

$-\text{OCH}_3$是强活化基（$+C$效应），使苯环电子云密度最高，亲电取代最快。

8. 最易与$\text{CH}_3\text{Br}$发生SN2 → D（$\text{C}_2\text{H}_5\text{O}^-$）
解析

SN2需要强亲核试剂。$\text{C}_2\text{H}_5\text{O}^-$亲核性最强（烷氧负离子，碱性和亲核性都强）。

9. $\text{CH}_3\text{I}$浓度3倍、$\text{OH}^-$浓度1/2 → 速度变为？ → C（提高50%）
解析

SN2速率 $v = k[\text{CH}_3\text{I}][\text{OH}^-]$，新速率 = $k \times 3[\text{CH}_3\text{I}] \times \frac{1}{2}[\text{OH}^-] = \frac{3}{2}v$，即提高50%。

10. 脱水最快 → A（苄醇类）
解析

醇脱水经碳正离子中间体。苄基碳正离子被苯环共振稳定，最容易形成，脱水最快。

11. 1,3-丁二烯+$\text{Br}_2$能量图 → B
解析

1,2-加成活化能低（动力学产物），1,4-加成产物更稳定（热力学产物）。B图正确反映了这种关系。

12. 手性碘化物在丙酮/水中转化为醇 → B（构型翻转）
解析

丙酮/水中，$\text{OH}^-$（或$\text{H}_2\text{O}$）作为亲核试剂进行SN2反应，Walden翻转导致构型翻转。

三、完成反应（24分，重点题型）

2a. $(CH_3)_3CCH_2OH + SOCl_2$/吡啶 → ？
答案

$(CH_3)_3CCH_2Cl$（新戊基氯，构型翻转，SN2机理）

2b. $(CH_3)_3CCH_2OH + $浓HCl/ZnCl₂ → ？
答案

发生新戊基重排！伯碳正离子→1,2-甲基迁移→叔碳正离子→叔碳上的Cl

新戊基体系的SN1反应必然重排，这是高频考点。
5. 手性氯代物分别与AgNO₃/乙醇 和 NaI/丙酮反应
答案

AgNO₃/乙醇：SN1 → 外消旋化（平面碳正离子，两面等概率进攻）

NaI/丙酮：SN2 → 构型翻转（Walden反转，背面进攻）

6. 2-甲基-1,3-丁二烯 + HBr(1mol)/ROOR
答案

过氧化物效应→自由基机理→1,4-反马加成产物（Br加在端碳上）

8b. 邻二醇 + $\text{H}^+$ → ？
答案

频哪醇重排：酸催化下邻二醇脱水，碳正离子经1,2-迁移重排生成酮/醛。

六、合成题（20分，重点）

3. 从苯合成邻硝基苯甲酸
答案

苯 →(CH₃Cl/AlCl₃)→ 甲苯 →(H₂SO₄)→ 对甲苯磺酸 →(HNO₃/H₂SO₄)→ 邻位硝化（对位被磺酸基占据）→(H₂SO₄/H₂O,Δ)→ 脱磺酸基 →(KMnO₄)→ 邻硝基苯甲酸

磺酸基占位策略：先磺化占据对位→硝化只能去邻位→再脱磺酸基。合成设计高频考点！
4. 从乙炔合成1-戊醇
答案

$\text{HC≡CH}$ →(NaNH₂)→ $\text{HC≡CNa}$ →(CH₃I)→ $\text{CH}_3\text{C≡CH}$ →(NaNH₂)→ $\text{CH}_3\text{C≡CNa}$ →(环氧乙烷)→ 炔醇 →(H₂/Pt)→ 1-戊醇

增碳策略：炔钠+RX（SN2）增碳 + 环氧乙烷开环增两碳

七、推结构（5分）

A($C_4H_{10}O$) → CrO₃/H₂SO₄ → B；A脱水→唯一烯烃C；C + 稀冷KMnO₄ → D；D + HIO₄ → 醛E + 酮F
答案

A = 2-丁醇 → B = 2-丁酮 → C = 2-丁烯 → D = 2,3-丁二醇 → E = 乙醛 + F = 乙醛

推断链条：醇的氧化类型判断(仲醇→酮)→脱水方向(Zaitsev)→KMnO₄顺式双羟基化→HIO₄断邻二醇
