第4章 炔烃和二烯烃 梳理

第 8 页

第四章 炔烃和二烯烃

(1) 结构与命名

炔烃结构

通式：$\text{C}_n\text{H}_{2n-2}$，含一个 $\text{C}\equiv\text{C}$ 三键（1 个 $\sigma$ 键 + 2 个 $\pi$ 键）。

$\text{HC}\equiv\text{CH}$：sp 杂化，线型分子（键角 $180^\circ$）。

两个相互垂直的 $\pi$ 键构成圆柱形 $\pi$ 电子云，包裹在 $\sigma$ 键周围。

$\Delta$ C-H 酸性

碳氢键酸性比较

键长：$d(\text{C}_{sp}\text{-H}) < d(\text{C}_{sp^2}\text{-H}) < d(\text{C}_{sp^3}\text{-H})$

酸性顺序：$\text{C}_{sp}\text{-H} > \text{C}_{sp^2}\text{-H} > \text{C}_{sp^3}\text{-H}$

原因：s 轨道成分越多（sp: 50%, sp$^2$: 33%, sp$^3$: 25%），碳的电负性越大，C-H 键极性越强，越容易失去 $\text{H}^+$。

命名

  • 母体的选择：含叁键的最长碳链
  • 编号：使叁键位次最小
  • 烯炔：先烯后炔（名称中先写烯再写炔）

物理性质

  • 低级炔烃是无色气体，略溶或不溶于水
  • 分子间不能形成氢键，沸点主要由分子量决定
  • 随碳原子数增大，沸点升高
  • 乙炔微溶于水，其在水中的溶解度略大于乙烯
  • 炔烃的密度小于1，轻于水（液态炔烃比重 < 1）
  • 乙炔是无色气体，由电石制得的含杂质（$\text{H}_2\text{S}$, $\text{PH}_3$）有臭味
芳香性判定（Huckel 规则）

环状、共面、全共轭体系，$\pi$ 电子数 = $4n+2$（$n=0,1,2,...$）时具有芳香性。

反芳香性：$\pi$ 电子数 = $4n$，平面共轭环不稳定。

Stability（稳定性）：$K_{\text{eq}}$ 大表示热力学更稳定；Reactivity（反应性）：速率快慢，和活化能有关。
基团比较中需要分清热力学稳定性和动力学反应性的区别。

同分异构

不饱和度 (IHD / Degree of Unsaturation)

公式：$\text{IHD} = \frac{2C + 2 + N - H - X}{2}$

一个双键 = IHD 1，一个叁键 = IHD 2，一个环 = IHD 1

例：$\text{C}_4\text{H}_6$（IHD = 2）的同分异构体：
  • $\text{CH}\equiv\text{C-CH}_2\text{-CH}_3$（1-丁炔）
  • $\text{CH}_3\text{-C}\equiv\text{C-CH}_3$（2-丁炔）
  • $\text{CH}_2\text{=CH-CH=CH}_2$（1,3-丁二烯）
  • $\text{CH}_2\text{=C=CH-CH}_3$（1,2-丁二烯）
  • 环丁烯
  • 甲基环丙烯等
基团优先级与命名
含有烯、炔双功能团时，编号使不饱和键位次之和最小；位次相同时，双键优先获得较小编号。

(5) 亲电取代 — 卤化 $\text{X}_2$ ($\alpha$, $\beta$)

催化卤化反应：

$\text{CH}_3\text{Br} + \text{FeBr}_3 \to \text{CH}_3^+ + \text{FeBr}_4^-$

$\text{CH}_3^+ + \text{C}_6\text{H}_6 \to \text{甲苯} + \text{HBr}$

活化顺序（邻对位定位基）：$\text{-OH} > \text{-OR} > \text{-NH}_2 > \text{-CH}_3$

选择性
  • 甲基为邻对位定位基（给电子基团活化芳环）
  • 卤素活泼性：$\text{F}_2 > \text{Cl}_2 > \text{Br}_2 > \text{I}_2$
注意：$\text{C}(sp)$ 电负性 > $\text{C}(sp^2)$，含炔基的碳吸电子。
中间体碳正离子可能发生重排（1,2-H 迁移或 1,2-甲基迁移），但在芳环上不发生。

(6) 加 HX (HCl, HBr, HI)

硝化反应：

$\text{ArH} + \text{HNO}_3 \xrightarrow{\text{H}_2\text{SO}_4} \text{Ar-NO}_2 + \text{H}_2\text{O}$

亲电试剂：$\text{NO}_2^+$（硝基正离子），由混酸产生：

$\text{HNO}_3 + 2\text{H}_2\text{SO}_4 \to \text{NO}_2^+ + \text{H}_3\text{O}^+ + 2\text{HSO}_4^-$

重要概念区分
Geminal（gem-，同碳）二卤代物：两个卤素在同一碳上
Vicinal（vic-，邻碳）二卤代物：两个卤素在相邻碳上

注：二卤化物的命名（CHEM）中这两个前缀需要牢记区分。三级碳正离子比二级稳定，可影响加成区域选择性。

(5) 催化还原 / 氧化反应（续）

催化加氢

试剂/条件产物立体化学
$\text{H}_2/\text{Pt}$（完全加氢）烷烃顺式加成
$\text{D}_2/\text{Pt}$$d$-标记烷烃顺式加成
Lindlar 催化剂顺式烯烃顺式（syn）
$\text{Na/NH}_3(\text{l})$反式烯烃反式（anti）

(2) 亲核加成 (Nucleophilic Addition)

亲核加成定义

亲核试剂（Nu:）进攻三键碳原子，发生加成反应。

常见亲核试剂：$\text{-OH}$, $\text{-SH}$, $\text{-NH}_2$, $\text{-MeL}$, $\text{-CONH}_2$, $\text{-COOR}$, $\text{-CN}$ 等

反应底物：$\text{R-C}\equiv\text{C-H}$ 末端炔烃（活性较高）

炔烃亲核加成比烯烃更容易
原因：(1) sp 碳的电负性更大（50% s 成分），三键碳带部分正电荷，更易接受亲核试剂进攻；
(2) 加成产物为烯烃（比炔烃更稳定），热力学有利。

亲核加成反应类型汇总

1. HCN 加成

$\text{RC}\equiv\text{CH} + \text{HCN} \xrightarrow{\text{CuCl}} \text{R-C(=CH}_2\text{)-CN}$

产物为$\alpha,\beta$-不饱和腈（丙烯腈类化合物）。

2. ROH 加成（乙烯基醚合成）

$\text{HC}\equiv\text{CH} + \text{ROH} \xrightarrow{\text{KOH}} \text{CH}_2\text{=CH-OR}$（乙烯基醚）

产物为烯醇醚（vinyl ether），可用于麻醉剂等。

3. CH$_3$COOH 加成（乙烯基酯）

$\text{HC}\equiv\text{CH} + \text{CH}_3\text{COOH} \xrightarrow{\text{Hg}^{2+}} \text{CH}_2\text{=CH-OOCCH}_3$（醋酸乙烯酯）

催化剂：$\text{Hg}^{2+}$；工业上用于合成聚醋酸乙烯酯（PVAc）。

4. RMgBr 反应（有机金属试剂）

$\text{RC}\equiv\text{CH} + \text{R'MgBr} \to \text{RC}\equiv\text{CMgBr} + \text{R'H}$

末端炔烃酸性 > 烷烃，因此格氏试剂可夺取端炔 H，生成炔基格氏试剂。

反应本质是酸碱反应（非加成），但归类为金属有机试剂制备。

关键：亲核加成的区域选择性 — 亲核试剂加到含取代基少的碳上（末端碳），生成马氏加成产物。
与亲电加成的选择性互补！

(七) 酸性炔烃的反应 — 炔化物

(1) 金属炔化物的生成

活泼金属反应

$\text{CH}\equiv\text{CH} + \text{Na} \xrightarrow{\text{NH}_3(\text{l})} \text{CH}\equiv\text{CNa} + \frac{1}{2}\text{H}_2$

$\text{CH}\equiv\text{CH} + \text{K} \xrightarrow{\text{NH}_3} \text{CH}\equiv\text{CK} + \frac{1}{2}\text{H}_2$

NaNH$_2$ 反应

$\text{RC}\equiv\text{CH} + \text{NaNH}_2 \to \text{RC}\equiv\text{CNa} + \text{NH}_3$

用于制备末端炔烃的钠盐。

(2) 亲核取代反应 — 碳链增长

炔钠的烷基化

$\text{RC}\equiv\text{CNa} + \text{R'CH}_2\text{CH}_2\text{Br} \xrightarrow{\text{DMF}} \text{RC}\equiv\text{C-CH}_2\text{CH}_2\text{R'} + \text{NaBr}$

用于碳-碳键的形成。

(3) 末端炔烃的鉴别 $\text{RC}\equiv\text{CH}$（多刘别银、铜检验）

银镜反应（Tollens 检验）

$\text{CH}\equiv\text{CH} + 2\text{AgNO}_3 + 2\text{NH}_4\text{OH} \to \text{Ag-C}\equiv\text{C-Ag}\downarrow + 2\text{NH}_4\text{NO}_3 + 2\text{H}_2\text{O}$

产生白色沉淀（炔化银）。

铜检验

$\text{CH}\equiv\text{CH} + 2\text{Cu}_2\text{Cl}_2 + 2\text{NH}_4\text{OH} \to \text{Cu-C}\equiv\text{C-Cu}\downarrow + 2\text{NH}_4\text{NO}_3 + 2\text{H}_2\text{O}$

产生红棕色沉淀（炔化亚铜）。

注意：金属炔化物干燥后受热或撞击会爆炸，实验后必须用稀 $\text{HNO}_3$ 分解处理！

(10) 末端炔烃的氧化

臭氧氧化

$\text{HOCH}\equiv\text{CH} + \text{ROH} \xrightarrow{\text{1) O}_3,\ \text{2) H}_2\text{O}} \text{R-CHO} + \text{CO}_2 \uparrow$

高锰酸钾氧化

$\text{RC}\equiv\text{CR'} \xrightarrow{\text{KMnO}_4} \text{RCOOH} + \text{R'COOH}$

$\text{RC}\equiv\text{CH} \xrightarrow{\text{KMnO}_4} \text{RCOOH} + \text{CO}_2$

臭氧化：$\text{R-C}\equiv\text{C-R'} \xrightarrow{\text{1) O}_3} \xrightarrow{\text{2) H}_2\text{O}} \text{RCOOH} + \text{R'COOH}$
末端炔氧化得到 $\text{CO}_2$，而非甲酸。
第 10 页

炔烃氧化反应（续）

臭氧化断裂

$\text{CH}_3\text{C}\equiv\text{CH} \xrightarrow{\text{1) O}_3} \text{CH}_3\text{COOH} + \text{H}_2\text{O (+ CO}_2\text{)}$

$\text{R-C(OH)=O}$（羧酸产物）

偶联反应

Glaser 偶联（氧化偶联）

$\text{RC}\equiv\text{CH} + \text{CuCl} \to \text{RC}\equiv\text{CCu}$

$2\ \text{RC}\equiv\text{CCu} \xrightarrow{\text{O}_2,\ \text{吡啶}} \text{RC}\equiv\text{C-C}\equiv\text{CR} + 2\text{CuCl}$

氧化偶联形成共轭二炔（1,3-二炔）。

Eglinton 偶联

$2\ \text{RC}\equiv\text{CH} \xrightarrow{\text{Cu(OAc)}_2,\ \text{吡啶}} \text{RC}\equiv\text{C-C}\equiv\text{CR}$

用 $\text{Cu}^{2+}$ 盐直接进行，无需先制备炔化铜。

乙炔的工业来源（电石法）

$\text{CaO} + 3\text{C} \xrightarrow{\text{电弧}} \text{CaC}_2 + \text{CO}$

$\text{CaC}_2 + 2\text{H}_2\text{O} \to \text{Ca(OH)}_2 + \text{CH}\equiv\text{CH}\uparrow$

即工业上从电石（碳化钙）制备乙炔的方法。

酸性炔的其它反应

酰化：$\text{RC}\equiv\text{CH} \xrightarrow[\text{(1) BuLi}]{\text{(2) R'COCl}} \text{R'C(=O)-C}\equiv\text{CR}$（炔基酮）

烷基化：$\text{RC}\equiv\text{CMgBr} + \text{R'X} \to \text{RC}\equiv\text{CR'} + \text{MgBrX}$

利用格氏试剂进行 $\text{S}_\text{N}2$ 碳碳键形成。注意只适用于伯卤代烃。

共轭二烯烃

命名

命名规则

$\text{CH}_2\text{=CH-CH=CH-CH=CH}_2$（1,3,5-己三烯）

  • 选含最多双键的最长碳链为母链
  • 从离双键最近的一端编号
  • 注意 s-cis 和 s-trans 构象

分类

  • 累积二烯烃（allene 型）：$\text{C=C=C}$，两个双键共用一个碳原子
  • 共轭二烯烃：$\text{C=C-C=C}$，双键被一个单键隔开
  • 隔离二烯烃：$\text{C=C-C-C=C}$，双键被两个或以上单键隔开

共轭二烯烃的结构

键长与共轭

以 1,3-丁二烯为例：$\text{CH}_2\text{=CH-CH=CH}_2$

C1=C2 和 C3=C4 双键键长约 $1.34\ \text{\AA}$

C2-C3 单键键长 $1.48\ \text{\AA}$（比正常 C-C 单键 $1.54\ \text{\AA}$ 短）

原因：四个碳均为 sp$^2$ 杂化，四个 p 轨道平行排列、连续重叠，$\pi$ 电子离域（共轭效应）。

s-cis 与 s-trans 构象

s-trans（$\sigma$-trans）：两个双键呈反式排列，更稳定

s-cis（$\sigma$-cis）：两个双键呈顺式排列，能量稍高但D-A 反应必须采用此构象

"s" 代表 single bond（绕 C2-C3 单键旋转）

共轭效应

  • $\pi$-$\pi$ 共轭：$\text{C=C-C=C}$
  • $p$-$\pi$ 共轭：杂原子孤对电子与 $\pi$ 键共轭
  • 超共轭：$\text{C-H}$ $\sigma$ 键与相邻 $\pi$ 键的共轭

共轭二烯烃的稳定性

氢化热比较

1,3-丁二烯氢化热 = $236\ \text{kJ/mol}$

两个独立的 1-丁烯氢化热 = $2 \times 126.8 = 253.6\ \text{kJ/mol}$

共轭稳定化能 $\approx 17.6\ \text{kJ/mol}$

共轭二烯烃的反应

1,2-加成 vs 1,4-加成

亲电加成选择性

$\text{CH}_2\text{=CH-CH=CH}_2 + \text{HBr} \to$

  • 1,2-加成：$\text{CH}_3\text{-CHBr-CH=CH}_2$（动力学产物，低温有利）
  • 1,4-加成：$\text{CH}_3\text{-CH=CH-CH}_2\text{Br}$（热力学产物，高温有利）
温度控制：
低温（$-80\ ^\circ\text{C}$）有利于 1,2-加成（动力学控制）
高温（$40\ ^\circ\text{C}$）有利于 1,4-加成（热力学控制）

Diels-Alder 反应（[4+2] 环加成）

D-A 反应（协同反应）

共轭二烯（双烯体，Diene，4$\pi$ 电子） + 亲双烯体（Dienophile，2$\pi$ 电子）

$\to$ 环己烯衍生物（一步成环，协同机理）

反应条件：加热即可（无需催化剂），属于周环反应

特点：

  • 双烯体必须能取 s-cis 构象（环状二烯如环戊二烯天然固定在 s-cis）
  • 亲双烯体通常含有吸电子基团（$\text{-CHO}$, $\text{-COOR}$, $\text{-CN}$, $\text{-NO}_2$, $\text{-COR}$）
  • 区域选择性：双烯体上给电子基和亲双烯体上吸电子基互补配对
  • 立体专一性：顺式加成（suprafacial），保持双烯体和亲双烯体原有的立体化学
  • 产物立体化学：endo 规则优先（次级轨道相互作用导致 endo 产物为动力学产物）
D-A 反应是有机合成中最重要的成环反应之一，一步构建六元环并同时形成两个新的 C-C 键。可逆（逆 D-A 反应，retro D-A）。
经典例子：
1,3-丁二烯 + 马来酸酐（maleic anhydride） $\to$ 四氢邻苯二甲酸酐
（Diene + Dienophile $\to$ 环己烯产物）
反应过程中双烯体和亲双烯体的取代基相对构型保持不变（顺式加成原理）。

识别方法：D-A 产物可通过逆 D-A 反应验证结构（加热分解回原料）。
第 11 页

炔烃反应总结

1) 催化还原

不同条件下的还原

完全还原：$\text{-C}\equiv\text{C-} \xrightarrow{\text{H}_2/\text{Pt}} \text{-CH}_2\text{CH}_2\text{-}$（烷烃）

部分还原（顺式烯烃）：

$\text{-C}\equiv\text{C-} \xrightarrow{\text{Lindlar}} \text{顺式 C=C}$（顺式）

$\text{Lindlar}$ 催化剂 = $\text{Pd/CaCO}_3$（被 $\text{PbO(Ac)}_2$ 和喹啉毒化的 Pd）

部分还原（反式烯烃）：

$\text{-C}\equiv\text{C-} \xrightarrow{\text{Na/NH}_3(\text{l})} \text{反式 C=C}$（反式，Birch 还原）

重要：Lindlar 催化剂给顺式，$\text{Na/NH}_3(\text{l})$ 给反式。这是必考知识点！

2) 亲电加成

卤素加成

$\text{R-C}\equiv\text{C-H} + \text{X}_2 \to \text{R-CX=CHX}$（反式加成，一分子）

$\text{R-CX=CHX} + \text{X}_2 \to \text{R-CX}_2\text{-CHX}_2$（四卤代物）

HX 加成

$\text{R-C}\equiv\text{CH} + \text{HX} \to$ 一卤代烯烃（遵循马氏规则，反马氏加成同 Markovnikov 规则）

继续加成得到 gem-二卤代物（Geminal，同碳双卤）。

水合反应（马氏加成 → 酮）

$\text{R-C}\equiv\text{CH} + \text{H}_2\text{O} \xrightarrow{\text{HgSO}_4/\text{H}_2\text{SO}_4} [\text{烯醇}] \xrightarrow{\text{互变}} \text{酮}$

机理：马氏加成 → 烯醇中间体 → 酮式-烯醇式互变异构（Keto-Enol Tautomerism）

乙炔水合得到乙醛（唯一例外）；其他端炔水合得到甲基酮（$\text{RCOCH}_3$）。

硼氢化-氧化（反马氏 → 醛）

$\text{R-C}\equiv\text{CH} \xrightarrow{\text{1) (sia)}_2\text{BH}} \xrightarrow{\text{2) H}_2\text{O}_2/\text{NaOH}} \text{R-CH}_2\text{-CHO}$（醛）

反马氏加成产物，B 加到末端碳上。

互补性记忆：
水合（$\text{HgSO}_4$）：马氏 → 酮
硼氢化-氧化：反马氏 → 醛
两种方法可分别将端炔转化为酮或醛！

3) 亲核加成

HCN 加成

$\text{R-C}\equiv\text{CH} + \text{HCN} \to \text{R-C(=CH}_2\text{)-CN}$

产物为 $\alpha,\beta$-不饱和腈。

ROH 加成

$\text{R-C}\equiv\text{CH} + \text{ROH} \xrightarrow{\text{NaOR}} \text{R-C(=CH}_2\text{)-OR}$（乙烯基醚）

(5) 亲核取代（碳链增长）

炔钠的亲核取代（$\text{S}_\text{N}2$ 反应）

第一步 — 生成炔负离子：$\text{RC}\equiv\text{CH} \xrightarrow{\text{NaNH}_2} \text{RC}\equiv\text{C}^-\text{Na}^+$

第二步 — $\text{S}_\text{N}2$ 取代：$\text{RC}\equiv\text{C}^-\text{Na}^+ + \text{R'X} \to \text{RC}\equiv\text{CR'} + \text{NaX}$

最终实现碳碳键的形成和碳链增长。

炔基格氏试剂路线

$\text{RC}\equiv\text{CH} \xrightarrow{\text{R'MgX}} \text{RC}\equiv\text{CMgX} + \text{R'H}$

$\text{RC}\equiv\text{CMgX} \xrightarrow{\text{R''X}} \text{RC}\equiv\text{CR''}$

也可通过 $\text{Ag(NH}_3)_2^+$ 或 $\text{Cu(NH}_3)_2^+$ 来检验是否为末端炔烃（沉淀法）。

亲核取代反应注意事项：
(1) 只有伯卤代烃（$\text{R'CH}_2\text{X}$）适合与炔钠进行 $\text{S}_\text{N}2$ 反应
(2) 仲卤代烃和叔卤代烃主要发生消除反应（E2），不适用
(3) 反应在极性非质子溶剂（如 DMF, DMSO）中效果更好

4) 金属炔化物

碱性条件选择

$\text{R-C}\equiv\text{CH} + \text{NaOH} \to$ 不反应（$\text{OH}^-$ 碱性不够强，$\text{p}K_a$ 15.7 vs 炔 $\text{p}K_a$ 25）

$\text{R-C}\equiv\text{CH} + \text{NaNH}_2 \to \text{RC}\equiv\text{CNa} + \text{NH}_3$（$\text{NH}_2^-$ 碱性足够，$\text{p}K_a$ 38）

端炔检验（鉴别反应）

$\text{RC}\equiv\text{CH} + [\text{Ag(NH}_3)_2]^+ \to \text{RC}\equiv\text{CAg}\downarrow$（白色沉淀，炔化银）

$\text{RC}\equiv\text{CH} + [\text{Cu(NH}_3)_2]^+ \to \text{RC}\equiv\text{CCu}\downarrow$（红棕色沉淀，炔化亚铜）

安全：干燥的金属炔化物受撞击或加热会爆炸！实验后须用稀 $\text{HNO}_3$ 分解。

$\text{共轭二烯}$ 的亲电加成包括 1,2-加成 和 1,4-加成：

  • 1,4-加成：加在 1,4-位上，中间形成新双键（热力学控制，高温有利）

(5) 聚合反应

合成橡胶

二烯烃的聚合

1,3-丁二烯聚合：

$n\text{CH}_2\text{=CH-CH=CH}_2 \to \text{[-CH}_2\text{-CH=CH-CH}_2\text{-]}_n$

以 1,4-加成 聚合为主，形成顺丁橡胶。

异戊二烯聚合（天然橡胶）

$\text{CH}_2\text{=C(CH}_3\text{)-CH=CH}_2 \to \text{[-CH}_2\text{-C(CH}_3\text{)=CH-CH}_2\text{-]}_n$

全顺式 1,4-加成聚合 = 天然橡胶（顺式聚异戊二烯）

氯丁二烯聚合（氯丁橡胶）

$\text{CH}_2\text{=CCl-CH=CH}_2 \to \text{[-CH}_2\text{-CCl=CH-CH}_2\text{-]}_n$

产物为氯丁橡胶（Neoprene），耐油性好。

共聚物

丁苯橡胶 = 1,3-丁二烯 + 苯乙烯共聚

$n\text{CH}_2\text{=CH-CH=CH}_2 + m\text{C}_6\text{H}_5\text{CH=CH}_2 \to$ 丁苯橡胶

丁腈橡胶 = 1,3-丁二烯 + 丙烯腈共聚

炔烃的化学性质汇总

(1) 催化氢化

  • 完全加氢：$\text{H}_2/\text{Pt}$
  • 部分还原（顺式）：$\text{Lindlar}$ 催化剂 → 顺式烯烃
  • 部分还原（反式）：$\text{Na/NH}_3(\text{l})$ → 反式烯烃
重要：Lindlar 给顺式，$\text{Na/NH}_3$ 给反式。原因分别是：顺式加氢与溶剂化电子还原（anti 加成）。

(2) 亲电加成

$\text{-C}\equiv\text{C-}$ 亲电加成总结：

试剂产物备注
$\text{X}_2$（1 eq）反式二卤代烯烃 $\text{R-CX=CHX}$反式加成（anti）
$\text{X}_2$（2 eq）四卤代物 $\text{R-CX}_2\text{-CHX}_2$继续加成
$\text{HX}$（1 eq）卤代烯烃（马氏加成）遵循 Markovnikov 规则
$\text{HX}$（2 eq）gem-二卤代物（同碳）两分子 HX 加到同一碳
$\text{H}_2\text{O}/\text{HgSO}_4/\text{H}_2\text{SO}_4$酮（经烯醇互变）马氏加成
$\text{BH}_3 \to \text{H}_2\text{O}_2/\text{OH}^-$醛 $\text{R-CH}_2\text{CHO}$反马氏加成

3) 亲核加成

亲核试剂加成到三键上：

  • $+ \text{HCN} \to \text{R-C(=CH}_2\text{)-CN}$（丙烯腈类）
  • $+ \text{ROH} \to \text{R-C(=CH}_2\text{)-OR}$（乙烯基醚）
  • $+ \text{RCOOH} \to$ 乙烯基酯

亲核加成活性：炔烃 > 烯烃（sp 碳电负性大，三键碳有部分正电荷）。

4) 金属炔化物 / 氧化 / 取代

金属炔化物

$\text{RC}\equiv\text{CH} \xrightarrow{\text{NaNH}_2} \text{RC}\equiv\text{CNa} \xrightarrow{\text{R'X}} \text{RC}\equiv\text{CR'}$（亲核取代，碳链增长）

末端炔检验

$\text{RC}\equiv\text{CH} + \text{[Ag(NH}_3\text{)}_2\text{]}^+ \to \text{RC}\equiv\text{CAg}\downarrow$（白色）

$+ \text{[Cu(NH}_3\text{)}_2\text{]}^+ \to \text{RC}\equiv\text{CCu}\downarrow$（红棕色）

氧化断裂

$\text{RC}\equiv\text{CR'} \xrightarrow{\text{KMnO}_4\text{ 或 O}_3} \text{RCOOH} + \text{R'COOH}$

$\text{RC}\equiv\text{CH} \xrightarrow{\text{KMnO}_4\text{ 或 O}_3} \text{RCOOH} + \text{CO}_2$

末端炔 → 羧酸 + $\text{CO}_2$；内炔 → 两分子羧酸。可用于确定三键位置。

5) 金属炔化物 / 取代

$\text{RC}\equiv\text{CH} \xrightarrow{\text{NaNH}_2} \text{RC}\equiv\text{CNa} \xrightarrow{\text{R'X}} \text{RC}\equiv\text{CR'}$（亲核取代，碳链增长）

$\text{RCOH} \xrightarrow{[\text{Ag(NH}_3)_2]^+} \text{RC}\equiv\text{CAg}\downarrow$（鉴别末端炔）

$[\text{Cu(NH}_3)_2\text{Cl}]$：$\text{RC}\equiv\text{CH} \to \text{RC}\equiv\text{CCu}\downarrow$（红棕色，鉴别末端炔）

炔烃化学性质总结：
(1) 亲电加成活性：炔烃 < 烯烃（三键的 $\pi$ 电子被束缚更紧，不易给出电子）
(2) 亲核加成活性：炔烃 > 烯烃（sp 碳电负性大）
(3) 末端炔的 C-H 有酸性（$\text{p}K_a \approx 25$），可与 $\text{NaNH}_2$ 等强碱反应
(4) 末端炔可被银氨、铜氨试剂检验（形成不溶性金属炔化物沉淀）
