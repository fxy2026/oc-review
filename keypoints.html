专题一：不饱和度(DBE) — 解题第一步 必考

不饱和度公式
$$\text{DBE} = \frac{2C + 2 + N - H - X}{2}$$

其中 C = 碳数，N = 氮数，H = 氢数，X = 卤素数。氧和硫不参与计算。

DBE含义可能结构
0饱和烷烃、醇、醚、卤代烃
11个双键或1个环烯烃、环烷烃、醛/酮
22个不饱和单元炔烃、二烯烃、两个环……
4高度不饱和很可能含苯环！（苯 = 3双键 + 1环 = 4）
常见陷阱：
  • $\text{C}_8\text{H}_{10}$ → DBE = 4 → 含苯环 → 四种二甲苯/乙苯异构体
  • 含 N 时别忘 $+N$：$\text{C}_6\text{H}_7\text{N}$ → DBE = $\frac{12+2+1-7}{2} = 4$ → 苯胺（不是饱和胺！）
  • 含 O 时 O 不参与计算：$\text{C}_6\text{H}_{12}\text{O}$ → DBE = 1 → 可能是醛/酮/环醇

专题二：碳正离子重排 — 最常丢分的考点 必考陷阱

何时考虑重排？

凡是经过碳正离子中间体的反应，都必须检查重排可能：

  • SN1 取代
  • E1 消除
  • HX 加成到烯烃（Markovnikov，经碳正离子）
  • 醇的脱水（酸催化，经碳正离子）
  • Friedel-Crafts 烷基化（酰基化不重排！）
重排规则
  • 1,2-氢迁移：相邻碳上的 H 迁移到碳正离子位
  • 1,2-甲基迁移（Wagner-Meerwein重排）：相邻碳上的 $\text{CH}_3$ 迁移
  • 方向：总是从低级 → 高级（$1° \to 2° \to 3°$，越稳定越好）
  • 还可发生环扩大/缩小重排
经典例题：新戊基溴在乙醇中的溶剂解 $$\text{(CH}_3\text{)}_3\text{CCH}_2\text{Br} \xrightarrow{\text{EtOH}} \text{?}$$

直接产生 $1°$ 碳正离子极不稳定 → 发生 1,2-甲基迁移 → 变为 $3°$ 碳正离子 → 生成2-甲基-2-丁烯（E1消除）或叔碳醚（SN1取代）

F-C 烷基化的重排陷阱：

用 $\text{CH}_3\text{CH}_2\text{CH}_2\text{Cl}/\text{AlCl}_3$ 做 F-C 烷基化，正丙基碳正离子 → 重排为异丙基碳正离子 → 得到的是异丙基苯（枯烯）而不是正丙基苯！

解决方案：用 F-C 酰基化（酰基正离子不重排）+ Clemmensen 还原

F-C 烷基化 vs 酰基化
烷基化 $\text{RCl/AlCl}_3$酰基化 $\text{RCOCl/AlCl}_3$
中间体碳正离子 $\text{R}^+$酰基正离子 $\text{RCO}^+$
重排？会重排！不重排（共振稳定）
多取代？会多烷基化（产物被活化）停在一取代（产物被钝化）
底物限制不能用于钝化苯环不能用于强钝化苯环

实战判断流程
  1. 看底物：
    • $\text{CH}_3\text{X}$ 或 $1°$：主要 SN2（强 Nu）或 E2（强碱+高温）
    • $3°$：主要 SN1/E1（弱 Nu/溶剂）或 E2（强碱）；不发生 SN2

  2. 看试剂：

    • 强碱 + 强亲核（如 $\text{EtO}^-$, $\text{OH}^-$）→ SN2 和 E2 竞争
    • 弱碱 + 弱亲核（如 $\text{H}_2\text{O}$, $\text{EtOH}$）→ SN1/E1
  3. 看温度：升温有利于消除（$\Delta S > 0$），降温有利于取代

SN2 不适用的底物（考试常设陷阱）：
  • $3°$ 碳 → 位阻太大
  • 桥头碳 → 无法 Walden 翻转（背面被环堵住）

  • 芳基卤 $\text{ArX}$ → $sp^2$ 碳不发生 SN2（除非有强吸电子基+高温）
E2 的立体化学要求：H 和 X 必须处于反式共平面（anti-periplanar）。
  • 开链化合物：通过旋转可达到 anti 构象
  • 环己烷：H 和 X 必须都处于直立键（axial）且trans 关系

合成策略

多取代苯的引入顺序

核心原则：先引入的基团决定后引入基团的位置

  • 目标：对位二取代 → 先引入邻对位定位基
  • 目标：间位二取代 → 先引入间位定位基
合成对硝基甲苯：先 F-C 烷基化得甲苯（$\text{-CH}_3$ 邻对位定位），再硝化 → 对位产物

若先硝化：$\text{-NO}_2$ 间位定位 + 钝化苯环 → F-C 烷基化不能进行！

四大合成技巧
  1. 磺酸基占位法：利用磺化可逆，占住不需要的位置 → 反应完脱去
    例：合成间溴甲苯 → 先磺化对位 → 再溴代 → 最后脱磺酸基
  2. 先酰基化后还原：解决 F-C 烷基化的重排问题
    $\text{ArH} \xrightarrow{\text{RCOCl/AlCl}_3} \text{ArCOR} \xrightarrow{\text{Clemmensen}} \text{ArCH}_2\text{R}$
  3. $\text{-NH}_2$ 保护为 $\text{-NHCOCH}_3$：避免氨基被氧化或过度活化
  4. 氧化时机：侧链 $\text{KMnO}_4$ 氧化必须有 $\alpha$-H → $t$-Bu 不被氧化
F-C 反应的三大限制：
  • 不能用于强钝化苯环（含 $\text{-NO}_2$, $\text{-COR}$, $\text{-COOH}$ 等）
  • 烷基化会多取代（产物比底物更活泼）
  • 烷基化会重排（碳正离子中间体）

专题七：Diels-Alder 反应 高频

[4+2] 环加成

双烯体（4π，必须 s-cis 构象）+ 亲双烯体（2π，含吸电子基）→ 六元环

条件要求：
  • 双烯体：共轭二烯，必须能采取 s-cis 构象（s-trans 无法反应）
  • 亲双烯体：含吸电子基的烯烃（如 $\text{-CHO}$, $\text{-COR}$, $\text{-COOH}$, $\text{-CN}$, $\text{-NO}_2$）→ 效果更好
  • 电子效应：双烯体上有给电子基、亲双烯体上有吸电子基 → 反应加速
立体选择性（两条规则）
  1. 顺式(cis)规则：双烯体和亲双烯体上取代基的相对构型在产物中保持
    • 双烯体上的 cis 取代基 → 产物中仍是 cis
    • 亲双烯体上的 cis 取代基 → 产物中仍是 cis
  2. endo 规则：endo 产物为主产物（动力学有利，二级轨道作用）
不能做 D-A 反应的二烯烃：
  • 被锁定在 s-trans 构象的二烯无法反应（如某些环状二烯）
  • 反应是可逆的：高温有利于逆反应

专题九：Grignard 试剂 — 合成题核心武器 必考

制备与性质

$\text{RX} + \text{Mg} \xrightarrow{\text{无水乙醚}} \text{RMgX}$

严格无水无氧！不能有活泼氢（$\text{-OH}$, $\text{-NH}$, $\text{-SH}$, $\text{-COOH}$, $\text{-C}\!\equiv\!\text{CH}$）

底物反应产物碳数变化
$\text{HCHO}$（甲醛）$\text{RMgX} + \text{HCHO}$伯醇 $\text{RCH}_2\text{OH}$$+1$
$\text{R'CHO}$（醛）$\text{RMgX} + \text{R'CHO}$仲醇 $\text{RR'CHOH}$增碳
$\text{R'}_2\text{CO}$（酮）$\text{RMgX} + \text{R'}_2\text{CO}$叔醇 $\text{RR'}_2\text{COH}$增碳

合成题常见增碳策略：
  • 增 1 个碳：$\text{RMgX} + \text{HCHO}$ → 伯醇，或 $\text{RMgX} + \text{CO}_2$ → 酸
  • 增 2 个碳：$\text{RMgX} + $ 环氧乙烷 → 伯醇，或 $\text{RC}\!\equiv\!\text{CNa} + \text{R'X}$ → 增碳
  • 增 3+ 个碳：$\text{RMgX} + \text{R'CHO/R'}_2\text{CO}$
Grignard 合成题三大禁忌：
  1. 分子内不能含活泼氢基团（$\text{-OH}$, $\text{-NH}_2$, $\text{-COOH}$, $\text{-C}\!\equiv\!\text{CH}$）→ 否则 Grignard 试剂会先和自身反应
  2. 必须无水条件 → 不能在水溶液中制备
  3. 产物水解前不能接触水 → 最后一步才加 $\text{H}_3\text{O}^+$ 水解

Corey-House 合成（偶联增碳）

反应方程
$$\text{2 RLi} + \text{CuI} \to \text{R}_2\text{CuLi}$$ $$\text{R}_2\text{CuLi} + \text{R'X} \to \text{R-R'} + \text{RCu} + \text{LiX}$$

$\text{R'X}$：伯卤代烃或仲卤代烃（叔不行）；可用于精确构建碳骨架

外消旋体等量对映体的混合物$[\alpha] = 0$可以拆分
内消旋体(meso)含手性碳但有对称面的单一化合物$[\alpha] = 0$不能拆分
非对映体立体异构但非镜像关系不同物理性质不同可分离
三大易混陷阱：
  1. 有手性碳 $\neq$ 有手性：meso-酒石酸有 2 个手性碳但分子内有对称面 → 无旋光性
  2. 无手性碳也可有手性：丙二烯型（累积二烯）、联苯轴手性、螺环手性
  3. $[\alpha] = 0$ 不等于一定是 meso：外消旋混合物也是 $[\alpha] = 0$，但可以拆分
ee% 和旋光纯度
$$ee\% = \frac{[\alpha]_{\text{obs}}}{[\alpha]_{\text{pure}}} \times 100\%$$ $$ee\% = \frac{|R - S|}{R + S} \times 100\%$$

$ee = 100\%$：纯对映体；$ee = 0\%$：外消旋体

例题：某样品 $[\alpha]_{\text{obs}} = +6.0°$，纯品 $[\alpha]_{\text{pure}} = +12.0°$

$ee\% = 6.0/12.0 \times 100\% = 50\%$

含 $75\%$ R 构型和 $25\%$ S 构型（因为多出的 50% 全是 R，剩余 50% 是外消旋）

异构体数目

  • $n$ 个不同手性碳 → 最多 $2^n$ 个立体异构体
  • 有相同取代的手性碳 → 少于 $2^n$（因为有 meso 体）
  • 例：$n = 2$，两个手性碳取代基相同 → 3 个异构体（1对对映体 + 1个 meso）而非 4 个

推结构题解题框架
  1. 算不饱和度 → 判断有无双键/苯环
  2. 分析光谱/化学性质 → 确定官能团
  3. 分析氧化/还原产物 → 确定碳骨架
  4. 验证 → 检查分子式、不饱和度、所有反应是否吻合
合成题逆合成分析法
  1. 分析目标分子：官能团类型、碳数、取代基位置
  2. 逆推关键键：找出目标分子中哪些 C-C 键/C-X 键可以用已学反应构建
  3. 增碳方法清单：
    • Grignard + $\text{HCHO}$（$+1C$）/ $\text{CO}_2$（$+1C$）/ 环氧乙烷（$+2C$）
    • 炔钠 + 伯 RX（$+nC$）
    • Corey-House 偶联
    • F-C 烷基化/酰基化（苯环上接烃链）
  4. 检查每一步：条件是否合理？会不会重排？立体化学是否正确？
