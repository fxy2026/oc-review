第8章 醇、酚、醚

一、醇 (Alcohols)

8.1 醇的分类与命名

按羟基所连碳的级别分类
类型结构特征示例
伯醇(1°)−OH连在伯碳上（RCH2OH）乙醇 CH3CH2OH
仲醇(2°)−OH连在仲碳上（R2CHOH）异丙醇 (CH3)2CHOH
叔醇(3°)−OH连在叔碳上（R3COH）叔丁醇 (CH3)3COH
命名规则（系统命名法）
  1. 选含−OH的最长碳链为主链
  2. 从离−OH最近的一端开始编号
  3. 命名为"某醇"，标注−OH位次

例：CH3CH(OH)CH2CH3 → 2-丁醇

多元醇：乙二醇(HOCH2CH2OH)、丙三醇/甘油(HOCH2CHOHCH2OH)等，命名时标出所有−OH的位次。

8.2 醇的物理性质

氢键与物理性质的关系
  • 沸点：醇能形成分子间氢键(O−H···O)，沸点远高于同碳数烷烃和醚
  • 溶解性：低级醇(C1~C3)能与水混溶（与水形成氢键）；随碳数增加，疏水性增强，溶解度下降
  • 碳数关系：碳数越多 → 沸点越高、水溶性越差
  • 同碳数异构体：直链醇 > 支链醇（沸点），因支链减少了分子间接触面积
化合物分子量沸点(°C)溶解性
CH3OH（甲醇）3265与水混溶
C2H5OH（乙醇）4678与水混溶
C3H7OH（正丙醇）6097与水混溶
C4H9OH（正丁醇）74118微溶
CH3OCH3（二甲醚）46−25微溶
对比：乙醇(MW=46, bp=78°C) vs 二甲醚(MW=46, bp=−25°C)——同分异构体，但醇能形成氢键，沸点高100多度！

8.3 醇的化学性质

8.3.1 醇的酸性

醇的酸碱性

醇的pKa ≈ 16−19，属于极弱酸（比水弱，水pKa=15.7）

溶液中酸性顺序：H2O > CH3OH > 1°ROH > 2°ROH > 3°ROH

原因：烷基的+I供电子效应增加O上的电子密度，不利于O−H键的离解

气相酸性相反！ 气相中：3°ROH > 2°ROH > 1°ROH > CH3OH
原因：气相无溶剂化效应，烷基的极化效应能稳定体积较大的负离子。
考试一般考溶液相顺序！
醇与活泼金属反应

$$2\text{ROH} + 2\text{Na} \rightarrow 2\text{RONa} + \text{H}_2\uparrow$$

  • 反应活性：CH3OH > 1°ROH > 2°ROH > 3°ROH（酸性越强，反应越快）
  • 产物醇钠(RONa)是强碱，遇水分解：RONa + H2O → ROH + NaOH
  • 可鉴别活泼氢的存在（产生H2气体）

8.3.2 醇的亲核取代反应

与氢卤酸反应 (ROH + HX → RX + H2O)

反应活性：

  • 醇的活性：3° > 2° > 1° > CH3OH（碳正离子稳定性决定）
  • HX的活性：HI > HBr > HCl（卤离子亲核性与H−X键强度）

机理：3°醇走SN1（经碳正离子），1°醇走SN2

Lucas试剂鉴别醇（ZnCl2/浓HCl）——考试高频考点！
醇的类型现象反应时间机理
叔醇(3°)立即浑浊<1 minSN1，快速形成稳定3°碳正离子
仲醇(2°)5分钟内浑浊2~5 minSN1/SN2边界
伯醇(1°)不反应/需加热很长/不反应SN2慢

浑浊原因：生成的卤代烃RCl不溶于反应液（ZnCl2水溶液），形成浑浊液滴。

与PBr3反应

$$3\text{ROH} + \text{PBr}_3 \rightarrow 3\text{RBr} + \text{H}_3\text{PO}_3$$

  • 机理：SN2
  • 立体化学：构型翻转（Walden翻转）
  • 适用于1°和2°醇，条件温和，无重排
与SOCl2反应（氯化亚砜法）

$$\text{ROH} + \text{SOCl}_2 \rightarrow \text{RCl} + \text{SO}_2\uparrow + \text{HCl}\uparrow$$

立体化学取决于条件：

条件机理立体结果
加吡啶(碱)SN2构型翻转
不加碱SNi(内返)构型保持
SNi(内返)机理：ROH先与SOCl2形成氯代亚硫酸酯中间体(ROSOCl)，然后Cl−从同侧离去基团的位置发生分子内递交（SO2离去的同时Cl−从同面取代），构型保持。加吡啶时，吡啶夺取HCl使Cl−游离，从背面进攻（SN2），构型翻转。
SOCl2法的优势：副产物SO2和HCl均为气体，自动逸出反应体系，产物纯净，后处理简单。工业上常用。

8.3.3 醇的脱水反应

分子内脱水 → 烯烃

条件：浓H2SO4, 170°C（高温有利于消除）

规则：遵循Zaitsev规则——优先脱去含氢较少的β碳上的H，生成取代度更高的烯烃（更稳定）

反应活性：3°醇 > 2°醇 > 1°醇（碳正离子稳定性决定）

特别注意（考试原题）：苄基醇(ArCH2CH2OH)或烯丙基醇脱水特别容易，因为产物生成与苯环/双键共轭的烯烃（如：PhCH2CH2OH脱水→苯乙烯PhCH=CH2，共轭稳定）。

分子间脱水 → 醚

条件：浓H2SO4, 140°C（较低温度有利于取代）

$$2\text{ROH} \xrightarrow{H_2SO_4,\,140\,{}^{\circ}\text{C}} \text{R-O-R} + \text{H}_2\text{O}$$

仅适用于伯醇生成对称醚（2°和3°醇在此条件下倾向脱水成烯烃）

碳正离子重排！（极其重要）
脱水反应经过碳正离子中间体，可能发生：
• 1,2-氢迁移：2°碳正 → 3°碳正
• 1,2-甲基迁移：2°碳正 → 3°碳正
结果：产物的双键位置可能与预期不同！
例：3,3-二甲基-2-丁醇脱水 → 2,3-二甲基-2-丁烯（经甲基迁移重排）
温度控制是关键：
• 140°C → 分子间脱水 → 醚（动力学控制，SN2）
• 170°C → 分子内脱水 → 烯烃（E1消除）
同一底物、同一催化剂、不同温度 → 完全不同的产物！

8.3.4 醇的氧化反应

不同醇的氧化产物
底物氧化剂产物备注
1°醇KMnO4, K2Cr2O7醛 → 羧酸强氧化剂不能停在醛
1°醇PCC(氯铬酸吡啶鎓盐)醛温和氧化剂，可停在醛！
2°醇KMnO4, K2Cr2O7, PCC酮酮不继续被氧化
3°醇一般条件不易氧化无α-H可脱，需强烈条件断C-C键
PCC (Pyridinium Chlorochromate) = 氯铬酸吡啶鎓盐
• 在CH2Cl2中进行，无水条件
• 将1°醇氧化为醛而不继续氧化到羧酸
• 这是合成醛的重要方法！（Jones试剂CrO3/H2SO4也常用但不能停在醛）
为什么3°醇不易氧化？
氧化醇的机理需要脱去与C−OH相连碳上的α-H。3°醇的α碳上无H，无法经历正常氧化机理。强烈条件下可断裂碳骨架（无合成价值）。

8.3.4b 邻二醇的氧化断裂——HIO4（高碘酸）

高碘酸(HIO4)氧化切断邻二醇——期中考试重点！

反应通式：邻二醇 + HIO4 → 醛/酮 + HIO3

HIO4专一性地切断相邻两个含OH碳之间的C−C键，将每个C−OH氧化为C=O：

底物结构产物示例
−CHOH−CHOH−2个醛乙二醇→2HCHO；2,3-丁二醇→2CH3CHO
R2COH−CHOH−酮 + 醛3-甲基-2,3-丁二醇→丙酮+乙醛
R2COH−CR2OH2个酮频哪醇(2,3-二甲基-2,3-丁二醇)→2丙酮
−CHOH−CHOH−CHOH−中间碳→甲酸(HCOOH)丙三醇→2HCHO+HCOOH
考试常考应用：
• 结构推断：通过HIO4氧化产物推断原始二醇结构（逆推法）
• 鉴别：邻二醇 + HIO4消耗碘酸，可用来区分邻二醇和非邻二醇
• 与KMnO4配合：烯烃→KMnO4稀冷→邻二醇→HIO4→醛/酮（等效于臭氧化分解）
经典例题（期中考试原题类型）：
化合物A(C5H12O)，CrO3/H2SO4氧化得B(酮)；A脱水主产物为烯烃C；C + 稀冷KMnO4→D(邻二醇)；D + HIO4→醛E + 酮F。求A-F。
解答

• A能被CrO3氧化为酮B → A是2°醇
• HIO4切断D得一个醛+一个酮 → 邻二醇两侧碳不等价（一侧仲C，一侧叔C）
• 综合推断：A = 3-甲基-2-丁醇 (CH3)2CHCH(OH)CH3

B = 3-甲基-2-丁酮 (CH3)2CHCOCH3
C = 2-甲基-2-丁烯 (CH3)2C=CHCH3（Saytzeff产物）
D = 2-甲基-2,3-丁二醇 (CH3)2C(OH)CH(OH)CH3
E = 乙醛 CH3CHO（来自-CHOH端）
F = 丙酮 (CH3)2CO（来自-C(CH3)2OH端）

记忆口诀：HIO4切邻二醇，C-C断裂变C=O；一级醇得醛，二级得酮，三级碳不含OH不参与。

8.3.5 醇的酯化反应

Fischer酯化反应

$$\text{ROH} + \text{R'COOH} \xleftrightharpoons{\text{H}^+,\Delta} \text{R'COOR} + \text{H}_2\text{O}$$

  • 可逆反应，需酸催化（浓H2SO4）
  • 提高产率的方法：过量一种反应物、除去水（分水器）
  • 机理：酸催化加成-消除（羧酸被质子化 → 醇亲核进攻羰基 → 失水）
  • 反应活性：CH3OH > 1°ROH > 2°ROH > 3°ROH（位阻影响亲核进攻）
同位素标记实验证明机理：
用R18OH与R'COOH反应，18O出现在酯中(R'CO18OR)而非水中。
证明：断裂的是羧酸的C−OH键（酰氧键），而非醇的C−OH键。
即：醇的O保留在酯中。

二、醚 (Ethers)

8.4 醚的结构与命名

醚的结构

通式：R−O−R'（氧上连两个碳基团）

  • 简单醚（对称醚）：两个R基相同，如 CH3OCH3（二甲醚）
  • 混合醚（不对称醚）：两个R基不同，如 CH3OC2H5（甲乙醚）
  • 环醚：氧在环中，如 环氧乙烷、四氢呋喃(THF)、1,4-二氧六环

8.5 醚的物理性质

与醇的对比
  • 无分子间氢键（O上无活泼H）→ 沸点远低于同碳数醇
  • 沸点接近同碳数烷烃
  • 乙醚(bp 34.6°C)可做优良溶剂：能溶解大多数有机物，易挥发易回收
  • 可作为氢键受体（O有孤对电子）→ 低级醚微溶于水
乙醚作溶剂的优势：沸点低、惰性强、溶解能力好。但需注意：极易着火（闪点−45°C）且比空气重，蒸气沿台面流动引火！实验室使用乙醚必须远离明火。

8.6 醚的化学性质

Williamson醚合成法

$$\text{RONa} + \text{R'X} \xrightarrow{SN2} \text{R-O-R'} + \text{NaX}$$

  • 机理：SN2——醇钠(RO−)作为亲核试剂进攻卤代烃
  • 只能用伯卤代烃(1°RX)！仲卤和叔卤会发生E2消除而非取代
  • 制备不对称醚时，选择策略：将较大基团放在醇钠端，较小基团(CH3X、1°RX)放在卤代烃端
Williamson合成的限制（考试易错）：
• 正确：(CH3)3CONa + CH3I → (CH3)3COCH3 (叔丁基甲醚)
• 错误：CH3ONa + (CH3)3CBr → 不行！（叔卤代烃会E2消除得到异丁烯）
记住：卤代烃端必须是伯卤（SN2需要位阻小的碳）
酸催化开裂（醚键断裂）

$$\text{R-O-R'} + \text{HI}(\text{过量}) \rightarrow \text{RI} + \text{R'I} + \text{H}_2\text{O}$$

  • 需要强酸HI或HBr（HCl一般不行）
  • 机理：质子化醚 → I−亲核进攻 → 断裂C−O键
  • 不对称醚开裂选择性：I−优先进攻碳级较低的碳(SN2较易)或碳正离子更稳定的碳(SN1)
过氧化物的形成

醚在空气中久置会缓慢氧化生成过氧化物（R−O−O−R'）

  • 过氧化物受热爆炸——实验室重大安全隐患！
  • 使用前必须检测：加KI溶液变黄/棕色表示含有过氧化物
  • 解决方法：加Na2SO3还原分解，或过Na/Al2O3柱
实验安全要点：乙醚、异丙醚、THF等醚类试剂，开瓶后应记录日期。放置超过6个月或瓶壁有结晶的醚绝对不能蒸馏，有爆炸危险！

8.7 环氧化合物

环氧化合物的合成

烯烃 + 过氧酸 → 环氧化物

$$\text{R}_2\text{C=CR}_2 + m\text{CPBA} \rightarrow \text{环氧化物}$$

  • mCPBA = 间氯过氧苯甲酸（常用的过氧酸）
  • 反应是顺式加成（syn addition），立体专一性
  • 烯烃取代基越多，反应越快（富电子烯烃更易被氧化）
环氧化物的开环反应（核心考点！）
条件亲核试剂进攻位置区域选择性立体化学
酸催化（H3O+, ROH/H+, HX）取代度较高的碳类似Markovnikov反式开环(anti)
碱/亲核试剂（OH−, RO−, RMgX）位阻较小的碳反Markovnikov反式开环(anti)
酸催化开环的解释：
酸性条件下，环氧O被质子化，形成类似碳正离子的过渡态。亲核试剂倾向进攻正电荷更多的碳（取代度高的碳能更好地稳定部分正电荷）→ 结果类似Markovnikov方向。

碱催化开环的解释：
碱性条件下，强亲核试剂直接SN2进攻。SN2对位阻敏感，优先进攻位阻小的碳 → 反Markovnikov方向。
Grignard试剂与环氧乙烷 → 增二碳的伯醇

$$\text{RMgBr} + \underset{\text{环氧乙烷}}{\boxed{\text{CH}_2\text{-O-CH}_2}} \xrightarrow{\text{H}_3\text{O}^+} \text{RCH}_2\text{CH}_2\text{OH}$$

  • 格氏试剂(RMgBr)作为碳亲核试剂进攻环氧位阻小的碳(SN2)
  • 结果：碳链增加2个碳，得到伯醇
  • 这是合成碳链增长两个碳的伯醇的经典方法！
环氧开环的合成价值：
• 酸性开环(H2O/H+) → 反式1,2-二醇（邻二醇）
• 碱性开环(RO−) → β-烷氧基醇
• HX开环(酸催化) → β-卤代醇（X加到取代度高的碳上，类Markovnikov方向）
• RMgX → 增碳得伯醇

三、酚 (Phenols)

8.8 酚的酸性

酚的酸性（pKa≈10）

酚(ArOH)的酸性比醇强106~107倍！

$$\text{ArOH} \rightleftharpoons \text{ArO}^- + \text{H}^+\quad pK_a \approx 10$$

原因：酚氧负离子(ArO−)的负电荷通过p-π共轭离域到苯环上（5个共振结构），大大稳定了共轭碱。

取代基对酚酸性的影响
取代基类型效应对酸性的影响实例
吸电子基(−NO2, −Cl等)稳定酚氧负离子增强酸性对硝基苯酚 pKa=7.15
供电子基(−CH3, −OCH3等)不稳定酚氧负离子降低酸性对甲基苯酚 pKa=10.26

位置效应：对位 > 邻位 > 间位（对位和邻位能通过共振直接参与稳定）

酸性比较（从强到弱）：
羧酸(pKa≈4~5) > 碳酸(6.4) > 对硝基苯酚(7.15) > 苯酚(10) > 水(15.7) > 醇(16~19) > 乙炔(25)
实用判据：苯酚能溶于NaOH但不能溶于NaHCO3（pKa=10 > H2CO3的pKa=6.4）；羧酸既溶于NaOH也溶于NaHCO3。

8.9 酚的鉴别——FeCl3显色反应

FeCl3显色反应

酚 + FeCl3(aq) → 紫色/蓝色/绿色络合物

  • 酚羟基与Fe3+形成配位络合物
  • 不同酚显色不同：苯酚→紫色；甲酚→蓝色；间苯二酚→深紫色
  • 醇不显色！可用来鉴别酚和醇
  • 烯醇式化合物（如乙酰乙酸乙酯烯醇式）也能显色
鉴别方案：未知物 + FeCl3(aq) → 显色 = 含有酚−OH或烯醇；不显色 = 醇或醚。

8.10 酚的芳环亲电取代

−OH是强活化基、邻对位定位基

酚的−OH通过+C共振效应大大增加苯环电子密度，使EAS反应极易发生。

  • 卤代：苯酚 + Br2(aq)（无需催化剂！）→ 2,4,6-三溴苯酚（白色沉淀）
  • 硝化：需用稀HNO3（浓硝酸会氧化酚），低温
  • 磺化：苯酚低温→邻位磺化产物为主；高温→对位产物为主
苯酚与溴水反应是定性鉴别酚的经典方法：
苯酚 + 3Br2(aq) → 2,4,6-三溴苯酚(白色↓) + 3HBr
注意：不需要Lewis酸催化！因为−OH活化能力太强。这与苯的溴代（需要FeBr3催化）形成对比。

8.11 酚的特殊反应

Kolbe反应（Kolbe-Schmitt反应）

$$\text{苯酚钠} + \text{CO}_2 \xrightarrow{125\,{}^{\circ}\text{C},\;\text{加压}} \text{水杨酸(邻羟基苯甲酸)}$$

  • 工业上用于制备水杨酸（阿司匹林的前体）
  • 机理：CO2作为亲电试剂进攻苯环邻位（苯酚钠的O−强活化）
  • 水杨酸 + 乙酸酐 → 阿司匹林（乙酰水杨酸）
Reimer-Tiemann反应

$$\text{苯酚} + \text{CHCl}_3 + \text{NaOH} \rightarrow \text{水杨醛(邻羟基苯甲醛)}$$

  • 亲电试剂：CHCl3在碱性条件下生成二氯卡宾(:CCl2)
  • 二氯卡宾进攻苯环邻位 → 水解 → 醛基
  • 产物为邻位取代（主要产物）
对比Kolbe与Reimer-Tiemann：
• Kolbe：苯酚钠 + CO2 → 邻羟基苯甲酸（−COOH）
• Reimer-Tiemann：苯酚 + CHCl3/NaOH → 邻羟基苯甲醛（−CHO）
两者都是在邻位引入含氧基团，但产物氧化态不同！

四、习题精练

1. 鉴别题：用简单化学方法鉴别下列化合物：
(A) 1-丁醇   (B) 2-丁醇   (C) 叔丁醇   (D) 正丁醚
解答
解题思路：利用Lucas试剂鉴别伯/仲/叔醇，醚不含OH不反应

第一步：区分醚和醇

加金属钠：A、B、C均放出H2气体（含活泼OH）；D不反应（醚无活泼H）。

第二步：Lucas试剂区分三种醇

将A、B、C分别与ZnCl2/浓HCl（Lucas试剂）混合：

  • C (叔丁醇)：立即浑浊（3°醇，SN1快速反应）
  • B (2-丁醇)：5分钟内浑浊（2°醇）
  • A (1-丁醇)：室温不反应（1°醇，需加热才能反应）
2. 反应产物题：写出下列反应的主要产物，并说明立体化学：
(a) (R)-2-丁醇 + SOCl2（不加吡啶）
(b) (R)-2-丁醇 + SOCl2 + 吡啶
(c) (R)-2-丁醇 + PBr3
解答
解题思路：SOCl2的立体化学取决于是否加碱；PBr3是SN2

(a) (R)-2-丁醇 + SOCl2（无碱）

机理：SNi（内返）→ 构型保持

产物：(R)-2-氯丁烷

(b) (R)-2-丁醇 + SOCl2 + 吡啶

机理：SN2 → 构型翻转

产物：(S)-2-氯丁烷

(c) (R)-2-丁醇 + PBr3

机理：SN2 → 构型翻转

产物：(S)-2-溴丁烷

总结：同一底物，不同试剂/条件，可得到保持或翻转构型的产物。这是有机合成中控制立体化学的关键策略。

3. 合成题：用环氧乙烷和合适的Grignard试剂合成1-己醇。
解答
解题思路：Grignard + 环氧乙烷 → 增2碳的伯醇

目标分析：1-己醇 = CH3(CH2)4CH2OH（6碳伯醇）

环氧乙烷提供2个碳，因此格氏试剂需要提供4个碳。

合成路线：

① CH3CH2CH2CH2Br + Mg/THF → CH3(CH2)3MgBr（正丁基溴化镁）

② CH3(CH2)3MgBr + 环氧乙烷 → [中间体]

③ 水解(H3O+) → CH3(CH2)4CH2OH（1-己醇）

原理：格氏试剂(R−)作为亲核试剂SN2进攻环氧乙烷位阻小的碳，开环后增加了2个碳，末端为−OH（伯醇）。

4. Williamson合成：设计合成叔丁基乙基醚 [(CH3)3C−O−C2H5] 的Williamson合成路线。解释为什么只有一种方案可行。
解答
解题思路：Williamson合成中卤代烃端必须是伯卤代烃

方案一（正确）：

(CH3)3CONa + CH3CH2Br → (CH3)3COCH2CH3

叔丁醇钠 + 溴乙烷(伯卤) → 叔丁基乙基醚

可行！CH3CH2Br是伯卤代烃，能发生SN2。

方案二（错误）：

C2H5ONa + (CH3)3CBr → 不能得到目标产物！

原因：(CH3)3CBr是叔卤代烃，遇到强碱(C2H5O−)会发生E2消除反应：

(CH3)3CBr + C2H5O− → (CH3)2C=CH2（异丁烯）+ C2H5OH + Br−

结论：Williamson合成中，大基团放在醇钠端，小基团(1°RX)放在卤代烃端。

5. 环氧开环选择性：2-甲基环氧乙烷（环氧丙烷）分别在 (a)酸催化(H2O/H+) 和 (b)碱催化(CH3O−) 条件下开环，画出产物并解释区域选择性。
解答
解题思路：酸催化→进攻取代度高的碳；碱催化→进攻位阻小的碳

(a) 酸催化开环 (H2O/H+)：

H2O进攻取代度较高的碳（C2，连甲基的碳）

产物：CH3CH(OH)CH2OH → 1,2-丙二醇（−OH加在取代度高的碳上）

解释：酸性条件下环氧O被质子化，C2上有更多正电荷（烷基稳定正电荷），H2O作为弱亲核试剂进攻C2。

(b) 碱催化开环 (CH3O−)：

CH3O−进攻位阻较小的碳（C1，−CH2−端）

产物：CH3CH(OH)CH2OCH3 → 1-甲氧基-2-丙醇

解释：强亲核试剂CH3O−走SN2机理，优先进攻位阻小的碳(C1)。

核心区别：酸催化 = 类SN1（正电荷控制）；碱催化 = SN2（位阻控制）。

6. 综合推断题：化合物A(C5H12O)有如下性质：
• 与Na反应放出H2
• 与Lucas试剂(ZnCl2/HCl)立即浑浊
• 与KMnO4不反应（常规条件）
• 脱水后得到两种烯烃
确定A的结构，写出脱水产物。
解答
解题思路：逐步缩小范围——分子式→官能团→碳级→具体结构

第一步：分析分子式

C5H12O，不饱和度 = (2×5+2−12)/2 = 0。无不饱和键或环。可能是醇或醚。

第二步：确认官能团

• 与Na反应放H2 → 含活泼H → 是醇（醚不反应）

第三步：确定碳级

• Lucas试剂立即浑浊 → 叔醇(3°)

• KMnO4不反应 → 印证是3°醇（3°醇不被氧化）

第四步：确定结构

C5H12O的叔醇：需要−OH连在叔碳上且脱水能产生两种烯烃（即有两种不等价的β-H）。

答案：2-甲基-2-丁醇 CH3C(OH)(CH3)CH2CH3

验证：叔碳上连−OH ✓；有两类β-H（一边是−CH3，另一边是−CH2CH3） ✓

第五步：脱水产物

CH3C(OH)(CH3)CH2CH3 脱水(H2SO4, 170°C)经E1消除：

  • 失去−CH2CH3端的β-H → 2-甲基-2-丁烯 CH3C(=CHCH3)CH3（三取代烯烃，Zaitsev主产物）
  • 失去−CH3端的β-H → 2-甲基-1-丁烯 CH2=C(CH3)CH2CH3（二取代烯烃，次产物）

依据Zaitsev规则，主产物为取代度更高的2-甲基-2-丁烯。

考前提醒

SOCl2的立体化学（必背！）
加吡啶 → SN2 → 翻转 | 不加碱 → SNi → 保持
PBr3始终是SN2 → 翻转
Lucas试剂鉴别醇（高频考点！）
3°立即浑浊 | 2°约5min浑浊 | 1°不反应（需加热）
原理：碳正离子稳定性决定SN1速率
醇脱水温度
140°C → 醚（分子间） | 170°C → 烯烃（分子内）
脱水活性：3° > 2° > 1° | 遵循Zaitsev规则 | 注意重排
PCC的特殊作用
PCC(温和氧化剂)：1°醇 → 醛（停在醛！不继续到羧酸）
KMnO4/K2Cr2O7（强氧化剂）：1°醇 → 直接到羧酸
Williamson合成陷阱
卤代烃端必须是1°RX！叔卤代烃 + 醇钠 → E2消除（得不到醚）
策略：大基团在RONa端，小基团在RX端
环氧开环方向（对比记忆）
酸催化 → 进攻取代度高的碳（类Markovnikov，正电荷控制）
碱催化 → 进攻位阻小的碳（反Markovnikov，SN2位阻控制）
两种条件都是反式(anti)开环

酚 pKa≈10（共振稳定酚氧负离子）| 醇 pKa≈16~19
酚溶于NaOH但不溶于NaHCO3 | 羧酸两者都溶
Kolbe vs Reimer-Tiemann
Kolbe：ArONa + CO2 → 水杨酸(−COOH)
Reimer-Tiemann：ArOH + CHCl3/NaOH → 水杨醛(−CHO)
