一、有机化学与有机化合物

定义

有机化合物 = 含碳化合物；有机物的特性：

  • 结构：碳骨架
  • 性质：易燃、熔点低、非电解质
  • 反应：反应慢、副反应多、可逆反应

  • 离子键：电子转移
  • 共价键
    • 极性共价键
    • 非极性共价键

杂化类型几何构型键角实例

分子轨道理论（HOMO / LUMO）

成键轨道 & 反键轨道：

  • HOMO（最高占据分子轨道）
  • LUMO（最低未占据分子轨道）
键的属性
  • 键能：共价键的强度
  • 键长：两原子核间距
  • 键角
  • 偶极矩：偏移量
诱导效应 vs 共轭效应
  • 诱导效应：沿 $\sigma$ 键传递，随距离递减 → 影响酸碱性
  • 共轭效应：$p$-$\pi$ 共轭、$\pi$-$\pi$ 共轭
共价键的断裂方式
  • 均裂（同裂）：A:B → A· + B· → 产生自由基（链锁反应/代替反应）
  • 异裂（不对称断裂）：A:B → A⁺ + B⁻ → 产生离子（碳正离子、碳负离子）
共价键断裂的反应类型
  • 均裂 → 自由基反应：引发 → 增长 → 终止
  • 异裂 → 碳正离子/碳负离子 → 亲电/亲核反应

三、有机酸碱理论

Brønsted 酸碱理论

Acid：质子（$\text{H}^+$）的给予者；Base：质子的接受者

共轭酸碱对

Lewis 酸碱理论
  • Lewis 酸（Acid）：接受电子对 / 缺电子物种，如 $\text{BF}_3$, $\text{AlCl}_3$
  • Lewis 碱（Base）：提供电子对，如 $\text{NH}_3$, $\text{H}_2\text{O}$
酸碱强度与 $\text{p}K_\text{a}$ 的关系：$\text{p}K_\text{a}$ 越小，酸性越强；$\text{p}K_\text{a} + \text{p}K_\text{b} = 14$
比较酸碱性时应注意：
  • 把一些 有机反应归纳为酸碱反应 来理解
  • $\text{p}K_\text{a}$ 值越小 → 酸性越强、其共轭碱越弱
  • 可以判断反应方向：强酸 + 强碱 → 弱酸 + 弱碱

第二章 烷烃和环烷烃

一、烷烃

链状烷烃
  • 通式：$\text{C}_n\text{H}_{2n+2}$
  • 特点：$\sigma$ 键、$sp^3$ 杂化、$109.5°$ 键角

二、命名

A. 开链烷烃（系统命名法）
  1. 选主链：最长碳链（长 > 多 > 支 → 多）
  2. 编号：近（近取代基端）、低（编号之和最小）
  3. 写名称：① 写名 ② 加取代 ③ 写明主链副位次 ④ 标顺序
  4. 书写格式：2,05-三甲基-7-乙基庚烷
B. 单环烷烃
  1. 编号：$\alpha$ 取代基的位置要尽量标记小的编号
  2. 先尝试从各方向编号
    • ② 先产生最低系列的 ③ 与 ④
    • 例：编号 1,2- ... 优于 1,3- ...
C. 桥环烷烃
  1. 基础：二环（入门）/ 三环（复杂），只需掌握二环
  2. 由桥环连接，各桥按碳数从大到小排列
  3. 命名格式：二环[X.Y.Z]烷，括号内标注各桥碳原子数（降序排列，如 [2.2.1]）
  4. 编号起点：从桥头碳开始，沿最长桥 → 次长桥 → 最短桥
  5. 例：二环[2.2.1]庚烷；1,7,7-三甲基二环[2.2.1]庚烷-2-酮（即樟脑）
多环环烷烃
  • 螺环化合物：两环共用一个碳原子（螺原子），命名为 螺[X.Y]烷
  • 桥环化合物：两环共用两个碳原子（桥头碳），即多个 C 组成的桥
椅式构象

三、结构

碳原子的分类

C 按上面所连的其他碳原子数分为：

  • 伯碳（1°C）：连 1 个 C
  • 仲碳（2°C）：连 2 个 C
  • 叔碳（3°C）：连 3 个 C
  • 季碳（4°C）：连 4 个 C

碳上的 H 同理分为伯 H、仲 H、叔 H

环烷烃的结构
  • 小环（3~4元环）：角张力大，不稳定
  • 常见环（5~6元环）：最稳定（接近 $109.5°$）
  • 特别注意：三元环：角张力 → 弯曲键
环烷烃的构象
  • 环丙烷、环丁烷的角张力分析
  • 环己烷最重要的构象：
    • 椅式构象 → 扭曲程度小（对位交叉），最稳定
    • 船式构象 → 不稳定

四、构象

乙烷构象
  • 交叉式：通过 Newman 投影，最稳定 / 扭转能最低（$H$ 之间距离最大）
  • 重叠式：扭转能最高，最不稳定
丁烷构象
  • 全交叉式（anti）：最稳定
  • 邻交叉式（gauche）：$\text{CH}_3$ 之间 gauche 相互作用
  • 全重叠式：最不稳定
  • 稳定性：对位交叉 > 邻位交叉 > 部分重叠 > 全重叠
环己烷构象
  • 椅式构象有直键(a键)和平键(e键)
  • 取代基在 e 键（平键）上更稳定（1,3-二轴向相互作用更小）
  • 翻环：椅式翻转，所有 a 键变 e 键、e 键变 a 键
  • 大取代基优先占 e 键！
构象分析口诀：
  • 取代基越大 → 越要占据 e 键（平伏键）
  • 交叉 > 重叠（扭转能低 → 稳定）
  • 大取代基避免 1,3-二轴向作用
注意：顺反异构与构象异构不同！
  • 构象异构：通过键的旋转互变，不需断键
  • 顺反异构：需要断键才能互变

六、化学性质

1. 卤代反应（自由基反应）

总反应：

$$\text{R-H} + \text{X}_2 \xrightarrow{h\nu \text{ 或 }\Delta} \text{R-X} + \text{HX}$$

（$\text{X} = \text{Cl}, \text{Br}$）反应活性 $\text{F}_2 > \text{Cl}_2 > \text{Br}_2 > \text{I}_2$（碘太慢不反应）

自由基链式反应机理：

  1. 链引发：$\text{Cl}_2 \xrightarrow{h\nu} 2\text{Cl}\cdot$
  2. 链增长：
    • $\text{Cl}\cdot + \text{CH}_4 \to \text{CH}_3\cdot + \text{HCl}$
    • $\text{CH}_3\cdot + \text{Cl}_2 \to \text{CH}_3\text{Cl} + \text{Cl}\cdot$
  3. 链终止：自由基相互结合
卤代反应的选择性：
  • 活性（H的种类）：$3°\text{C-H} > 2°\text{C-H} > 1°\text{C-H}$
  • 判断卤代产物时须同时考虑：H 的种类 + H 的个数
  • 选择性：$\text{Br}_2$ 选择性 > $\text{Cl}_2$（$\text{Br}_2$ 几乎只取代 3°H）
自由基稳定性顺序：
$$3°\text{C}\cdot > 2°\text{C}\cdot > 1°\text{C}\cdot > \text{CH}_3\cdot$$

原因：含 C° 越多越稳定，超共轭效应（$\alpha$-H 越多）越强

2. 加成反应（环烷烃）

小环烷烃的开环加成（环丙烷特有）

环丙烷的弯曲键使其像烯烃一样可发生加成反应：

(1) 加 $\text{H}_2$（催化加氢）：

$$\triangle + \text{H}_2 \xrightarrow{\text{Ni}, \Delta} \text{CH}_3\text{CH}_2\text{CH}_3$$

(2) 加 $\text{Cl}_2$ / $\text{Br}_2$：

$$\triangle + \text{Cl}_2 \xrightarrow{\text{FeCl}_3} \text{ClCH}_2\text{CH}_2\text{CH}_2\text{Cl}$$ $$\triangle + \text{Br}_2 \to \text{BrCH}_2\text{CH}_2\text{CH}_2\text{Br}$$

(3) 加 HBr：

$$\triangle + \text{HBr} \to \text{CH}_3\text{CH}_2\text{CH}_2\text{Br}$$
取代环丙烷加 HX：
  • 遵循 Markovnikov 规则
  • H 加在含 H 较多的碳上（类似烯烃加成）
  • 开环断裂的是取代最多的 C-C 键

3. 氧化反应

氧化反应

(1) 燃烧（完全氧化）

(2) 受控氧化：

$$\text{RCHCOR'} + \text{O}_2 \xrightarrow{\text{Mn}^{2+}/\text{Co}^{2+}} \text{RCOOH} + \text{R'COOH} + \text{羧酸/酮}$$

烷烃在催化条件下可被氧化为羧酸（工业用途）

第 4 页

第三章 烯烃

1. 同分异构

同分异构类型
  • ① 构造异构：碳链异构 / 官能团位置异构
  • ② 顺反异构（几何异构）：双键不能旋转 → 顺式/反式

2. 命名

烯烃命名规则
  • ① 选含 $\text{C}=\text{C}$ 的最长碳链为主链
  • ② 从靠近 $\text{C}=\text{C}$ 的一端开始编号：$Z$-/$E$- 标注（重要！）
  • ③ 烯烃 → $-\text{CH}=\text{CH}_2$（乙烯基）；烯丙基 → $-\text{CH}=\text{CH-CH}_3$
$Z$/$E$ 与 顺/反 命名法

$Z$/$E$ 命名法：按 CIP 次序规则（Cahn-Ingold-Prelog）

  • 比较双键两个碳上各连接基团的优先顺序
  • $E$（entgegen，相反）：两个较优先基团分居双键两侧
  • $Z$（zusammen，相同）：两个较优先基团在双键同侧

CIP 次序规则要点：

  1. 原子序数大者优先
  2. 同位素：质量数大者优先
  3. 逐级比较至区分为止
  4. 重键展开为等价单键（如 $\text{C}=\text{O}$ 展开为两个 C-O 单键）
注意：$Z$/$E$ 与顺/反不一定一致！
$Z$ 不一定是 "顺式"，$E$ 不一定是 "反式"，取决于取代基的优先次序

3. 化学性质

烯烃的特征反应

烯烃 $\pi$ 键电子易受亲电试剂进攻 → 亲电加成反应为核心

烯烃的稳定性
  • 可通过氢化热（加 $\text{H}_2$ 释放的热量）判断稳定性
  • 氢化热越低 → 烯烃越稳定
  • 取代基越多 → 烯烃越稳定：$\text{R}_2\text{C}=\text{CR}_2 > \text{R}_2\text{C}=\text{CHR} > \text{RCH}=\text{CHR} > \text{RCH}=\text{CH}_2$
  • 超共轭效应：$\alpha$-H 的 C-H 键与 $\pi$ 键的 $\sigma$-$\pi$ 超共轭，$\alpha$-H 越多越稳定
  • 反式（$E$）比顺式（$Z$）更稳定（空间位阻小）
HX 酸催化加成与 Markovnikov 规则

烯烃催化加 HX 时H 加到含 H 较多的碳上，$\text{X}$ 加到含 H 较少的碳上（Markovnikov 规则）

本质：反应经过较稳定的碳正离子中间体（$3° > 2° > 1°$）

碳正离子稳定性：$3° > 2° > 1° > \text{CH}_3^+$，这决定了加成的区域选择性和可能的重排

$\text{H}_2\text{SO}_4$ / $\text{H}_3\text{PO}_4$ 加成

间接水合法制醇：先加 $\text{H}_2\text{SO}_4$，再水解

卤化反应

(1) 加 $\text{Cl}_2$ / $\text{Br}_2$（反应活性区别）

$$\text{C}=\text{C} + \text{Cl}_2/\text{Br}_2 \xrightarrow{} \text{二卤代物}$$

反应机理：

  1. $\pi$ 电子进攻 $\text{Br}_2$ → 形成溴鎓离子（环状三元中间体）
  2. $\text{Br}^-$ 从背面进攻 → 反式加成（anti addition）
重要结论：卤素加成是经过卤鎓离子中间体的反式加成。
  • 溴鎓离子是桥环正离子
  • $\text{Br}^-$ 从背面（anti）进攻
  • 产物为 anti 加成产物（反式邻二卤代物）
亲电加成的机理总结

① 亲电试剂进攻 $\pi$ 键 → 碳正离子（或环状鎓离子）

② 亲核试剂与碳正离子结合 → 产物

机理决定产物的区域选择性（Markovnikov）和立体选择性（反式加成）

卤化氢（$\text{HX}$）加成

活性：$\text{HI} > \text{HBr} > \text{HCl} > \text{HF}$

遵循 Markovnikov 规则（经碳正离子中间体，遵循稳定性序列）

$\text{Cl}^-$ 不生成氯鎓离子 → 无立体选择性；一般生成顺反混合物

地位和意义：碳正离子稳定性的比较是有机化学的核心考点之一，贯穿亲电加成、消除反应、重排反应等多个章节。
第 5 页

亲电加成的区域选择性（续）

Markovnikov 规则（马氏规则）
  • 不对称烯烃加 HX 时，H 加在含 H 较多的碳上，X 加在含 H 较少的碳上
  • 本质：经过较稳定的碳正离子中间体
  • 碳正离子稳定性：$3° > 2° > 1° > \text{CH}_3^+$
  • 可用超共轭效应和诱导效应解释
碳正离子重排

当初始生成的碳正离子不够稳定时，可通过1,2-迁移（H 迁移或 $\text{CH}_3$ 迁移）重排为更稳定的碳正离子：

  • 1,2-氢迁移：$2°\text{C}^+ \xrightarrow{} 3°\text{C}^+$
  • 1,2-甲基迁移：同理
  • 重排发生的驱动力：碳正离子趋向更稳定的结构
碳正离子重排的判断要点：
  • 氢酸和硫酸的加成反应，常伴随碳正离子重排
  • 氯化氢与烯烃的加成反应：常有重排产物
  • 注意：碳正离子不仅参与加成反应，也参与消除反应和取代反应
例题：判断烯烃亲电加成的产物

例题：下列化合物中的烯烃能否与下列试剂进行加成反应？如不反应写出其产物；如果有过渡态的复杂化合物产物写出：

  1. $\text{C}=\text{C}-\text{C}-\text{C}$ 加 $\text{HBr}$ → $\text{H}_3\text{C}-\overset{\text{Br}}{|}\text{C}-\text{C}-\text{C}$（马氏产物）
  2. $\text{CH}_2=\text{C}(\text{CH}_3)-\text{C}(\text{CH}_3)_2$ 加 HBr → 可能发生重排
  3. $\text{CH}_3\text{CH}=\text{CHCH}_3 + \text{Cl}_2$ → 反式加成产物

(8) 烯烃的自由基加成（反Markovnikov加成）

过氧化物效应（反马氏加成）

在过氧化物（ROOR）存在下：

$$\text{ROOR} \xrightarrow{\Delta} 2\text{RO}\cdot$$ $$\text{RO}\cdot + \text{HBr} \to \text{ROH} + \text{Br}\cdot$$

自由基加成机理：

$$\text{Br}\cdot + \text{CH}_2=\text{CHCH}_2\text{CH}_3 \to \text{CH}_3\text{CHCH}_2\text{CH}_3 \xrightarrow{\text{HBr}} \text{CH}_3\text{CH}(\text{Br})\text{CH}_2\text{CH}_3$$

产物为反 Markovnikov加成产物

过氧化物效应的限制：
  • 仅限于 $\text{HBr}$；$\text{HCl}$ 和 $\text{HI}$ 不发生（$\text{HCl}$ 难以夺取 Br·，$\text{HI}$ 中 I· 太稳定不反应）

(10) 硼氢化-氧化（重要反应）

硼氢化-氧化反应

这是一个协同反应（特殊反应）：不产生碳正离子/自由基中间体

烯烃的硼氢化 —— 氧化反应（反马氏加成、顺式加成）：

反应条件与步骤：

  1. 硼氢化：$\text{烯烃} + (\text{BH}_3)_2 \xrightarrow{\text{THF}} \text{烷基硼}$
  2. 氧化：$\text{烷基硼} + \text{H}_2\text{O}_2 / \text{NaOH} \to \text{醇}$

特点：

  • 反马氏方向：B 加在含 H 多的碳上（$\text{OH}$ 最终在含 H 多的碳上）
  • 顺式加成（syn addition）：B 和 H 从同侧加成
  • 本反应为协同反应，无碳正离子中间体，不会重排

例：

$$\text{CH}_3\text{CH}=\text{CH}_2 \xrightarrow{1.\,(\text{BH}_3)_2} \xrightarrow{2.\,\text{H}_2\text{O}_2,\,\text{NaOH}} \text{CH}_3\text{CH}_2\text{CH}_2\text{OH}$$

（产物为反马氏的伯醇）

硼氢化-氧化 vs 酸催化水合：
方法方向立体重排产物
酸催化水合马氏无选择性可能重排仲/叔醇
硼氢化-氧化反马氏顺式加成不重排伯醇

环氧化反应

烯烃的环氧化（过氧酸氧化）

使用 mCPBA（间氯过氧苯甲酸）或其他过氧酸：

$$\text{烯烃} + \text{RCOOOH} \xrightarrow{} \text{环氧化物} + \text{RCOOH}$$

特点：顺式加成（协同机理），保持双键两侧取代基的相对构型

对比氧化的反应机理

四氧化锇（$\text{OsO}_4$）氧化

四氧化锇 $\text{OsO}_4$ 与烯烃进行顺式双羟基化反应：

$$\text{烯烃} \xrightarrow{\text{OsO}_4} \text{锇酸酯中间体} \xrightarrow{\text{还原（如 }\text{NaHSO}_3\text{）}} \text{邻二醇（顺式）}$$

特点：

  • 协同机理：$\text{OsO}_4$ 从烯烃同一面进攻 → 顺式邻二醇
  • 产物保持 syn 立体化学
高锰酸钾（$\text{KMnO}_4$）氧化
  • 冷、稀、碱性条件（Baeyer 试验）：$\text{烯烃} \xrightarrow{\text{KMnO}_4(\text{冷，稀})} \text{邻二醇}$（顺式双羟基化，类似 $\text{OsO}_4$）
  • 热、浓、酸性条件：氧化断键 → 产生酮 / 羧酸

第四章 炔烃和二烯烃

(1) 结构与命名

炔烃结构

通式：$\text{C}_n\text{H}_{2n-2}$，含一个 $\text{C}\equiv\text{C}$ 三键（1 个 $\sigma$ 键 + 2 个 $\pi$ 键）。

$\text{HC}\equiv\text{CH}$：sp 杂化，线型分子（键角 $180^\circ$）。

两个相互垂直的 $\pi$ 键构成圆柱形 $\pi$ 电子云，包裹在 $\sigma$ 键周围。

$\Delta$ C-H 酸性

碳氢键酸性比较

键长：$d(\text{C}_{sp}\text{-H}) < d(\text{C}_{sp^2}\text{-H}) < d(\text{C}_{sp^3}\text{-H})$

酸性顺序：$\text{C}_{sp}\text{-H} > \text{C}_{sp^2}\text{-H} > \text{C}_{sp^3}\text{-H}$

原因：s 轨道成分越多（sp: 50%, sp$^2$: 33%, sp$^3$: 25%），碳的电负性越大，C-H 键极性越强，越容易失去 $\text{H}^+$。

命名

  • 母体的选择：含叁键的最长碳链
  • 编号：使叁键位次最小
  • 烯炔：先烯后炔（名称中先写烯再写炔）

物理性质

  • 低级炔烃是无色气体，略溶或不溶于水
  • 分子间不能形成氢键，沸点主要由分子量决定
  • 随碳原子数增大，沸点升高
  • 乙炔微溶于水，其在水中的溶解度略大于乙烯
  • 炔烃的密度小于1，轻于水（液态炔烃比重 < 1）
  • 乙炔是无色气体，由电石制得的含杂质（$\text{H}_2\text{S}$, $\text{PH}_3$）有臭味
芳香性判定（Huckel 规则）

环状、共面、全共轭体系，$\pi$ 电子数 = $4n+2$（$n=0,1,2,...$）时具有芳香性。

反芳香性：$\pi$ 电子数 = $4n$，平面共轭环不稳定。

Stability（稳定性）：$K_{\text{eq}}$ 大表示热力学更稳定；Reactivity（反应性）：速率快慢，和活化能有关。
基团比较中需要分清热力学稳定性和动力学反应性的区别。

同分异构

不饱和度 (IHD / Degree of Unsaturation)

公式：$\text{IHD} = \frac{2C + 2 + N - H - X}{2}$

一个双键 = IHD 1，一个叁键 = IHD 2，一个环 = IHD 1

例：$\text{C}_4\text{H}_6$（IHD = 2）的同分异构体：
  • $\text{CH}\equiv\text{C-CH}_2\text{-CH}_3$（1-丁炔）
  • $\text{CH}_3\text{-C}\equiv\text{C-CH}_3$（2-丁炔）
  • $\text{CH}_2\text{=CH-CH=CH}_2$（1,3-丁二烯）
  • $\text{CH}_2\text{=C=CH-CH}_3$（1,2-丁二烯）
  • 环丁烯
  • 甲基环丙烯等
基团优先级与命名
含有烯、炔双功能团时，编号使不饱和键位次之和最小；位次相同时，双键优先获得较小编号。

亲核加成定义

亲核试剂（Nu:）进攻三键碳原子，发生加成反应。

常见亲核试剂：$\text{-OH}$, $\text{-SH}$, $\text{-NH}_2$, $\text{-MeL}$, $\text{-CONH}_2$, $\text{-COOR}$, $\text{-CN}$ 等

炔烃亲核加成比烯烃更容易

(2) 加成产物为烯烃（比炔烃更稳定），热力学有利。

$\text{RC}\equiv\text{CH} + \text{HCN} \xrightarrow{\text{CuCl}} \text{R-C(=CH}_2\text{)-CN}$

$\text{HC}\equiv\text{CH} + \text{ROH} \xrightarrow{\text{KOH}} \text{CH}_2\text{=CH-OR}$（乙烯基醚）

催化剂：$\text{Hg}^{2+}$；工业上用于合成聚醋酸乙烯酯（PVAc）。

4. RMgBr 反应（有机金属试剂）

$\text{RC}\equiv\text{CH} + \text{R'MgBr} \to \text{RC}\equiv\text{CMgBr} + \text{R'H}$

末端炔烃酸性 > 烷烃，因此格氏试剂可夺取端炔 H，生成炔基格氏试剂。

反应本质是酸碱反应（非加成），但归类为金属有机试剂制备。

$\text{CaO} + 3\text{C} \xrightarrow{\text{电弧}} \text{CaC}_2 + \text{CO}$

$\text{CaC}_2 + 2\text{H}_2\text{O} \to \text{Ca(OH)}_2 + \text{CH}\equiv\text{CH}\uparrow$

即工业上从电石（碳化钙）制备乙炔的方法。

酸性炔的其它反应

酰化：$\text{RC}\equiv\text{CH} \xrightarrow[\text{(1) BuLi}]{\text{(2) R'COCl}} \text{R'C(=O)-C}\equiv\text{CR}$（炔基酮）

烷基化：$\text{RC}\equiv\text{CMgBr} + \text{R'X} \to \text{RC}\equiv\text{CR'} + \text{MgBrX}$

利用格氏试剂进行 $\text{S}_\text{N}2$ 碳碳键形成。注意只适用于伯卤代烃。

共轭二烯烃

命名

命名规则

$\text{CH}_2\text{=CH-CH=CH-CH=CH}_2$（1,3,5-己三烯）

  • 选含最多双键的最长碳链为母链
  • 从离双键最近的一端编号
  • 注意 s-cis 和 s-trans 构象

分类

  • 累积二烯烃（allene 型）：$\text{C=C=C}$，两个双键共用一个碳原子
  • 共轭二烯烃：$\text{C=C-C=C}$，双键被一个单键隔开
  • 隔离二烯烃：$\text{C=C-C-C=C}$，双键被两个或以上单键隔开

共轭二烯烃的结构

键长与共轭

以 1,3-丁二烯为例：$\text{CH}_2\text{=CH-CH=CH}_2$

C1=C2 和 C3=C4 双键键长约 $1.34\ \text{\AA}$

C2-C3 单键键长 $1.48\ \text{\AA}$（比正常 C-C 单键 $1.54\ \text{\AA}$ 短）

原因：四个碳均为 sp$^2$ 杂化，四个 p 轨道平行排列、连续重叠，$\pi$ 电子离域（共轭效应）。

s-cis 与 s-trans 构象

s-trans（$\sigma$-trans）：两个双键呈反式排列，更稳定

s-cis（$\sigma$-cis）：两个双键呈顺式排列，能量稍高但D-A 反应必须采用此构象

"s" 代表 single bond（绕 C2-C3 单键旋转）

共轭效应

  • $\pi$-$\pi$ 共轭：$\text{C=C-C=C}$
  • $p$-$\pi$ 共轭：杂原子孤对电子与 $\pi$ 键共轭
  • 超共轭：$\text{C-H}$ $\sigma$ 键与相邻 $\pi$ 键的共轭

共轭二烯烃的稳定性

氢化热比较

1,3-丁二烯氢化热 = $236\ \text{kJ/mol}$

两个独立的 1-丁烯氢化热 = $2 \times 126.8 = 253.6\ \text{kJ/mol}$

共轭稳定化能 $\approx 17.6\ \text{kJ/mol}$

共轭二烯烃的反应

1,2-加成 vs 1,4-加成

亲电加成选择性

$\text{CH}_2\text{=CH-CH=CH}_2 + \text{HBr} \to$

  • 1,2-加成：$\text{CH}_3\text{-CHBr-CH=CH}_2$（动力学产物，低温有利）
  • 1,4-加成：$\text{CH}_3\text{-CH=CH-CH}_2\text{Br}$（热力学产物，高温有利）
温度控制：
低温（$-80\ ^\circ\text{C}$）有利于 1,2-加成（动力学控制）
高温（$40\ ^\circ\text{C}$）有利于 1,4-加成（热力学控制）

Diels-Alder 反应（[4+2] 环加成）

D-A 反应（协同反应）

共轭二烯（双烯体，Diene，4$\pi$ 电子） + 亲双烯体（Dienophile，2$\pi$ 电子）

$\to$ 环己烯衍生物（一步成环，协同机理）

反应条件：加热即可（无需催化剂），属于周环反应

特点：

  • 双烯体必须能取 s-cis 构象（环状二烯如环戊二烯天然固定在 s-cis）
  • 亲双烯体通常含有吸电子基团（$\text{-CHO}$, $\text{-COOR}$, $\text{-CN}$, $\text{-NO}_2$, $\text{-COR}$）
  • 区域选择性：双烯体上给电子基和亲双烯体上吸电子基互补配对
  • 立体专一性：顺式加成（suprafacial），保持双烯体和亲双烯体原有的立体化学
  • 产物立体化学：endo 规则优先（次级轨道相互作用导致 endo 产物为动力学产物）
D-A 反应是有机合成中最重要的成环反应之一，一步构建六元环并同时形成两个新的 C-C 键。可逆（逆 D-A 反应，retro D-A）。
经典例子：
1,3-丁二烯 + 马来酸酐（maleic anhydride） $\to$ 四氢邻苯二甲酸酐
（Diene + Dienophile $\to$ 环己烯产物）
反应过程中双烯体和亲双烯体的取代基相对构型保持不变（顺式加成原理）。

识别方法：D-A 产物可通过逆 D-A 反应验证结构（加热分解回原料）。

炔烃反应总结

1) 催化还原

不同条件下的还原

完全还原：$\text{-C}\equiv\text{C-} \xrightarrow{\text{H}_2/\text{Pt}} \text{-CH}_2\text{CH}_2\text{-}$（烷烃）

部分还原（顺式烯烃）：

$\text{-C}\equiv\text{C-} \xrightarrow{\text{Lindlar}} \text{顺式 C=C}$（顺式）

$\text{Lindlar}$ 催化剂 = $\text{Pd/CaCO}_3$（被 $\text{PbO(Ac)}_2$ 和喹啉毒化的 Pd）

部分还原（反式烯烃）：

$\text{-C}\equiv\text{C-} \xrightarrow{\text{Na/NH}_3(\text{l})} \text{反式 C=C}$（反式，Birch 还原）

重要：Lindlar 催化剂给顺式，$\text{Na/NH}_3(\text{l})$ 给反式。这是必考知识点！

2) 亲电加成

卤素加成

$\text{R-C}\equiv\text{C-H} + \text{X}_2 \to \text{R-CX=CHX}$（反式加成，一分子）

$\text{R-CX=CHX} + \text{X}_2 \to \text{R-CX}_2\text{-CHX}_2$（四卤代物）

HX 加成

$\text{R-C}\equiv\text{CH} + \text{HX} \to$ 一卤代烯烃（遵循马氏规则，反马氏加成同 Markovnikov 规则）

继续加成得到 gem-二卤代物（Geminal，同碳双卤）。

水合反应（马氏加成 → 酮）

$\text{R-C}\equiv\text{CH} + \text{H}_2\text{O} \xrightarrow{\text{HgSO}_4/\text{H}_2\text{SO}_4} [\text{烯醇}] \xrightarrow{\text{互变}} \text{酮}$

机理：马氏加成 → 烯醇中间体 → 酮式-烯醇式互变异构（Keto-Enol Tautomerism）

乙炔水合得到乙醛（唯一例外）；其他端炔水合得到甲基酮（$\text{RCOCH}_3$）。

硼氢化-氧化（反马氏 → 醛）

$\text{R-C}\equiv\text{CH} \xrightarrow{\text{1) (sia)}_2\text{BH}} \xrightarrow{\text{2) H}_2\text{O}_2/\text{NaOH}} \text{R-CH}_2\text{-CHO}$（醛）

反马氏加成产物，B 加到末端碳上。

互补性记忆：
水合（$\text{HgSO}_4$）：马氏 → 酮
硼氢化-氧化：反马氏 → 醛
两种方法可分别将端炔转化为酮或醛！

3) 亲核加成

HCN 加成

$\text{R-C}\equiv\text{CH} + \text{HCN} \to \text{R-C(=CH}_2\text{)-CN}$

产物为 $\alpha,\beta$-不饱和腈。

ROH 加成

$\text{R-C}\equiv\text{CH} + \text{ROH} \xrightarrow{\text{NaOR}} \text{R-C(=CH}_2\text{)-OR}$（乙烯基醚）

(5) 亲核取代（碳链增长）

炔钠的亲核取代（$\text{S}_\text{N}2$ 反应）

第一步 — 生成炔负离子：$\text{RC}\equiv\text{CH} \xrightarrow{\text{NaNH}_2} \text{RC}\equiv\text{C}^-\text{Na}^+$

第二步 — $\text{S}_\text{N}2$ 取代：$\text{RC}\equiv\text{C}^-\text{Na}^+ + \text{R'X} \to \text{RC}\equiv\text{CR'} + \text{NaX}$

最终实现碳碳键的形成和碳链增长。

炔基格氏试剂路线

$\text{RC}\equiv\text{CH} \xrightarrow{\text{R'MgX}} \text{RC}\equiv\text{CMgX} + \text{R'H}$

$\text{RC}\equiv\text{CMgX} \xrightarrow{\text{R''X}} \text{RC}\equiv\text{CR''}$

也可通过 $\text{Ag(NH}_3)_2^+$ 或 $\text{Cu(NH}_3)_2^+$ 来检验是否为末端炔烃（沉淀法）。

亲核取代反应注意事项：
(1) 只有伯卤代烃（$\text{R'CH}_2\text{X}$）适合与炔钠进行 $\text{S}_\text{N}2$ 反应
(2) 仲卤代烃和叔卤代烃主要发生消除反应（E2），不适用
(3) 反应在极性非质子溶剂（如 DMF, DMSO）中效果更好

4) 金属炔化物

碱性条件选择

$\text{R-C}\equiv\text{CH} + \text{NaOH} \to$ 不反应（$\text{OH}^-$ 碱性不够强，$\text{p}K_a$ 15.7 vs 炔 $\text{p}K_a$ 25）

$\text{R-C}\equiv\text{CH} + \text{NaNH}_2 \to \text{RC}\equiv\text{CNa} + \text{NH}_3$（$\text{NH}_2^-$ 碱性足够，$\text{p}K_a$ 38）

端炔检验（鉴别反应）

$\text{RC}\equiv\text{CH} + [\text{Ag(NH}_3)_2]^+ \to \text{RC}\equiv\text{CAg}\downarrow$（白色沉淀，炔化银）

$\text{RC}\equiv\text{CH} + [\text{Cu(NH}_3)_2]^+ \to \text{RC}\equiv\text{CCu}\downarrow$（红棕色沉淀，炔化亚铜）

安全：干燥的金属炔化物受撞击或加热会爆炸！实验后须用稀 $\text{HNO}_3$ 分解。

6) 氧化

$\text{R-C}\equiv\text{CH} \xrightarrow{\text{KMnO}_4} \text{RCOOH} + \text{CO}_2\uparrow$

$\text{R-C}\equiv\text{C-R'} \xrightarrow{\text{KMnO}_4} \text{RCOOH} + \text{R'COOH}$

合成橡胶

二烯烃的聚合

1,3-丁二烯聚合：

第五章 芳烃

一、结构

苯的结构特点
  1. 碳原子均为 $sp^2$ 杂化，构成正六边形骨架
  2. 所有原子共平面，键角均为 $120°$
  3. 碳碳键长 $0.140\;\text{nm}$（介于 $\text{C-C}$ 单键 $0.154\;\text{nm}$ 和 $\text{C=C}$ 双键 $0.134\;\text{nm}$ 之间），是平均化的"离域键"
  4. 闭合共轭体系：6个 $p$ 轨道垂直于平面、肩并肩重叠，形成环形大 $\pi$ 键（$\pi_6^6$）
化学键长

苯的 $\text{C-C}$ 键长 $= 0.140\;\text{nm}$，介于单键与双键之间，说明苯环的键是完全均等的离域键。

二、同分异构

  • 一般叫法：烷基苯结构
  • 取代时产生异构体：苯环上的取代基位置不同（邻位 ortho、间位 meta、对位 para）
  • 各组同分异构体个数与取代基种类和数目有关

三、命名

芳烃命名规则
  1. 苯环作母体的命名：以苯为母体，烷基为取代基，如 $\text{1,2-}$二甲苯（邻二甲苯）
  2. 烷基作母体的命名：当侧链较长或含有官能团时，苯基($\text{C}_6\text{H}_5\text{-}$)作为取代基，命名为"苯基某烷"
  3. 常用俗名：甲苯(toluene)、二甲苯(xylene)、苯乙烯(styrene)、萘(naphthalene)等
  4. 多取代时，编号遵循最低系列原则，编号之和最小
二甲苯有三种异构体：邻二甲苯（$o\text{-}$xylene）、间二甲苯（$m\text{-}$xylene）、对二甲苯（$p\text{-}$xylene）。

四、物理性质

  • 芳烃通常无色液体，有特殊芳香气味
  • 不溶于水，溶于有机溶剂（乙醚、$\text{CCl}_4$、石油醚）
  • 一般芳烃的比水轻，$d < 1$
  • 沸点随相对分子量增大而升高；液态芳烃的沸点与相对分子质量有关，还与分子结构/对称性有关。邻/间/对二甲苯中，对二甲苯对称性最高，熔点最高；而沸点差距不大
  • $^{1}\text{H NMR}$ 化学位移：芳环质子 $\delta = 6\text{-}7\;\text{ppm}$
芳环质子 NMR 化学位移范围：$\delta = 6\text{-}7\;\text{ppm}$（此处原笔记标注为苯环 $\text{H}$ 的特征位移范围）

五、化学反应

（一）一般的亲电取代反应

亲电取代反应概述

芳环上的亲电取代反应通式：

$$\text{ArH} + \text{E}^+ \longrightarrow \text{ArE} + \text{H}^+$$

其中 $\text{E}^+$ 为亲电试剂。苯环提供 $\pi$ 电子作为亲核体，与亲电试剂结合。

亲电取代机理（两步）
  1. 第一步（慢，决速步）：亲电试剂进攻苯环 $\pi$ 电子，生成 $\sigma$ 络合物（Wheland中间体/碳正离子中间体），芳香性被破坏
  2. 第二步（快）：$\sigma$ 络合物失去一个质子($\text{H}^+$)，恢复芳香性，生成取代产物
$$\text{ArH} + \text{E}^+ \xrightarrow{\text{慢}} [\text{ArHE}]^+ \xrightarrow{-\text{H}^+,\;\text{快}} \text{ArE}$$
$\sigma$ 络合物（又叫 Wheland 中间体、碳正离子中间体）：苯环芳香性被破坏，一个碳变为 $sp^3$ 杂化，正电荷在环上离域。这是亲电取代的速控步骤的产物。

(1) 烷基化反应（续）

碳正离子来源

方法一：$\text{AlCl}_3 + \text{H}_2\text{O} \longrightarrow \text{Al(OH)}_3\cdot\text{(in)}^- + \text{H}^+$

$\text{CH}_3\text{CH=CH}_2 + \text{H}^+ \longrightarrow \text{CH}_3\text{CH}^+\text{CH}_3$（烯烃质子化得到碳正离子）

方法二：$\text{BF}_3 + \text{H}_2\text{O} \longrightarrow \text{BF}_3(\text{OH})^- + \text{H}^+$（BF$_3$ 也可作Lewis酸催化剂）

F-C 反应的问题

F-C 烷基化的四大问题（非常重要！考试重点）
(1) 多烷基化（连续烷基化）

烷基是给电子基团（供电子）→ 引入第一个烷基后芳环活化 → 更容易发生第二次、第三次烷基化 → 难以停留在一取代。

例：苯 + 过量 $\text{CH}_3\text{CH}_2\text{Cl}$ → 不仅得到乙苯，还得到二乙苯、三乙苯…

控制方法：使用大量过量的苯（$50:1$ 等比例），稀释烷基化试剂。

(2) 异构化（碳正离子重排）

碳正离子会发生重排：伯碳正离子 → 仲碳正离子 → 叔碳正离子（更稳定）

$$\text{CH}_3\text{CH}_2\text{CH}_2\text{-Cl} + \text{AlCl}_3 \longrightarrow \text{CH}_3\text{CH}_2\text{CH}_2^+ \xrightarrow{\text{重排}} \text{CH}_3\text{CH}^+\text{CH}_3$$

结果：正丙基苯和异丙基苯同时生成，甚至以异丙基苯为主要产物。

(3) 烷基迁移（歧化反应）

傅-克烷基化后的烷基在AlCl$_3$存在下可从一个苯环迁移到另一个苯环上，最终达到热力学平衡。

$\text{AlCl}_3$ 催化下，多烷基苯可与苯发生歧化反应。

(4) 对吸电子基团取代苯的限制

傅-克烷基化反应不适用于强吸电子取代基存在的芳环（如 $\text{-NO}_2$, $\text{-COR}$, $\text{-COOH}$ 等）。

这些基团使苯环钝化严重，活性过低，反应往往不能进行。

(2) 酰基化反应

F-C 酰基化
$$\text{C}_6\text{H}_6 + \text{RCOCl} \xrightarrow{\text{AlCl}_3} \text{C}_6\text{H}_5\text{COR} + \text{HCl}$$

产物为芳酮。

酰基正离子（$\text{RCO}^+$，也称酰基阳离子 acylium ion）是亲电试剂：

$$\text{RCOCl} + \text{AlCl}_3 \longrightarrow \text{RCO}^+ + [\text{AlCl}_4]^-$$ $$\text{AlCl}_3^- + \text{H}^+ = \text{AlCl}_3 + \text{HCl}$$
酰基阳离子的结构：$\text{R-C}^+=\text{O} \leftrightarrow \text{R-C}\equiv\text{O}^+$，因共振稳定，不会发生重排。

歧化反应

歧化（disproportionation）

在 $\text{AlCl}_3$ 等Lewis酸催化下，可发生分子间烷基转移：

例如：甲苯 + 甲苯 $\xrightarrow{\text{AlCl}_3}$ 苯 + 二甲苯

F-C 酰基化反应特点

酰基化 vs 烷基化的优势
  • 酰基是吸电子基，引入后钝化苯环 → 不会发生连续酰基化（不会多取代）
  • $\text{RCO}^+$ 因共振而稳定，不会发生碳正离子重排
  • 另外，傅-克酰基化反应也不可逆，不会发生歧化
  • 产物酮可通过还原得到烷基苯：$\text{ArCOR} \xrightarrow{\text{Zn-Hg / HCl}} \text{ArCH}_2\text{R}$（Clemmensen还原）
酰基化 + 还原 = 间接烷基化：先酰基化再还原，可以避免碳正离子重排和多烷基化问题。这是获得直链烷基苯的重要方法。 $$\text{ArH} \xrightarrow[\text{AlCl}_3]{\text{RCOCl}} \text{ArCOR} \xrightarrow{\text{Zn-Hg / HCl}} \text{ArCH}_2\text{R}$$
注意：F-C 酰基化需要超过1当量的 $\text{AlCl}_3$（$\text{AlCl}_3 > 2.0\;\text{equiv}$），因为产物酮会与 $\text{AlCl}_3$ 配位形成络合物，消耗催化剂。
Clemmensen 还原：$\text{Zn-Hg / HCl}$，将芳酮的羰基还原为亚甲基 $\text{-CH}_2\text{-}$，可以得到直链烷基苯。
$$\text{C}_6\text{H}_5\text{COCH}_3 \xrightarrow{\text{Zn-Hg / HCl}} \text{C}_6\text{H}_5\text{CH}_2\text{CH}_3$$

取代基定位规则与芳环侧链反应

一、一取代苯的定位规则

邻对位定位基（第一类定位基）

供电子基 —— 使苯环活化，新的取代基优先进入邻位和对位：

$\text{-O}^-$, $\text{-NR}_2$, $\text{-NHR}$, $\text{-NH}_2$, $\text{-OH}$, $\text{-OR}$, $\text{-NHCOR}$, $\text{-OCOCH}_3$,

$\text{-R}$（$\text{-CH}_3$等烷基）, $\text{-Ar}$（芳基）

特例：卤素（$\text{-F}$, $\text{-Cl}$, $\text{-Br}$, $\text{-I}$）是邻对位定位基但使苯环钝化（吸电子诱导效应 > 供电子共轭效应，总效果为吸电子，但共轭效应决定了定位方向为邻对位）。
间位定位基（第二类定位基）

吸电子基 —— 使苯环钝化，新的取代基优先进入间位：

$\text{-N}^+\text{R}_3$ / $\text{-N}^+\text{H}_3$, $\text{-CF}_3$, $\text{-CCl}_3$, $\text{-CN}$, $\text{-SO}_3\text{H}$, $^+\text{CHO}$,

$\text{-COR}$, $\text{-COOH}$, $\text{-COOR}$, $\text{-CONHR}$, $\text{-CONH}_2$,

$\text{-NO}_2$（最强钝化基之一）

Warning! 卤素虽是邻对位定位基（定位效应由共轭效应决定），但它们使苯环钝化（总体为吸电子效应），这一点容易出错！

二、二取代苯的定位规则

二取代苯上第三个取代基的位置

当苯环上已有两个取代基时，判断第三个取代基进入位置的规则：

  1. 两个定位基效应一致（协同定位）：取代基进入二者共同指定的位置
  2. 强活化基团的定位效应 > 弱活化基团
  3. 两个定位基效应矛盾时：以强定位基为主，并且一般不进入两个取代基之间（空间位阻）

芳环侧链反应

苯环侧链氧化

侧链氧化

不论侧链多长，$\text{KMnO}_4$ / $\text{K}_2\text{Cr}_2\text{O}_7$（强氧化剂）氧化烷基苯时，均氧化为苯甲酸：

$$\text{C}_6\text{H}_5\text{-R} \xrightarrow{\text{KMnO}_4 / \text{H}^+} \text{C}_6\text{H}_5\text{COOH}$$

前提：苯环相邻碳（$\alpha$-碳）上必须有氢原子（$\alpha\text{-H}$）。若 $\alpha$-碳上无氢（如叔丁基苯），不被氧化。

侧链催化加氢： $$\text{C}_6\text{H}_5\text{-CH=CH}_2 + \text{H}_2 \xrightarrow{\text{Ni}, 175°\text{C}, 180\;\text{atm}} \text{C}_6\text{H}_{11}\text{-CH}_2\text{CH}_3$$ 在高温高压下，苯环也可被加氢成为环己烷衍生物。
苄位（$\alpha$-碳）的特殊反应性：
  • R（苄位有 $\alpha\text{-H}$）：可氧化，被 $\text{KMnO}_4$ 氧化为 $\text{-COOH}$
  • R（苄位无 $\alpha\text{-H}$）：如叔丁基（$\text{-C(CH}_3)_3$），不被氧化

苯环侧链的卤代反应（自由基反应）

侧链卤代（自由基机理）
$$\text{C}_6\text{H}_5\text{CH}_3 \xrightarrow[h\nu]{\text{Cl}_2} \text{C}_6\text{H}_5\text{CH}_2\text{Cl} + \text{HCl}$$

在光照（$h\nu$）或过氧化物引发下，甲苯侧链发生自由基取代反应：

  • 链引发：$\text{Cl}_2 \xrightarrow{h\nu} 2\text{Cl}\cdot$
  • 链传递：$\text{Cl}\cdot + \text{C}_6\text{H}_5\text{CH}_3 \longrightarrow \text{C}_6\text{H}_5\text{CH}_2\cdot + \text{HCl}$
  • $\text{C}_6\text{H}_5\text{CH}_2\cdot + \text{Cl}_2 \longrightarrow \text{C}_6\text{H}_5\text{CH}_2\text{Cl} + \text{Cl}\cdot$

（七）F-C 反应判定点阵图

哪些底物不能发生 F-C 反应？

强吸电子基团取代的芳环不能发生 F-C 反应：

  • $\text{-NO}_2$（硝基苯）
  • $\text{-COR}$, $\text{-COOH}$（苯甲酰基/苯甲酸）
  • $\text{-SO}_3\text{H}$
  • 多卤代苯（环钝化严重）

不是所有芳环都能发生 F-C 反应！杂环（如吡啶）也不行。

F-C 反应的局限性总结
  • 傅-克反应需要加入催化剂（Lewis酸），如 $\text{AlCl}_3$, $\text{FeCl}_3$ 等
  • 傅-克烷基化反应的烷基化试剂可以用卤代烃、烯烃和醇等
  • 傅-克烷基化反应易发生重排反应，反应往往不能停在一元取代（多烷基化）
  • 傅-克酰基化反应不会生成多元取代的混合物

氯甲基化（Blanc反应）

(2) 第二类定位基 — 间位定位基：（均为吸电子基，钝化苯环）

记忆口诀：邻对位定位基 → 连接原子上有孤对电子（可给电子入环）；间位定位基 → 连接原子上有重键或正电荷（吸电子出环）。

合成应用与稠环芳烃

以上反应在有机合成中的应用

利用定位效应的多步合成策略

合成目标产物时，关键在于选择正确的反应顺序，利用定位效应引导取代基进入期望的位置。

合成实例

目标：合成对硝基苯甲酸

路线：

  1. 甲苯 $\xrightarrow[\text{AlCl}_3]{\text{CH}_3\text{COCl}}$ 对甲基苯乙酮 — $\text{-CH}_3$ 是邻对位定位基（F-C酰基化）
  2. $\xrightarrow{\text{KMnO}_4}$ 对甲基苯甲酸（氧化 $\text{-COCH}_3 \rightarrow \text{-COOH}$）
  3. 先氧化，再进行硝化（或调整顺序以得到正确的对位产物）
合成思路举例

例1：由甲苯合成对溴苯甲酸

  1. 甲苯 $\xrightarrow[\text{FeBr}_3]{\text{Br}_2}$ 对溴甲苯（$\text{-CH}_3$ 邻对位定位，取对位产物）
  2. 对溴甲苯 $\xrightarrow{\text{KMnO}_4 / \text{H}^+}$ 对溴苯甲酸（侧链氧化）

例2：由甲苯合成间溴苯甲酸

  1. 甲苯 $\xrightarrow{\text{KMnO}_4}$ 苯甲酸（先氧化）
  2. 苯甲酸 $\xrightarrow[\text{FeBr}_3]{\text{Br}_2}$ 间溴苯甲酸（$\text{-COOH}$ 是间位定位基）

例3：由苯合成对硝基苯甲酸

  1. 苯 $\xrightarrow[\text{AlCl}_3]{\text{CH}_3\text{Cl}}$ 甲苯
  2. 甲苯 $\xrightarrow[\text{混酸}]{\text{HNO}_3/\text{H}_2\text{SO}_4}$ 对硝基甲苯（$\text{-CH}_3$ 是邻对位定位基）
  3. 对硝基甲苯 $\xrightarrow{\text{KMnO}_4}$ 对硝基苯甲酸

关键原则：

  • 先引入邻对位定位基，再引入间位定位基 → 得到对位产物
  • 先氧化为 $\text{-COOH}$（间位定位基），再引入其他基团 → 得到间位产物
  • 注意：先硝化后氧化；先氧化后溴化；顺序不同→产物不同
合成中的注意事项
  • 侧链氧化（$\text{KMnO}_4$）将 $\text{-CH}_3$ 变为 $\text{-COOH}$：定位效应从邻对位定位变为间位定位
  • 利用磺酸基占位（磺化→占位→其他反应→脱磺）的策略
  • 注意 $\text{-NH}_2$ 的保护：先酰化保护为 $\text{-NHCOCH}_3$（仍为邻对位定位基，但降低活性，避免多取代和氧化），反应后再水解脱保护

稠环芳烃的定义

两个或多个苯环通过共用边（共用两个碳原子）稠合在一起的多环芳烃。

常见稠环芳烃：萘（naphthalene）、蒽（anthracene）、菲（phenanthrene）

1. 萘 $\text{C}_{10}\text{H}_8$（两个苯环稠合）

(1) 萘的结构：共平面

萘的四种键长：

  • $a = 0.136\;\text{nm}$
  • $b = 0.140\;\text{nm}$
  • $c = 0.142\;\text{nm}$
  • $d = 0.139\;\text{nm}$

10个碳原子均为 $sp^2$ 杂化，全部共平面。键长不完全相等，说明萘的键长平均化程度不如苯。

$\alpha$ 位的键 $a$ 较短（$0.136\;\text{nm}$），接近双键键长 → 这解释了萘的化学活性比苯高。

(2) 萘的位置编号

萘有两种不等价的氢原子位置：

  • $\alpha$ 位（C-1, C-4, C-5, C-8）：与稠合键相邻
  • $\beta$ 位（C-2, C-3, C-6, C-7）：与稠合键不直接相邻
a) 萘的取代反应

萘的亲电取代反应比苯容易得多（因为萘的键长平均化不如苯，芳香性稍弱，$\pi$ 电子更容易提供）。

亲电取代优先进入 $\alpha$ 位（动力学控制）：

  • $\alpha$ 位比 $\beta$ 位反应活性更高
  • 进攻 $\alpha$ 位形成的 $\sigma$ 络合物中间体更稳定（可画出更多不破坏另一个苯环芳香性的共振式）
亲电取代进攻 $\alpha$ 位 vs $\beta$ 位

进攻 $\alpha$ 位：$\sigma$ 络合物中间体可画出2个不破坏另一苯环芳香性的共振结构

进攻 $\beta$ 位：$\sigma$ 络合物中间体只能画出1个不破坏另一苯环芳香性的共振结构

→ $\alpha$ 位中间体更稳定，反应速率更快，$\alpha$ 取代为主

萘的取代规律总结
  • 动力学控制（低温）：亲电取代主要在 $\alpha$ 位
  • 热力学控制（高温）：如磺化在高温下，$\beta$ 位产物（$\beta$-萘磺酸）更稳定，为主要产物
  • 硝化反应：优先得到 $\alpha$-硝基萘（1-硝基萘）
磺化的温度效应：
  • 低温（$80°\text{C}$）磺化 → 主要得到 $\alpha$-萘磺酸（动力学产物）
  • 高温（$160°\text{C}$）磺化 → 主要得到 $\beta$-萘磺酸（热力学产物，$\beta$ 位产物空间位阻小，更稳定）
萘的氧化

萘比苯更容易被氧化：

  • 催化氧化：$\text{萘} \xrightarrow[\text{V}_2\text{O}_5]{\text{O}_2} \text{邻苯二甲酸酐}$（工业上重要反应）
  • $\text{CrO}_3$ 氧化 → 1,4-萘醌
蒽与菲：三个苯环的稠合方式不同。蒽为直线型稠合（linear），菲为角型稠合（angular）。菲的 9,10-位（K区）反应活性最高。
编号规则：从稠合键旁边的碳原子开始编号。

芳香性

Huckel 规则 [离王，有芳香性]
  1. 单环共轭多烯环，环外骨架共面（平面结构）
  2. 环上每个原子都有 $p$ 轨道
  3. 环域 $\pi$ 电子数为 $4n+2$（$n=0,1,2...$）

芳香性判据

(1) 亲电取代反应（首选）

  • $+ \text{E}^+ \longrightarrow +\text{H}^+$，$\text{E}^+ = \text{NO}_2^+, \text{X}^+, \text{R}^+-\text{SO}_3\text{H}, \text{R-}\dot{\text{C}}\text{O}^+$

(2) 磁环流

  • $\vec{E}$ 环流（顺H, 即为 $p$ 电子环流）
  • 芳香性：环外质子低场移动（$\delta$偏大）；环内质子高场移动（$\delta$偏小）
  • 反芳香性：正好相反

(3) 热力学稳定性

  • $\Delta H$ 比较：如氢化热低于理论值，说明有额外稳定化能

非苯芳烃

(4) 满足 Huckel 规则的非苯芳烃：

  • 轮烯：连续双键 $\rightarrow$ 必须是 $4n+2$ 个 $\pi$ 电子
  • [10]轮烯、[14]轮烯、[18]轮烯 → 有芳香性
  • [4]轮烯、[8]轮烯 → 反芳香性（$4n$ 个 $\pi$ 电子）

芳香离子

(5) 具有芳香性的离子和杂环化合物：

  • 环丙烯正离子 $\text{C}_3\text{H}_3^+$：$2$ 个 $\pi$ 电子（$4 \times 0 + 2 = 2$），芳香性
  • 环戊二烯负离子 $\text{C}_5\text{H}_5^-$：$6$ 个 $\pi$ 电子（$4 \times 1 + 2 = 6$），芳香性
  • 环庚三烯正离子（对应"卓"） $\text{C}_7\text{H}_7^+$：$6$ 个 $\pi$ 电子，芳香性
判断步骤：①确认是否共面且环上每个原子都有 $p$ 轨道 → ②数 $\pi$ 电子数 → ③是否 $4n+2$

(6) 薁（甘菊环）= 五元环 + 七元环稠合

  • $10$ 个 $\pi$ 电子（$4 \times 2 + 2 = 10$），有芳香性
  • 偶极矩较大：五元环端带部分负电，七元环端带部分正电

(7) 稠环芳烃的判断：

第六章 对映异构体

概念

手性：物质的实体与镜像不能完全重叠的性质。[相当于左手和右手的关系]

方向性与条件：①没有对称面、对称中心、$S_4$交替轴 → 才有手性

R/S 构型

由 CIP 优先次序规则判断：将最小基团朝离观察者最远方向（向后），剩余三个基团按优先次序排列：

  • 顺时针 → R 构型
  • 逆时针 → S 构型
有旋光性时，每台旋光的标记为 $R/S$

ee% 光学纯度

对映体过量百分率
$$ee\% = \frac{[R]-[S]}{[R]+[S]} \times 100\%$$

也可表示为：

$$ee\% = \frac{[\alpha]_{\text{obs}}}{[\alpha]_{\max}} \times 100\%$$

外消旋体的 $ee\% = 0$，纯对映体的 $ee\% = 100\%$

对映体 vs 非对映体

  • 对映体（enantiomers）：互为镜像关系的立体异构体，所有手性碳构型全部相反
  • 非对映体（diastereomers）：不是镜像关系的立体异构体，有些手性碳相同/有些相反
两个 $E/Z$ 构型化合物不能算对映体和非对映体！指的是手性碳的构型关系。

指甲利对映体：$R$和$S$之间是对映关系，$RR$和$SS$之间也是对映关系

外消旋体：$R/S$ 等量混合物（等量+旋光相互抵消），旋光度 = 0

$M$ 构型：| 以 $\dot{C}$ 为中心，$4$个不同基团按CIP优先序从 $1$ 到 $4$ 排列后的左旋/右旋判断排列方向 |

$(+)$：右旋（dextrorotatory）   $(-)$：左旋（levorotatory）  （与 $R/S$ 无必然对应关系，需实验测定）

构型的表示方法

1. 透视式（楔形式）

用楔形键、虚线键表示空间方向：

  • 实楔形：伸向观察者（前方）
  • 虚线楔形：远离观察者（后方）
  • 要标清楚氢的位置，可直观判断 $R/S$

2. Fischer 投影式

"横前竖后"：横线上的取代基朝向观察者（前方），竖线上的取代基远离观察者（后方）

Fischer 投影式规则
  • ①投影后为"十"字，手性碳C在交叉点
  • ②竖直方向固定（碳链纵向），最高氧化态碳在上方
  • ③旋转 $180°$ 后，构型不变；旋转 $90°$ 后，构型改变
  • ④不能离开纸面翻转（翻转 = 构型改变），不能随意交换两个基团的位置
相互转化：Fischer投影式、Sawhorse（锯架式）、Newman投影式之间可以相互转化，但要注意规则。

比旋光度

比旋光度公式：

$$[\alpha]_D^{t} = \frac{\alpha}{l \cdot c}$$

大写希腊 $\alpha$ = 实测旋光度，$l$ = 光程长度（dm），$c$ = 浓度（g/mL），$t$ = 温度，$D$ = 钠光波长（589 nm）

构型标记方法：D-L 构型标记法

D-L 构型标记法：糖、氨基酸等化合物的标记中使用最广。

D-L 标记规则
  • 以甘油醛为标准进行标记（Fischer投影式）。实质是指与标准物的构型关系，右为 D，左为 L。
  • 以甘油醛为参考标准，看不改变手性碳构型的条件下，能否从D-甘油醛或L-甘油醛出发合成该化合物 → 即定为 $D$ 或 $L$。

D-甘油醛（$-\text{OH}$ 在右）  →   L-甘油醛（$-\text{OH}$ 在左）

D-丙氨酸     L-丙氨酸     D-(-)-甘油醛     L-(+)-甘油醛

D, L 表示构型，(+) 为右旋，(-) 为左旋。D 不一定是右旋，L 不一定是左旋。需由实验测定旋光方向。

  • 顺式：两个取代基在环平面同侧

亲核性 vs 碱性

亲核性与碱性的区别

(1) 一般同族：碱性越强 → 亲核性越强。如 $\text{NH}_2^- > \text{HO}^- > \text{F}^-$

(2) 可极化度增大 → 亲核性增强。如 $\text{HS}^- > \text{HO}^-$，$\text{I}^- > \text{Cl}^-$

试剂的碱性与亲核性不是一回事！
碱性用 $\text{p}K_a$ 表示，通过化学平衡测量，描述其与质子结合的能力。
亲核性是描述其与碳（原子/底物中心）成键的能力。

例如：$\text{HO}^- > \text{CN}^-$（碱性）   vs   $\text{CN}^- > \text{HO}^-$（亲核性）

消除反应 (E)

定义

$\beta$-消除：从相邻的两个碳原子上脱去$\ce{H}$和一个基团（如$\ce{-X}$），生成碳碳双键或三键的反应。

$$\ce{-C(-H)-C(-X)- -> C=C + HX}$$

碱的作用是脱去$\beta$位的$\ce{H}$并与离去基一起产生$\Delta +\ce{HX}$。

消除反应的分类

  • (1) $\alpha$-消除（1,1-消除）：从同一碳上脱去$\ce{H}$和$\ce{X}$ $\longrightarrow$ 形成卡宾（$\ce{:CR2}$），如$\ce{CHCl3 ->[\text{强碱}] :CCl2}$
  • (2) $\beta$-消除（1,2-消除）：卤代烃的消除反应主要为$\ce{1,2}$-消除
  • 按速度规律分类：双分子消除反应$\ce{E2}$和单分子消除反应$\ce{E1}$
消除反应分类
  • $\ce{E1}$：单分子消除
  • $\ce{E2}$：双分子消除反应

(1) $\ce{E2}$ 反应（双分子消除）

$\ce{E2}$反应可以理解为：碱进攻$\beta$-$\ce{H}$，同时$\ce{C-X}$键断裂，一步完成。

例：

$$\ce{C2H5O^- + CH3CH2Br ->[\text{EtOH}] CH2=CH2 + EtOH + Br^-}$$
注意区别：这种碱进攻$\beta\text{-H}$的反应是$\ce{E2}$消除反应。而碱直接进攻碳的是$\ce{S_N2}$反应。
E2与S_N2竞争：在$\ce{EtO^-/EtOH}$条件下，反应实际上同时发生$\ce{E2}$消除和$\ce{S_N2}$取代反应。

(2) $\ce{E1}$ 消除

E1消除：分两步，先离解后消除。

$$\ce{R_2C(-H)-C(-X)R_2 -> R_2C^+-CR_2H + X^- -> R_2C=CR_2 + H^+}$$

第一步：$\ce{C-X}$键断裂，生成碳正离子（与$\ce{S_N1}$第一步相同）；

第二步：碳正离子脱去$\beta$-$\ce{H}$，形成双键。

S_N1 与 E1 共享第一步！碳正离子形成后，若亲核试剂进攻碳正离子 $\to$ $\ce{S_N1}$取代；若碱夺取$\beta$-$\ce{H}$ $\to$ $\ce{E1}$消除。

升温有利于消除反应（$\ce{E1}$），因为升温后有利于形成更稳定的烯烃产物。

(3) 消除反应的取向问题

① $\ce{E1}$ 和Saytzeff（扎伊采夫）规则

过渡态：$\alpha$-C上的$\beta$-H被脱去。

Saytzeff规则

消除反应中，优先脱去含氢较少的$\beta$碳上的$\ce{H}$，生成取代基较多的烯烃（更稳定的烯烃）。

$$\ce{H-C=C-CH3}$$

双取代 < 三取代 < 四取代烯烃（稳定性依次增大）

② $\ce{E2}$ 的取向

当消除基团较大、$\beta$-$\ce{H}$位阻较大时，进攻试剂体积又大时，重排反应较大（Hofmann消除）。

E2的特点
  • 进攻试剂 $\longleftrightarrow$ $\beta$-$\ce{H}$方向
  • 取大体积的$\beta$-$\ce{H}$受阻时，导致反应取向相反
  • 也遵循Saytzeff规则（大多数情况下）

$\ce{E2}$ 是反式共面消除

反式消除 (anti-periplanar)

$\ce{E2}$反应要求$\ce{H}$和$\ce{X}$处于反式共面（anti-periplanar）构象，即$\ce{H-C-C-X}$的二面角为$180°$。

这是$\ce{E2}$反应的立体电子效应要求。

② 进攻试剂

碱性较强，趋向于消除。

极性弱，处于溶剂对其有利消除反应的一侧。

消除反应的活化能通常要高于$\ce{C-H}$亲核取代反应的活化能，升温有利于消除。

高温有利于消除反应，低温有利于取代反应。

醇的结构

根据与羟基相连的碳原子的级数不同，醇分为：

(2) 命名

1) 含−OH的最长碳链作为主链，C−OH 不在母体的碳链上。

2) 伯醇和仲醇属于同一类R和H组的顺反异构。

3) 如: $\text{CH}_3\text{CH(OH)CH}_3$ — 2-丙醇

多种官能团同时存在，按优先级确定主官能团名字。

醇的命名：
类别IUPAC名结构

(3) 物理性质

  • 甲/乙/丙/丁醇与水混溶，1-戊醇以上微溶，有氢键

$\ce{ROH + ZnCl2 -> R-O-ZnCl + HCl}$

1. 异丙苯氧化（工业法）

异丙苯法制苯酚

$\ce{C6H5CH(CH3)2 ->[\ce{O2}][\text{催化}] C6H5C(CH3)2OOH ->[\ce{H+}] C6H5OH + CH3COCH3}$

异丙苯（cumene）先氧化为过氧化异丙苯，再在酸催化下分解为苯酚和丙酮。

工业上最重要的苯酚制备方法，同时得到丙酮副产物。

2. Ar−Cl 水解法

$\ce{Ar-Cl ->[\ce{NaOH(aq)}][\text{高温高压}] Ar-OH}$

芳环上的 C−Cl 键很稳定（p-π 共轭），需要高温高压条件下水解。

3. 磺酸钠碱熔法

碱熔法

$\ce{Ar-SO3H ->[\ce{NaOH}] Ar-SO3Na ->[\ce{NaOH}][\text{300-350°C 碱熔}] Ar-ONa ->[\ce{H+}] Ar-OH}$

磺化 → 磺酸钠 → 碱熔（与 NaOH 固体在 300–350°C 熔融） → 酚钠 → 酸化得酚。

4. 重氮盐水解

重氮盐法

$\ce{Ar-NH2 ->[\ce{NaNO2}/\ce{H2SO4}][\text{0-5°C}] Ar-N2+ HSO4- ->[\ce{H2O}][\Delta] Ar-OH + N2 ^ + H2SO4}$

芳胺重氮化后水解，适合制备特定取代酚。

重氮盐的制备需在低温（0–5°C）进行以防止分解，然后加热水解。
补充：含卤芳烃的 Br 可先用 NaNO2、H2SO4 进行重氮化处理：

$\ce{ArNH2 + NaNO2 + H2SO4 -> ArN2+HSO4- ->[\ce{H3PO2}] ArH}$（还原脱氨基）

$\ce{ArN2+HSO4- + Br- -> ArBr + N2}$（Sandmeyer 反应）

$\ce{ArN2+HSO4- + H2O -> ArOH + N2}$（水解制酚）

醚的结构与性质

1) 结构

醚的结构

通式：$\ce{R-O-R'}$，氧为 $sp^3$ 杂化，碳氧碳键角 ~112°。

2) 命名与分类

$\ce{R-O-R'}$：$R = R'$ 时为简单醚（对称醚）；例：$\ce{CH3OCH3}$（二甲醚/甲醚）

$R \ne R'$ 时为混合醚（不对称醚）；例：$\ce{CH3OC2H5}$（甲乙醚）

  • 甲醚（二甲醚）
  • 混合醚：大基团为母体，小基团为取代基（"甲氧基"、"乙氧基"）
  • 环醚：环氧乙烷（oxirane），四氢呋喃（THF），1,4-二氧六环等

3) 对称醚的鉴别

对比结构：醚 $\ce{C_{n}H_{2n+2}O}$ 与 醇 $\ce{C_{n}H_{2n+2}O}$ 互为同分异构体。

醚对碱、对稀酸（常温下）均不反应 → 化学性质"惰性"（相对醇而言）。

7) 物理性质

  • 分子间无氢键，沸点比醇低
  • DI 型分子间力：沸点与 $\ce{H2O}$ 交叉溶解性下降
  • 化学惰性：常用作反应溶剂

(四) 醚的化学性质

1. 氧鎓盐的生成 -- 溶液质酸

醚的碱性 -- 形成氧鎓盐

醚中的氧原子有孤对电子，可以接受质子形成氧鎓盐：

$$R\text{-}O\text{-}R + H^+ \longrightarrow R\text{-}\overset{+}{O}H\text{-}R$$

因此醚可以溶解在浓酸中：

$$R\text{-}\overset{+}{O}H\text{-}R \longrightarrow R\text{-}F + R'\text{-}OH \quad (\text{或 } R' \text{-}X)$$

2. 与路易斯酸形成络合物

醚可以与路易斯酸（如 $BF_3$、$AlCl_3$）形成络合物：

$$R_2O + BF_3 \longrightarrow R_2O \cdot BF_3$$

$BF_3 \cdot Et_2O$（三氟化硼乙醚络合物）是常用的路易斯酸催化剂。

3. 醚键的断裂（需要较强的酸，如 HX，X = I, Br）

醚键断裂机理
$$CH_3\text{-}O\text{-}CH_2CH_3 + HI \longrightarrow CH_3\text{-}\overset{+}{O}H\text{-}CH_2CH_3 + I^-$$ $$\longrightarrow CH_3I + C_2H_5OH$$

或反过来：

$$C_2H_5I + CH_3OH$$
反应规律：在含不同烃基的醚中，$I^-$ 攻击 $C\text{-}O$ 键时遵循以下规律：
• 当为 SN2 机理时，$I^-$ 攻击较小的烃基（位阻小）；断裂后生成较小烃基的碘代烃
• 当含有叔碳时，倾向于 SN1 机理
示例 1（SN2）：
$CH_3\text{-}O\text{-}CH_3 + HI \longrightarrow CH_3I + CH_3OH$
（$I^-$ 攻击甲基）
示例 2（芳基醚）：
$\varphi\text{-}O\text{-}CH_3 + HI \longrightarrow \varphi\text{-}OH + CH_3I$
（芳基碳 $sp^2$ 不能发生 SN2，所以 $I^-$ 只能攻击甲基）
叔丁基醚在与 HX 反应时倾向于在叔碳位上断裂（SN1 机理）。
混合醚断裂举例
$$CH_3\text{-}O\text{-}C(CH_3)_3 + H^+ \longrightarrow CH_3\text{-}\overset{+}{O}H\text{-}C(CH_3)_3$$

然后：叔碳正离子离去（SN1），生成 $(CH_3)_3C^+$，再与 $X^-$ 结合。

$$\longrightarrow CH_3OH + (CH_3)_3CX$$

4. 过氧化物的生成

$$CH_3\text{-}O\text{-}C_2H_5 \xrightarrow{O_2} CH_3\text{-}O\text{-}\overset{O\text{-}O\text{-}H}{|}CH\text{-}CH_3 \quad (\text{含 }\color{red}{-O\text{-}O-}\text{ 键})$$

醚中 $\alpha$-H 容易被氧化生成过氧化物（含 $-O-O-$ 键）。

检测方法：加入 $FeCl_3$ 和 $KSCN$，若有过氧化物：
• 防止：加入抗氧剂（对苯二酚）
• 检测：加入还原剂（$Fe^{2+}$）
$$Fe^{2+} \xrightarrow{H_2O_2} Fe^{3+} \xrightarrow{SCN^-} \text{血红色}$$

5. 环氧化物的性质

a) 合成

环氧化物的制备（Williamson 合成法）

由邻卤醇在碱的作用下关环：

$$CH_2\text{-}CH_2 \xrightarrow[\text{(1) } Cl_2/H_2O]{\text{(2) } NaOH} \quad \text{环氧乙烷}$$

卤醇法（$Cl_2/H_2O$）：

$$CH_2\!=\!CH_2 \xrightarrow{Cl_2, H_2O} ClCH_2CH_2OH \xrightarrow{NaOH} \text{环氧乙烷}$$

其他方法：

  • 过氧酸氧化：$RCH\!=\!CHR' \xrightarrow{mCPBA} \text{环氧化物}$ （mCPBA = 间氯过氧苯甲酸）
  • 直接工业法：$CH_2\!=\!CH_2 \xrightarrow{O_2, Ag} \text{环氧乙烷}$
HOCl 加成法：先 $Cl_2/H_2O$ 得卤醇，再用 $NaOH$ 闭环（Markovnikov 方向添加 OH，anti 加成）。

b) 开环反应

环氧化物开环反应总结

酸催化开环

酸催化条件下，亲核试剂进攻取代基较多的碳（类似 SN1，Markovnikov 方向）：

$$\text{环氧化物} \xrightarrow{H^+/ROH} \text{反式-二醇或醚醇}$$
  • $H^+/H_2O$ → 反式 1,2-二醇
  • $H^+/ROH$ → 醚醇
  • $H^+/HX$ → 卤醇

碱催化开环 / 亲核试剂开环

碱性条件下（SN2），亲核试剂进攻取代基较少的碳（anti-Markovnikov，位阻控制）：

$$\text{环氧丙烷} \xrightarrow{CH_3O^-} CH_3O\text{-}CH_2CH(OH)CH_3$$

常见亲核试剂开环：

  • $Grignard$ 试剂（$RMgX$）→ 增长碳链
  • $CH_3COO^-$ → 酯
  • $HCN$ → 氰基化合物
  • $NH_3$ → 氨基醇
  • $LiAlH_4$ → 醇（还原开环）
开环方向判断口诀：
• 酸催化：亲核试剂攻击取代基多的碳 → Markovnikov
• 碱催化：亲核试剂攻击取代基少的碳 → anti-Markovnikov（SN2 位阻控制）

c) 酸催化开环（续：环氧化物的开环选择性与立体化学）

醚键断裂的 SN2 与 SN1 竞争

环氧化合物的酸催化开环，既有 SN2 也有 SN1 的特征：

$$CH_3\text{-}CH\text{-}CH_2 \xrightarrow{H^+} CH_3\text{-}\overset{+}{C}H\text{-}CH_2 \longrightarrow \text{开环产物}$$

酸催化开环先质子化再开环，然后 $Nu$ 进攻碳正离子（有 SN1 特征）。

醚键断裂的构型保持与翻转

醚键断裂的机理：取决于底物结构：

  • 若断裂处为伯碳或仲碳：SN2 机理 → 构型翻转（Walden翻转）
  • 若断裂处为叔碳：SN1 机理 → 构型难以预测（外消旋化）
如果进攻的亲核试剂是带负电的：按照 SN2 处理；若亲核试剂较弱（中性），酸催化，按照碳正离子稳定性判断位置。
记忆：如果是酸催化的醚的裂解，本质是 均裂 + 异裂。如果速度快的醚键的裂解是有利的，按照碳正离子稳定性判断。
2-甲基环氧丙烷的开环：
  • 酸催化：$H^+/CH_3OH$ → 亲核试剂攻击取代基多的碳（2号碳）→ 2-甲氧基-1-丙醇
  • 碱催化：$CH_3O^-$ → 攻击取代基少的碳（1号碳）→ 1-甲氧基-2-丙醇

d) 碱性条件下的开环反应（SN2 反应）

碱性开环的区域选择性
$$\text{环氧丙烷} + CH_3O^- \xrightarrow{S_N2} CH_3O\text{-}CH_2\text{-}CHOH\text{-}CH_3$$

SN2 机理：亲核试剂进攻位阻小的碳（取代基少的碳）。

构型翻转：产物构型与原料相反（Walden 翻转）。

(5) 聚醚反应

环氧乙烷的聚合
$$HOCH_2CH_2OH + n \underset{\text{环氧乙烷}}{\overset{O}{\frown}} \xrightarrow{} HO(CH_2CH_2O)_nCH_2CH_2OH$$

环氧乙烷开环聚合：生成聚乙二醇（PEG）

$$CH_3(CH_2)_nCH_2OH + n \underset{\text{环氧乙烷}}{\overset{O}{\frown}} \xrightarrow{OH^-} CH_3(CH_2)_nCH_2(OCH_2CH_2)_nOH$$

此即非离子型表面活性剂的合成方法。

缩水甘油缩合：
$$HOCH_2 - \underset{\text{环氧}}{\overset{O}{\frown}} - CH_2 \longrightarrow \text{聚合物}$$

烯丙基乙烯基醚在加热条件下发生 [3,3]-$\sigma$ 迁移：

芳香族 Claisen 重排

苯基烯丙基醚的 Claisen 重排分两步：

  1. 邻位 Claisen 重排：烯丙基先迁移到邻位（生成环己二烯酮中间体），然后互变异构得到邻烯丙基酚。
  2. 对位 Claisen 重排：若邻位被占据，烯丙基经 Cope 重排 迁移到对位。
$$\text{苯基烯丙基醚} \xrightarrow{200\,°C} \text{邻-烯丙基酚} \quad (\text{邻位})$$ $$\text{邻位被封} \xrightarrow{200\,°C} \text{对-烯丙基酚} \quad (\text{对位})$$
注意：Claisen 重排后的产物中，烯丙基发生了翻转（即原来连接氧的碳变成了远端碳）。

应用

Claisen 重排的合成价值：可以用于在芳环上特定位置引入碳链（邻位或对位烯丙基化）。这是一种 $C\text{-}C$ 键形成的重要方法。

(3) Williamson 醚合成法

Williamson 合成
$$RX + NaOR' \longrightarrow ROR' + NaX$$

SN2 反应，卤代烃与醇钠反应生成醚。

副反应：

  • $RX$ 与 $NaOR'$：$RX + R'O^-Na^+ \longrightarrow R\text{-}O\text{-}R' + NaX$（期望产物）
  • 若 $RX$ 为叔卤代烃：以消除反应（E2）为主，而非取代。
Williamson 合成的适用范围：
• 卤代烃最好是伯卤代烃（SN2 易进行，副反应少）
• 仲卤代烃也可以但产率降低
• 叔卤代烃不能用（主要发生 E2 消除）

冠醚 (Crown Ethers)

Wolff-Kishner-黄鸣龙碱性（$NH_2NH_2/KOH$）亚甲基 ($-CH_2-$)

(5) 形成缩醛/缩酮

缩醛/缩酮的生成
$$R\text{-}CHO + 2 R'OH \xrightarrow{H^+} R\text{-}CH(OR')_2 + H_2O \quad (\text{缩醛})$$ $$R_2C\!=\!O + 2 R'OH \xrightarrow{H^+} R_2C(OR')_2 + H_2O \quad (\text{缩酮})$$

缩醛/缩酮在碱性条件下稳定，可用作羰基保护基。

其他反应

(1) $RX \xrightarrow{NaCN 或 Mg} R\text{-}OH \quad (\text{格氏试剂法或氰基法增长碳链})$

(2) 羟醛缩合：

$$2\;R\text{-}CHO \xrightarrow{NaOH, \text{稀}} R\text{-}CH(OH)\text{-}CH(R)\text{-}CHO$$

(3) Wittig 反应：

$$R_2C\!=\!O + Ph_3P\!=\!CHR' \longrightarrow R_2C\!=\!CHR' + Ph_3P\!=\!O$$

Wittig 试剂将醛酮转化为烯烃。

(4) 坎尼扎罗 (Cannizzaro) 反应（无 $\alpha$-H 的醛）：

$$2\;\text{HCHO} \xrightarrow{NaOH} HCOONa + CH_3OH$$

一分子醛被氧化为羧酸盐，另一分子被还原为醇。

Baeyer-Villiger 氧化（用 mCPBA / $H_2O_2$）：
$$\text{酮} \xrightarrow{mCPBA} \text{酯}$$ 将酮氧化为酯，迁移基团的迁移能力：叔烷基 > 仲烷基 > 苯基 > 伯烷基 > 甲基。
注意：这是一个酮到酯的转化。
$$R\text{-}CO\text{-}R' \xrightarrow{mCPBA} R\text{-}COO\text{-}R' \quad (\text{或} R'\text{-}COO\text{-}R)$$
醇到醛的选择性氧化试剂：$RCH_2OH \xrightarrow{PCC 或 Swern} RCHO$（不会过度氧化到羧酸）。
第 41 页

酚

(1) 酸性（比较）

酚的酸性

酚的酸性比醇强，但比碳酸弱：

$$PhOH + Na_2CO_3 \longrightarrow NaOH + PhONa$$

即酚可以与 $Na_2CO_3$（碳酸钠）反应，但不能与 $NaHCO_3$ 反应。（注意：酚不能使石蕊变色。）

(2) FeCl3 显色

$FeCl_3$ 与酚反应显蓝紫色。

这是酚类化合物的特征鉴定反应。

(3) 芳环上的亲电取代反应

酚的芳环反应

酚羟基（$-OH$）是强活化基团和邻对位定位基：

卤代反应

  • 溴代：
    • $PhOH + 3Br_2 \longrightarrow 2,4,6\text{-}三溴酚 \downarrow + 3HBr$（白色沉淀，定量反应）
    • 这是酚的鉴定反应之一

硝化反应

  • 稀 $HNO_3$（室温）→ 邻硝基酚 + 对硝基酚
  • 浓 $HNO_3/H_2SO_4$ → 2,4,6-三硝基酚（苦味酸）

磺化反应

  • 低温（RT）→ 邻位产物为主
  • 高温 → 对位产物为主（热力学控制）

Friedel-Crafts 反应

  • 酚 + $RX \xrightarrow{AlCl_3}$ 烷基化（但要注意多烷基化）
  • 酚 + $RCOCl \xrightarrow{AlCl_3}$ 酰基化

Kolbe-Schmitt 反应

  • $PhONa + CO_2 \xrightarrow{高温高压} \text{水杨酸（邻羟基苯甲酸）}$

Reimer-Tiemann 反应

  • $PhOH + CHCl_3 \xrightarrow{NaOH} \text{邻羟基苯甲醛（水杨醛）}$

与甲醛缩合

  • 酚 + $HCHO$ → 酚醛树脂（高分子材料）

(4) 氧化反应

酚容易被氧化：

$$\text{PhOH} \xrightarrow{[O]} \text{对苯醌} \quad (p\text{-benzoquinone})$$

这就是酚在空气中变色发黑的原因。

(5) 酯化（酚入酰氯反应）

酚的酯化

酚 + 酰氯（或酸酐）：

$$PhOH + R\text{-}CH\!=\!CH_2 \xrightarrow{酸催化} PhO\text{-}CHR\text{-}CH_3 \quad (C^0\text{烷基化为主})$$

酚 + 酰氯：

$$PhOH + RCOCl \longrightarrow PhOCOR + HCl$$

Fries 重排：酚酯在 $AlCl_3$ 催化下重排为羟基芳酮：

  • 低温 → 对位产物
  • 高温 → 邻位产物

酚的制备

酚的制备方法
  1. 异丙苯氧化法（工业法）： $$\text{异丙苯} \xrightarrow{O_2} \text{异丙苯过氧化氢} \xrightarrow{H^+} PhOH + (CH_3)_2CO$$
  2. 磺化碱熔法： $$PhH \xrightarrow{H_2SO_4} PhSO_3H \xrightarrow{NaOH, 熔融} PhONa \xrightarrow{H^+} PhOH$$
  3. 卤代芳烃水解： $$PhCl \xrightarrow{NaOH, 高温高压} PhONa \xrightarrow{H^+} PhOH$$
  4. 重氮盐水解： $$ArN_2^+ \xrightarrow{H_2O, \Delta} ArOH + N_2 + H^+$$

$SO_3H$ 到 $OH$ 的转化：

$$R\text{-}SO_3H \xrightarrow{NaOH, 熔融} R\text{-}OH$$ $$R\text{-}SO_3H \xrightarrow{PCl_5} R\text{-}Cl \xrightarrow{} \cdots$$

(2) 醚键的制备

醚的制备方法总结

a) 醇脱水：

$$2\;R\text{-}OH \xrightarrow{H_2SO_4} R\text{-}O\text{-}R + H_2O$$

适用于伯醇制对称醚（仅限 $1°$ 醇，$2°/3°$ 醇易消除）。

b) Williamson 合成：

$$R\text{-}OM \xrightarrow{\Delta} R\text{-}O\text{-}R' \quad [\text{回扣}]$$
制备方法
$$RX + NaOR' \longrightarrow ROR' + NaX$$

（仲卤代烃会产生更多的 E2 副反应消除产物。）

过氧化物

醚在空气中放置容易生成过氧化物：

$$R\text{-}O\text{-}CHR' + O_2 \longrightarrow R\text{-}O\text{-}C(OOH)\text{-}R'$$

过氧化物受热易爆炸，使用前需要检测。

(3) Claisen 重排 & Cope 重排

Claisen 重排

烯丙基芳基醚的 [3,3]-$\sigma$ 迁移：

$$\text{烯丙基苯基醚} \xrightarrow{加热} \text{邻-烯丙基酚}$$

若邻位被封闭，则迁移到对位。

1,5-己二烯型底物的 [3,3]-$\sigma$ 迁移：

$$\text{1,5-己二烯} \xrightarrow{\Delta} \text{异构化的1,5-己二烯}$$

Williamson 合成中的 SN2 vs E2
