一、IUPAC系统命名法通用步骤

  1. 1选母体：找含主要官能团的最长碳链（或最大环）作为母体
  2. 2编号：从离主要官能团最近的一端开始编号，使官能团位次最小
  3. 3命名取代基：把支链/取代基按名称列出
  4. 4排列顺序：取代基按英文字母序（或中文笔画序）排列
  5. 5写全名：取代基位次-取代基名称-母体名称
母体选择优先级：含官能团 > 最长 > 支链最多 > 取代基位次之和最小

三、支链烷烃命名（五步法）

五步法
  1. 1选主链：含取代基最多的最长碳链
    口诀：长 > 多 > 近（最长 → 支链最多 → 取代基位次最小）
  2. 2编号：从离取代基最近的一端开始编号，使取代基位次之和最小
    最低系列原则：逐位比较法，如 [2,3,5] 优于 [2,4,5]（第二位 3 < 4）
  3. 3命名取代基：确定各支链的名称
  4. 4排列：多个取代基按先小后大（中文）或字母序（英文）排列
  5. 5写全名：位次用阿拉伯数字，位次之间用逗号，位次与名称之间用短线
实例：

$\text{CH}_3\text{CH(CH}_3\text{)CH}_2\text{CH(C}_2\text{H}_5\text{)CH}_2\text{CH}_3$

主链：6个碳 → 己烷；取代基：2-位甲基、4-位乙基

命名：4-乙基-2-甲基己烷（英文字母 E 在 M 前）

复杂取代基命名
  • 取代基本身有支链时，用圆括号括起
  • 从与主链直接相连的碳开始编号（1' 碳）
  • 例：$\text{5-(1,1-二甲基丙基)-2-甲基癸烷}$

四、环烷烃命名

单环烷烃

  1. 无取代基：直接在烷烃名前加"环"字 → 环丙烷、环丁烷、环戊烷、环己烷
  2. 仅1个取代基：无需编号 → 如 甲基环己烷
  3. 多个取代基：编号使取代基位次之和最小，按字母序排列
环 vs 链：谁做母体？
  • 环碳数 $\geq$ 链碳数 → 环为母体，链为取代基
  • 环碳数 $\leq 3$ 且链碳数 > 环碳数 → 链为母体，环为取代基
  • 例：丁基环丙烷 → 写成"环丙基丁烷"（链4 > 环3，链做母体）

多环环烷烃的环数判断

公式
$$\text{环数} = \text{C-C 键数} - \text{碳原子数} + 1$$

五、桥环化合物命名

桥环命名五条规则
  1. 1"二环"作为词头，总碳数定母体名称（如"二环[2.2.1]庚烷"= 7个碳）
  2. 2找桥头碳（两环共用的碳）→ 其余为桥碳
  3. 3编号：桥头碳开始 → 最长桥 → 次长桥 → 最短桥
  4. 4各桥碳数由大到小排列，用圆点分隔，放在方括号中
  5. 5有取代基时，位次尽可能小，写在"二环"之前
实例：
  • 降冰片烷 = 二环[2.2.1]庚烷（三个桥分别有 2、2、1 个碳）
  • 7,7-二甲基二环[4.1.0]庚烷

六、螺环化合物命名

螺环命名四条规则
  1. 1"螺"作词头，按总碳数命名（两环共一个螺原子）
  2. 2编号：从较小环邻接螺原子的碳开始 → 小环 → 螺原子 → 大环
  3. 3方括号中数字 = 两环除螺原子外的碳数，小到大排列
  4. 4有取代基时，位次尽可能小
实例：
  • 螺[2.4]庚烷（小环2碳 + 螺原子 + 大环4碳 = 7碳）
  • 螺[4.5]癸烷（4 + 1 + 5 = 10碳）
  • 5-甲基螺[3.4]辛烷

七、烯烃命名

烯烃系统命名规则
  1. 选含 C=C 双键的最长碳链为主链
  2. 从靠近双键的一端开始编号（双键位次尽可能小）
  3. 双键位次以编号较小的碳表示，写在"烯"字前面
  4. 取代基的命名和编号与烷烃相同
实例：
  • $\text{CH}_3\text{CH=CHCH}_2\text{CH}_3$ → 2-戊烯（pent-2-ene）
  • $\text{CH}_2\text{=C(CH}_3\text{)CH}_2\text{CH}_3$ → 2-甲基-1-丁烯
  • 环己烯：双键碳编为 1,2 位

八、炔烃命名 & 烯炔共存

炔烃命名规则
  1. 选含 $\text{C}\!\equiv\!\text{C}$ 叁键的最长碳链为主链
  2. 从靠近叁键的一端开始编号
  3. 叁键位次写在"炔"字前面
烯炔共存时的命名规则：
  1. 选同时包含双键和叁键的最长碳链为母体
  2. 编号使双键和叁键的位次之和最小
  3. 若位次相同时，双键编号优先（双键位次更小）
  4. 母体名称："X-烯-Y-炔"
实例：

$\text{HC}\!\equiv\!\text{C-CH}_2\text{-CH=CH}_2$ → 1-戊烯-4-炔（pent-1-en-4-yne）

（双键编号 1，叁键编号 4；若反向编号则为 4-烯-1-炔，位次和相同但双键优先取小号）

九、CIP 次序规则（Cahn-Ingold-Prelog）

CIP次序规则是 Z/E命名 和 R/S构型 的共同基础，务必熟练掌握。

规则一：原子序数大者优先
$$\text{I}(53) > \text{Br}(35) > \text{Cl}(17) > \text{S}(16) > \text{O}(8) > \text{N}(7) > \text{C}(6) > \text{H}(1)$$

同位素按原子质量排列：$\text{D} > \text{H}$，${}^{14}\text{C} > {}^{12}\text{C}$

规则二：逐层外推比较法
  • 第一层：比较直接与立体中心相连的原子
  • 若相同 → 比较第二层（各原子各自连的原子）
  • 再相同 → 比较第三层……依此类推直到分出高低
规则三：不饱和键的拆分处理
键类型处理方式
$\text{C=C}$每个 C 各补一个虚拟 C 原子
$\text{C=O}$C 补虚拟 O，O 补虚拟 C
$\text{C}\!\equiv\!\text{C}$每个 C 各补两个虚拟 C 原子
$\text{C}\!\equiv\!\text{N}$C 补两个虚拟 N，N 补两个虚拟 C

虚拟原子不再向外扩展，仅用于该层比较。

常考优先级排序：
  • $-\text{OH} > -\text{NH}_2 > -\text{COOH} > -\text{CHO} > -\text{CH}_2\text{OH} > -\text{C}_6\text{H}_5 > -\text{CH=CH}_2 > -\text{C(CH}_3)_3 > -\text{CH(CH}_3)_2 > -\text{CH}_2\text{CH}_3 > -\text{CH}_3 > -\text{H}$
  • 卤素：$-\text{I} > -\text{Br} > -\text{Cl} > -\text{F}$

十、Z/E 与 cis/trans 命名

Z/E 命名法

判断方法
  1. 对双键每一端的两个基团，用 CIP 规则分出优先级
  2. 两端的优先基团在双键同侧 → Z（zusammen，德语"一起"）
  3. 两端的优先基团在双键异侧 → E（entgegen，德语"相反"）

cis/trans 命名法（传统）

  • cis（顺式）：两个相同基团在双键同侧
  • trans（反式）：两个相同基团在双键异侧
Z/E 与 cis/trans 不一定对应！
  • Z/E 按 CIP 优先级判断（客观唯一）
  • cis/trans 按相同基团判断（有时不适用）
  • 当双键两端各有三个或四个不同基团时，只能用 Z/E
判断练习：

$\text{BrHC=CHI}$ 中，一端 Br>H，另一端 I>H → Br 和 I 同侧 = Z

按 cis/trans 无法命名（四个基团全不同），只能用 Z/E。

多个双键时：每个双键独立判断 Z/E，如 (2E,4Z)-庚二烯
不存在顺反异构的条件：双键某一端的两个基团相同时（如 $\text{CH}_2\text{=CHR}$），该分子无顺反异构体。

十一、R/S 构型标记

R/S 判定三步法
  1. 1按 CIP 次序规则排列手性碳上四个基团的优先级：$a > b > c > d$
  2. 2将最小基团 $d$（通常是 H）放到离眼睛最远处（纸面后方）
  3. 3从 $a \to b \to c$ 的方向：
    • 顺时针 = R（rectus，拉丁语"右"）
    • 逆时针 = S（sinister，拉丁语"左"）
d 不在后方时的技巧：
  • 方法一：做两次交换把 d 换到后方（偶数次交换不改变构型）
  • 方法二：直接按 $a \to b \to c$ 判断，若 d 在前方则结果取反（R↔S）
实例：
  • 乳酸 $\text{CH}_3\text{CH(OH)COOH}$：$\text{OH} > \text{COOH} > \text{CH}_3 > \text{H}$
  • $\text{H}$ 朝后看，$\text{OH} \to \text{COOH} \to \text{CH}_3$ 逆时针 → S 构型

Fischer 投影式判 R/S 技巧

速判法
  • H 在竖线上（上或下）→ 按 $a \to b \to c$ 直接判
  • H 在横线上（左或右）→ 按 $a \to b \to c$ 判完后 取反

十二、D/L 构型标记

适用范围：仅用于糖类和氨基酸的构型标记。

D/L 判断规则
  1. 写出 Fischer 投影式，碳链竖直放置
  2. 氧化态最高的碳（$\text{-CHO}$、$\text{-COOH}$）放在上端
  3. 看最下方手性碳上的 $\text{-OH}$（糖）或 $\text{-NH}_2$（氨基酸）：
    • 在右边 → D 型
    • 在左边 → L 型
三个"无关"：
  • D/L 与 R/S 无固定对应关系
  • D/L 与 (+)/(-) 旋光方向无直接关系（旋光方向需实验测定）
  • D 不一定右旋，L 不一定左旋
参考标准：以 D-(+)-甘油醛为参考。与它最下方手性碳构型相同者为 D 型，相反者为 L 型。

十三、Fischer 投影式操作规则

基本规则：横前竖后
  • 横线上的基团朝向观察者（伸出纸面）
  • 竖线上的基团远离观察者（伸入纸面）
操作结果说明
旋转 $180°$同一化合物偶数次交换
固定一个基团，转其余三个同一化合物等价于空间旋转
旋转 $90°$对映体奇数次交换
交换任意两个基团对映体一次交换 = 构型翻转
交换两次同一化合物两次翻转 = 恢复
翻面（上下翻转）不允许！破坏横前竖后规则
含多个手性碳的异构体数目：
  • $n$ 个不同手性碳 → 最多 $2^n$ 个旋光异构体
  • 有相同取代的手性碳 → 少于 $2^n$ 个（因存在 meso 体）
  • meso 化合物：有手性碳但整体有对称面 → 无旋光性
判手性的关键：
  • 有对称面 → 无手性（不论有几个手性碳）
  • 有对称中心 → 无手性
  • 有手性碳 $\neq$ 有手性（meso 化合物的反例）
  • 无手性碳也可有手性（丙二烯型、联苯轴手性）

十五、醇、酚、醚命名

醇的命名

系统命名法
  1. 选含 $\text{-OH}$ 的最长碳链为母体
  2. 从离 $\text{-OH}$ 最近的一端编号
  3. 命名为"X-某醇"
分类结构特征示例
伯醇 (1°)$\text{RCH}_2\text{OH}$1-丁醇
仲醇 (2°)$\text{R}_2\text{CHOH}$2-丁醇
叔醇 (3°)$\text{R}_3\text{COH}$2-甲基-2-丙醇
多元醇：标出所有 $\text{-OH}$ 位次，如 1,2-乙二醇、1,2,3-丙三醇（甘油）

酚的命名

  • 以"苯酚"为母体，取代基位次从 $\text{-OH}$ 所在碳（1位）开始编号
  • 例：2-甲基苯酚（邻甲酚）、2,4,6-三硝基苯酚（苦味酸）

醚的命名

  • 简单醚：两个烃基名 + "醚"，小基在前 → 甲乙醚、二乙醚
  • 复杂醚：以较大的烃基为母体，$\text{-OR}$ 为烷氧基取代基 → 2-甲氧基丁烷
  • 环氧化合物：以"环氧"为前缀 → 1,2-环氧丙烷

十六、卤代烃命名

系统命名法
  1. 选最长碳链为母体
  2. 卤素作为取代基（氟、氯、溴、碘），不是官能团
  3. 编号使取代基位次最小
分类结构特征示例
伯卤代烃 (1°)$\text{RCH}_2\text{X}$1-溴丁烷
仲卤代烃 (2°)$\text{R}_2\text{CHX}$2-溴丁烷
叔卤代烃 (3°)$\text{R}_3\text{CX}$2-溴-2-甲基丙烷
常见俗名： 氯仿 ($\text{CHCl}_3$)、碘仿 ($\text{CHI}_3$)、四氯化碳 ($\text{CCl}_4$)、氯乙烯 ($\text{CH}_2\text{=CHCl}$)

十七、官能团优先次序 & 后缀总表

当分子中有多个官能团时，优先级最高的官能团决定母体名称（后缀），其余作为取代基（前缀）。

综合命名实例：
  • $\text{HOCH}_2\text{CH}_2\text{COOH}$ → 3-羟基丙酸（$\text{-COOH}$ 优先级最高做后缀，$\text{-OH}$ 做前缀"羟基"）

  • $\text{BrCH}_2\text{CH=CHCOOH}$ → 4-溴-2-丁烯酸（$\text{-COOH}$ 做后缀，Br 做前缀，含双键写为"烯酸"）
