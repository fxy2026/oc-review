第 4 页

第三章 烯烃

1. 同分异构

同分异构类型
  • ① 构造异构：碳链异构 / 官能团位置异构
  • ② 顺反异构（几何异构）：双键不能旋转 → 顺式/反式

2. 命名

烯烃命名规则
  • ① 选含 $\text{C}=\text{C}$ 的最长碳链为主链
  • ② 从靠近 $\text{C}=\text{C}$ 的一端开始编号：$Z$-/$E$- 标注（重要！）
  • ③ 烯烃 → $-\text{CH}=\text{CH}_2$（乙烯基）；烯丙基 → $-\text{CH}=\text{CH-CH}_3$
$Z$/$E$ 与 顺/反 命名法

$Z$/$E$ 命名法：按 CIP 次序规则（Cahn-Ingold-Prelog）

  • 比较双键两个碳上各连接基团的优先顺序
  • $E$（entgegen，相反）：两个较优先基团分居双键两侧
  • $Z$（zusammen，相同）：两个较优先基团在双键同侧

CIP 次序规则要点：

  1. 原子序数大者优先
  2. 同位素：质量数大者优先
  3. 逐级比较至区分为止
  4. 重键展开为等价单键（如 $\text{C}=\text{O}$ 展开为两个 C-O 单键）
注意：$Z$/$E$ 与顺/反不一定一致！
$Z$ 不一定是 "顺式"，$E$ 不一定是 "反式"，取决于取代基的优先次序

3. 化学性质

烯烃的特征反应

烯烃 $\pi$ 键电子易受亲电试剂进攻 → 亲电加成反应为核心

烯烃的稳定性
  • 可通过氢化热（加 $\text{H}_2$ 释放的热量）判断稳定性
  • 氢化热越低 → 烯烃越稳定
  • 取代基越多 → 烯烃越稳定：$\text{R}_2\text{C}=\text{CR}_2 > \text{R}_2\text{C}=\text{CHR} > \text{RCH}=\text{CHR} > \text{RCH}=\text{CH}_2$
  • 超共轭效应：$\alpha$-H 的 C-H 键与 $\pi$ 键的 $\sigma$-$\pi$ 超共轭，$\alpha$-H 越多越稳定
  • 反式（$E$）比顺式（$Z$）更稳定（空间位阻小）
HX 酸催化加成与 Markovnikov 规则

烯烃催化加 HX 时H 加到含 H 较多的碳上，$\text{X}$ 加到含 H 较少的碳上（Markovnikov 规则）

本质：反应经过较稳定的碳正离子中间体（$3° > 2° > 1°$）

碳正离子稳定性：$3° > 2° > 1° > \text{CH}_3^+$，这决定了加成的区域选择性和可能的重排

$\text{H}_2\text{SO}_4$ / $\text{H}_3\text{PO}_4$ 加成

间接水合法制醇：先加 $\text{H}_2\text{SO}_4$，再水解

卤化反应

(1) 加 $\text{Cl}_2$ / $\text{Br}_2$（反应活性区别）

$$\text{C}=\text{C} + \text{Cl}_2/\text{Br}_2 \xrightarrow{} \text{二卤代物}$$

反应机理：

  1. $\pi$ 电子进攻 $\text{Br}_2$ → 形成溴鎓离子（环状三元中间体）
  2. $\text{Br}^-$ 从背面进攻 → 反式加成（anti addition）
重要结论：卤素加成是经过卤鎓离子中间体的反式加成。
  • 溴鎓离子是桥环正离子
  • $\text{Br}^-$ 从背面（anti）进攻
  • 产物为 anti 加成产物（反式邻二卤代物）
亲电加成的机理总结

① 亲电试剂进攻 $\pi$ 键 → 碳正离子（或环状鎓离子）

② 亲核试剂与碳正离子结合 → 产物

机理决定产物的区域选择性（Markovnikov）和立体选择性（反式加成）

卤化氢（$\text{HX}$）加成

活性：$\text{HI} > \text{HBr} > \text{HCl} > \text{HF}$

遵循 Markovnikov 规则（经碳正离子中间体，遵循稳定性序列）

$\text{Cl}^-$ 不生成氯鎓离子 → 无立体选择性；一般生成顺反混合物

地位和意义：碳正离子稳定性的比较是有机化学的核心考点之一，贯穿亲电加成、消除反应、重排反应等多个章节。
第 5 页

亲电加成的区域选择性（续）

Markovnikov 规则（马氏规则）
  • 不对称烯烃加 HX 时，H 加在含 H 较多的碳上，X 加在含 H 较少的碳上
  • 本质：经过较稳定的碳正离子中间体
  • 碳正离子稳定性：$3° > 2° > 1° > \text{CH}_3^+$
  • 可用超共轭效应和诱导效应解释
碳正离子重排

当初始生成的碳正离子不够稳定时，可通过1,2-迁移（H 迁移或 $\text{CH}_3$ 迁移）重排为更稳定的碳正离子：

  • 1,2-氢迁移：$2°\text{C}^+ \xrightarrow{} 3°\text{C}^+$
  • 1,2-甲基迁移：同理
  • 重排发生的驱动力：碳正离子趋向更稳定的结构
碳正离子重排的判断要点：
  • 氢酸和硫酸的加成反应，常伴随碳正离子重排
  • 氯化氢与烯烃的加成反应：常有重排产物
  • 注意：碳正离子不仅参与加成反应，也参与消除反应和取代反应
例题：判断烯烃亲电加成的产物

例题：下列化合物中的烯烃能否与下列试剂进行加成反应？如不反应写出其产物；如果有过渡态的复杂化合物产物写出：

  1. $\text{C}=\text{C}-\text{C}-\text{C}$ 加 $\text{HBr}$ → $\text{H}_3\text{C}-\overset{\text{Br}}{|}\text{C}-\text{C}-\text{C}$（马氏产物）
  2. $\text{CH}_2=\text{C}(\text{CH}_3)-\text{C}(\text{CH}_3)_2$ 加 HBr → 可能发生重排
  3. $\text{CH}_3\text{CH}=\text{CHCH}_3 + \text{Cl}_2$ → 反式加成产物

(8) 烯烃的自由基加成（反Markovnikov加成）

过氧化物效应（反马氏加成）

在过氧化物（ROOR）存在下：

$$\text{ROOR} \xrightarrow{\Delta} 2\text{RO}\cdot$$ $$\text{RO}\cdot + \text{HBr} \to \text{ROH} + \text{Br}\cdot$$

自由基加成机理：

$$\text{Br}\cdot + \text{CH}_2=\text{CHCH}_2\text{CH}_3 \to \text{CH}_3\text{CHCH}_2\text{CH}_3 \xrightarrow{\text{HBr}} \text{CH}_3\text{CH}(\text{Br})\text{CH}_2\text{CH}_3$$

产物为反 Markovnikov加成产物

过氧化物效应的限制：
  • 仅限于 $\text{HBr}$；$\text{HCl}$ 和 $\text{HI}$ 不发生（$\text{HCl}$ 难以夺取 Br·，$\text{HI}$ 中 I· 太稳定不反应）

(10) 硼氢化-氧化（重要反应）

硼氢化-氧化反应

这是一个协同反应（特殊反应）：不产生碳正离子/自由基中间体

烯烃的硼氢化 —— 氧化反应（反马氏加成、顺式加成）：

反应条件与步骤：

  1. 硼氢化：$\text{烯烃} + (\text{BH}_3)_2 \xrightarrow{\text{THF}} \text{烷基硼}$
  2. 氧化：$\text{烷基硼} + \text{H}_2\text{O}_2 / \text{NaOH} \to \text{醇}$

特点：

  • 反马氏方向：B 加在含 H 多的碳上（$\text{OH}$ 最终在含 H 多的碳上）
  • 顺式加成（syn addition）：B 和 H 从同侧加成
  • 本反应为协同反应，无碳正离子中间体，不会重排

例：

$$\text{CH}_3\text{CH}=\text{CH}_2 \xrightarrow{1.\,(\text{BH}_3)_2} \xrightarrow{2.\,\text{H}_2\text{O}_2,\,\text{NaOH}} \text{CH}_3\text{CH}_2\text{CH}_2\text{OH}$$

（产物为反马氏的伯醇）

硼氢化-氧化 vs 酸催化水合：
方法方向立体重排产物
酸催化水合马氏无选择性可能重排仲/叔醇
硼氢化-氧化反马氏顺式加成不重排伯醇

环氧化反应

烯烃的环氧化（过氧酸氧化）

使用 mCPBA（间氯过氧苯甲酸）或其他过氧酸：

$$\text{烯烃} + \text{RCOOOH} \xrightarrow{} \text{环氧化物} + \text{RCOOH}$$

特点：顺式加成（协同机理），保持双键两侧取代基的相对构型

第 6 页

对比氧化的反应机理

四氧化锇（$\text{OsO}_4$）氧化

四氧化锇 $\text{OsO}_4$ 与烯烃进行顺式双羟基化反应：

$$\text{烯烃} \xrightarrow{\text{OsO}_4} \text{锇酸酯中间体} \xrightarrow{\text{还原（如 }\text{NaHSO}_3\text{）}} \text{邻二醇（顺式）}$$

特点：

  • 协同机理：$\text{OsO}_4$ 从烯烃同一面进攻 → 顺式邻二醇
  • 产物保持 syn 立体化学
高锰酸钾（$\text{KMnO}_4$）氧化
  • 冷、稀、碱性条件（Baeyer 试验）：$\text{烯烃} \xrightarrow{\text{KMnO}_4(\text{冷，稀})} \text{邻二醇}$（顺式双羟基化，类似 $\text{OsO}_4$）
  • 热、浓、酸性条件：氧化断键 → 产生酮 / 羧酸
对比记忆：
  1. 双键上的顺式加成氧化试剂不会发生碳骨架断裂反应
  2. 双键上的剧烈氧化（热 $\text{KMnO}_4$）则电子彻底代谢，对应有断键作用
烯烃在 $\text{KMnO}_4$ 下的断裂规则

被氧化断键后的产物判断：

双键碳类型氧化产物
$\text{R}_2\text{C}=$ （两个 R 取代）酮 $\text{R}_2\text{C}=\text{O}$
$\text{RCH}=$（一个 R 取代）羧酸 $\text{RCOOH}$
$\text{CH}_2=$（无 R 取代）$\text{CO}_2 + \text{H}_2\text{O}$

(2) $\alpha$-H 卤代反应（自由基反应）

烯丙位 $\alpha$-H 的卤代（NBS 卤代）

烯丙型自由基：$\text{p}$-$\pi$ 共轭稳定 → 比一般自由基更稳定

$$\text{H}_2\text{C}=\text{CH}-\text{CH}_3 \xrightarrow{} \text{烯丙基自由基}$$

使用 NBS（N-溴代丁二酰亚胺）进行 $\alpha$-H 溴代：

$$\text{CH}_2=\text{CH-CH}_3 \xrightarrow{\text{NBS, }h\nu} \text{CH}_2=\text{CH-CH}_2\text{Br}$$
高温、气相条件下的取代反应
  • 常温、室温：$\text{Cl}_2$ 发生加成反应（离子型，双键加成）：$\text{H}_3\text{C}-\text{CH}=\text{CH}_2 + \text{Cl}_2 \to$ 二氯代物
  • 高温（500°C）：$\text{Cl}_2$ 发生取代反应（自由基型，$\alpha$-H 取代）：$\text{H}_3\text{C}-\text{CH}=\text{CH}_2 + \text{Cl}_2 \xrightarrow{500°\text{C}} \text{ClCH}_2\text{-CH}=\text{CH}_2 + \text{HCl}$
反应条件的决定性影响：
  • 低温、溶液中 → 离子型加成（双键参与）
  • 高温、气相 → 自由基取代（$\alpha$-H 参与）

(六) 双键位置的测定

臭氧化反应

臭氧化-还原水解用于确定烯烃双键的位置：

$$\text{烯烃} \xrightarrow{\text{O}_3} \text{臭氧化物} \xrightarrow{\text{Zn}/\text{H}_2\text{O}(\text{或 Me}_2\text{S})} \text{醛/酮}$$

机理：

  1. $\text{O}_3$ 进行 [3+2] 环加成 → 摩尔臭氧化物（molozonide）
  2. 重排 → 臭氧化物（ozonide）
  3. 还原水解 → 两个羰基化合物（醛和/或酮）

产物判断规则：双键断裂处各变为 $\text{C}=\text{O}$

双键碳类型臭氧化产物
$\text{R}_2\text{C}=$酮 $\text{R}_2\text{C}=\text{O}$
$\text{RCH}=$醛 $\text{RCHO}$
$\text{CH}_2=$甲醛 $\text{HCHO}$
应用：反推双键位置

给出臭氧化产物 → 反推原来烯烃的结构：

将两个羰基化合物的 $\text{C}=\text{O}$ 去掉氧，拼成 $\text{C}=\text{C}$ → 即可得原来的烯烃

臭氧化与 $\text{KMnO}_4$ 氧化的区别：
  • 臭氧化（$\text{O}_3$ + $\text{Zn}$/$\text{H}_2\text{O}$）：$\text{RCH}=$ 得到醛 $\text{RCHO}$（不被继续氧化）
  • $\text{KMnO}_4$（热、浓）：$\text{RCH}=$ 得到羧酸 $\text{RCOOH}$（醛被进一步氧化）
  • $\text{R}_2\text{C}=$ 两者均得酮
  • $\text{CH}_2=$ 分别得到 $\text{HCHO}$ 或 $\text{CO}_2 + \text{H}_2\text{O}$

第 7 页

(五) 聚合反应

定义

由低相对分子质量的化合物通过自相加或互相加成反应转变为高相对分子质量化合物的反应。

自由基聚合反应

$n\text{CH}_2\text{=CH}_2 \xrightarrow[\text{100-300}^\circ\text{C, 高压}]{\text{自由基引发剂}} \text{[-CH}_2\text{-CH}_2\text{-]}_n$

用上法可制得高压聚乙烯（密度较低），支链较多、支链度较高、结晶度低。

某些烯烃也可用自由基聚合在较低压下进行，得到低压聚乙烯（高密度、支链少、结晶度高、线型分子）。

高压聚乙烯 vs 低压聚乙烯：高压法得到的产品分子量约 20,000-50,000；低压法（Ziegler-Natta）产品分子量更高。

Ziegler-Natta 催化聚合

催化剂为 $\text{TiCl}_4$ / $\text{Al(C}_2\text{H}_5\text{)}_3$

$n\text{CH}_2\text{=CH}_2 \xrightarrow{\text{TiCl}_4\text{-Al(C}_2\text{H}_5\text{)}_3,\ 50\ ^\circ\text{C},\ 2.0\text{ MPa}} \text{[-CH}_2\text{-CH}_2\text{-]}_n$

Ziegler-Natta 催化剂能实现定向聚合（等规聚合），产物结构规整、结晶度高。
