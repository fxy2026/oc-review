第3章 烯烃 梳理

第 4 页

第三章 烯烃

1. 同分异构

同分异构类型
  • ① 构造异构：碳链异构 / 官能团位置异构
  • ② 顺反异构（几何异构）：双键不能旋转 → 顺式/反式

2. 命名

烯烃命名规则
  • ① 选含 $\text{C}=\text{C}$ 的最长碳链为主链
  • ② 从靠近 $\text{C}=\text{C}$ 的一端开始编号：$Z$-/$E$- 标注（重要！）
  • ③ 常见基团名称：
    • 乙烯基（vinyl）：$-\text{CH}=\text{CH}_2$
    • 烯丙基（allyl）：$-\text{CH}_2\text{-CH}=\text{CH}_2$
    • 亚乙烯基：$-\text{CH}=\text{CH}-$
$Z$/$E$ 与 顺/反 命名法

$Z$/$E$ 命名法：按 CIP 次序规则（Cahn-Ingold-Prelog）

  • 比较双键两个碳上各连接基团的优先顺序
  • $E$（entgegen，相反）：两个较优先基团分居双键两侧
  • $Z$（zusammen，相同）：两个较优先基团在双键同侧

CIP 次序规则要点：

  1. 原子序数大者优先
  2. 同位素：质量数大者优先
  3. 逐级比较至区分为止
  4. 重键展开为等价单键（如 $\text{C}=\text{O}$ 展开为两个 C-O 单键）
注意：$Z$/$E$ 与顺/反不一定一致！
$Z$ 不一定是 "顺式"，$E$ 不一定是 "反式"，取决于取代基的优先次序。

3. 物理性质

物理性质概要
  • 沸点：随碳数增加而升高；支链增多 → 沸点降低
  • 密度：比水小
  • 溶解度：不溶于水，溶于有机溶剂
  • 顺式异构体的沸点通常略高于反式（偶极矩较大）

4. 化学性质概览

烯烃的特征反应

烯烃 $\pi$ 键电子云暴露在外，易受亲电试剂进攻 → 亲电加成反应为核心反应类型

(一) 催化加氢

催化氢化

催化剂：$\text{Pt}$、$\text{Pd}$、$\text{Ni}$（异相催化）；Wilkinson 催化剂（均相催化）

$$\text{C}=\text{C} + \text{H}_2 \xrightarrow{\text{Pt (或 Pd, Ni)}} \text{C}-\text{C}$$

立体化学：顺式加成（syn addition）—— $\text{H}_2$ 从同一面加到双键上

烯烃的稳定性与氢化热
  • 可通过氢化热（$\Delta H_{\text{hydrog}}$，加 $\text{H}_2$ 释放的热量）判断稳定性
  • 氢化热越低 → 烯烃越稳定
  • 取代基越多 → 烯烃越稳定：
$$\text{R}_2\text{C}=\text{CR}_2 > \text{R}_2\text{C}=\text{CHR} > \text{RCH}=\text{CHR} > \text{RCH}=\text{CH}_2 > \text{CH}_2=\text{CH}_2$$
  • 超共轭效应：$\alpha$-$\text{C-H}$ 键 $\sigma$ 轨道与 $\pi$ 键的 $\sigma$-$\pi$ 超共轭，$\alpha$-H 越多越稳定
  • 反式（$E$）比顺式（$Z$）更稳定（空间位阻小）
  • 氢化热顺序（同分子式）：$\text{H}_1 > \text{H}_2 > \text{H}_3$，氢化热越高越不稳定

(二) HX 加成与 Markovnikov 规则

HX 酸催化加成

不对称烯烃加 HX 时H 加到含 H 较多的碳上，$\text{X}$ 加到含 H 较少的碳上 — Markovnikov 规则

本质：反应经过较稳定的碳正离子中间体（$3° > 2° > 1°$）

碳正离子稳定性：$3° > 2° > 1° > \text{CH}_3^+$，决定加成的区域选择性和可能的重排

HX 反应活性：$\text{HI} > \text{HBr} > \text{HCl} > \text{HF}$

地位和意义：碳正离子稳定性的比较是有机化学的核心考点之一，贯穿亲电加成、消除反应、重排反应等多个章节。
$\text{C}^+$ 的 $p$ 空轨道：稳定性可通过超共轭和诱导效应解释。

(三) $\text{H}_2\text{SO}_4$ / $\text{H}_3\text{PO}_4$ 加成

间接水合法制醇

先加 $\text{H}_2\text{SO}_4$，生成硫酸氢烷基酯，再水解得到醇（Markovnikov 方向）：

$$\text{烯烃} \xrightarrow{\text{H}_2\text{SO}_4} \text{硫酸氢烷基酯} \xrightarrow{\text{H}_2\text{O}} \text{醇} + \text{H}_2\text{SO}_4$$

遵循 Markovnikov 规则 → 产物为仲/叔醇

(四) 卤化反应（加 $\text{Cl}_2$ / $\text{Br}_2$）

卤素加成机理（反式加成，经卤鎓离子）
$$\text{C}=\text{C} + \text{Br}_2 \to \text{邻二溴代物}$$

反应机理：

  1. $\pi$ 电子进攻 $\text{Br}_2$（亲电进攻） → 形成溴鎓离子（bromonium ion，环状三元中间体）+ $\text{Br}^-$ 离去
  2. $\text{Br}^-$ 从溴鎓离子的背面进攻（$S_N2$ 类型） → 反式加成（anti addition）
重要结论：卤素加成是经过卤鎓离子（桥环正离子）中间体的反式加成。
  • 溴鎓离子是三元环正离子
  • $\text{Br}^-$ 从背面（anti）进攻 → 产物为反式加成产物
  • $\text{Cl}^-$ 不生成稳定的氯鎓离子 → 无明显立体选择性

(五) 卤化氢（$\text{HX}$）加成的机理

亲电加成的分步机理

① 亲电试剂（$\text{H}^+$）进攻 $\pi$ 键 → 碳正离子中间体（决速步）

② 亲核试剂（$\text{X}^-$）与碳正离子结合 → 产物

机理决定：

  • 区域选择性：Markovnikov 规则（经较稳定碳正离子）
  • 立体选择性：HX 加成一般无立体选择性（平面碳正离子，两面均可被进攻）
碳正离子重排

当初始生成的碳正离子不够稳定时，可通过1,2-迁移重排为更稳定的碳正离子：

  • 1,2-氢迁移：$2°\text{C}^+ \to 3°\text{C}^+$
  • 1,2-甲基迁移：$2°\text{C}^+ \to 3°\text{C}^+$
  • 重排的驱动力：碳正离子趋向更稳定的结构
碳正离子重排的判断要点：
  • HX 加成反应中，当邻碳有可能形成更稳定碳正离子时，常伴随重排产物
  • 重排产物可能是主要产物——一定要检查邻碳是否能形成更稳定的 $\text{C}^+$
  • 碳正离子不仅参与加成反应，也参与消除反应和取代反应（$S_N1$）
第 5 页

亲电加成的进一步讨论

亲电试剂的鉴别 — $\text{Ag}^+$ 检验

检验卤素离子（$\text{Cl}^-$、$\text{Br}^-$）时，需注意离子型反应产生的卤素离子才能被 $\text{AgNO}_3$ 检出。

亲电加成过程中，在溶液里如果有 $\text{Ag}^+$ 存在，可以帮助拉动 $\text{Br}_2$ 的异裂，加速反应。

碱性与亲核性的关系
  • 氮酸（$\text{HX}$）与烯烃的加成反应：常伴随碳正离子中间体和可能的重排
  • 碱性强的试剂一般亲核性也强，但需注意碱性和亲核性是不同概念
例题：判断烯烃亲电加成的产物

例题：下列化合物中的烯烃能否与下列试剂进行加成反应？如果能反应，写出主要产物：

  1. $\text{CH}_3\text{CH}_2\text{CH}=\text{CH}_2 + \text{HBr}$ → $\text{CH}_3\text{CH}_2\text{CHBrCH}_3$（马氏产物，$\text{Br}$ 加在 $2°$ 碳上）
  2. $\text{CH}_2=\text{C}(\text{CH}_3)\text{C}(\text{CH}_3)_3 + \text{HBr}$ → 可能发生碳正离子重排
  3. $\text{CH}_3\text{CH}=\text{CHCH}_3 + \text{Cl}_2$ → 反式加成产物（邻二氯代物）
  4. $\text{CH}_3\text{CH}_2\text{CH}=\text{CH(CH}_3\text{)} + \text{Cl}_2$ → 反式加成产物

(六) 烯烃的自由基加成（反 Markovnikov 加成）

过氧化物效应（反马氏加成）— 自由基机理

在过氧化物（$\text{ROOR}$）存在下，$\text{HBr}$ 对烯烃发生反马氏加成：

引发：

$$\text{ROOR} \xrightarrow{\Delta} 2\text{RO}\cdot$$ $$\text{RO}\cdot + \text{HBr} \to \text{ROH} + \text{Br}\cdot$$

链增长（自由基加成）：

$$\text{Br}\cdot + \text{CH}_2\text{=CHCH}_2\text{CH}_3 \to \text{BrCH}_2\text{-}\dot{\text{C}}\text{HCH}_2\text{CH}_3$$ $$\text{BrCH}_2\dot{\text{C}}\text{HCH}_2\text{CH}_3 + \text{HBr} \to \text{BrCH}_2\text{CH}_2\text{CH}_2\text{CH}_3 + \text{Br}\cdot$$

产物为反 Markovnikov加成产物（$\text{Br}$ 加在含 H 多的碳上）

本质：$\text{Br}\cdot$ 加成时生成较稳定的自由基（$2° > 1°$）

过氧化物效应的限制：
  • 仅限于 $\text{HBr}$
  • $\text{HCl}$：$\text{Cl}\cdot$ 与烯烃反应放热太多，选择性差，且链传递中夺 H 困难
  • $\text{HI}$：$\text{I}\cdot$ 太稳定，加成到双键上的反应吸热，不能维持链反应
  • 只有 $\text{HBr}$ 的两步链传递都是放热的，链反应得以维持

(七) 硼氢化-氧化反应（重要反应）

硼氢化-氧化反应（Hydroboration-Oxidation）

这是一个协同反应（特殊反应）：不产生碳正离子/自由基中间体

反应步骤：

  1. 硼氢化：$\text{烯烃} + (\text{BH}_3)_2 \xrightarrow{\text{THF}} \text{烷基硼}$（$\text{BH}_3$ 以 THF 配合物形式使用）
  2. 氧化：$\text{烷基硼} \xrightarrow{\text{H}_2\text{O}_2,\;\text{NaOH}} \text{醇}$

特点：

  • 反马氏方向：$\text{B}$ 加在含 H 多的碳上（位阻小的碳），$\text{OH}$ 最终也在含 H 多的碳上
  • 顺式加成（syn addition）：$\text{B}$ 和 $\text{H}$ 从同一面加成
  • 协同机理（四元环过渡态），无碳正离子中间体，不会发生重排

例（环己烯 → 环己醇）：

$$\bigcirc\!\!= \xrightarrow{1.\,(\text{BH}_3)_2,\;\text{THF}} \xrightarrow{2.\,\text{H}_2\text{O}_2,\;\text{NaOH}} \text{环己醇（顺式 syn 加成产物）}$$

例（丙烯 → 正丙醇）：

$$\text{CH}_3\text{CH}=\text{CH}_2 \xrightarrow{1.\,(\text{BH}_3)_2} \xrightarrow{2.\,\text{H}_2\text{O}_2,\,\text{NaOH}} \text{CH}_3\text{CH}_2\text{CH}_2\text{OH}$$

（产物为反马氏的伯醇）

硼氢化-氧化 vs 酸催化水合：
方法方向立体重排产物
酸催化水合马氏无选择性可能重排仲/叔醇
硼氢化-氧化反马氏顺式加成不重排伯醇

(八) 环氧化反应

烯烃的环氧化（过氧酸氧化）

使用 mCPBA（间氯过氧苯甲酸）或其他过氧酸（$\text{RCO}_3\text{H}$）：

$$\text{烯烃} + \text{RCO}_3\text{H} \xrightarrow{} \text{环氧化物} + \text{RCOOH}$$

特点：顺式加成（协同机理），保持双键两侧取代基的相对构型

本反应为协同反应（蝴蝶形过渡态），不经过碳正离子，不会重排

第 6 页

对比氧化的反应机理（续）

四氧化锇（$\text{OsO}_4$）氧化 —— 顺式双羟基化

$\text{OsO}_4$ 与烯烃进行顺式双羟基化反应：

$$\text{烯烃} \xrightarrow{\text{OsO}_4} \text{锇酸酯（环状中间体）} \xrightarrow{\text{还原（}\text{NaHSO}_3\text{ 或 } \text{H}_2\text{S}\text{）}} \text{邻二醇（顺式）}$$

特点：

  • 协同机理：$\text{OsO}_4$ 从烯烃同一面进攻（[3+2] 环加成）→ 顺式邻二醇
  • 产物保持 syn 立体化学
  • $\text{OsO}_4$ 有毒且昂贵，可用催化量的 $\text{OsO}_4$ + 共氧化剂（如 $\text{NMO}$）代替
对比记忆：
  1. 双键上的顺式加成氧化试剂（$\text{OsO}_4$、冷稀 $\text{KMnO}_4$）不会发生碳骨架断裂
  2. 双键上的剧烈氧化（热浓酸性 $\text{KMnO}_4$）则电子被彻底夺取，对应有断键作用

(九) $\alpha$-H 卤代反应（自由基反应）

烯丙位 $\alpha$-H 的卤代（NBS 卤代）

烯丙型自由基：$\text{p}$-$\pi$ 共轭稳定 → 比一般自由基更稳定（中间体稳定）

$$\text{H}_2\text{C}=\text{CH}-\text{CH}_3 \xrightarrow{\text{NBS},\;h\nu\text{ 或 }\Delta} \text{H}_2\text{C}=\text{CH}-\text{CH}_2\text{Br}$$

使用 NBS（N-溴代丁二酰亚胺）进行选择性 $\alpha$-H 溴代

高温 vs 常温：加成与取代的竞争
  • 常温、溶液中：$\text{Cl}_2$ 发生加成反应（离子型，双键参与） $$\text{CH}_3\text{CH}=\text{CH}_2 + \text{Cl}_2 \xrightarrow{\text{常温}} \text{CH}_3\text{CHClCH}_2\text{Cl}$$
  • 高温（$500\,°\text{C}$）、气相：$\text{Cl}_2$ 发生取代反应（自由基型，$\alpha$-H 取代） $$\text{CH}_3\text{CH}=\text{CH}_2 + \text{Cl}_2 \xrightarrow{500\,°\text{C}} \text{ClCH}_2\text{CH}=\text{CH}_2 + \text{HCl}$$
反应条件的决定性影响：
  • 低温、溶液中 → 离子型加成（双键参与）
  • 高温、气相 → 自由基取代（$\alpha$-H 参与）

(十) 双数化物（副产物）

$\text{KMnO}_4$ 氧化
  • 冷、稀、碱性条件（Baeyer 试验）： $$\text{烯烃} \xrightarrow{\text{KMnO}_4\text{（冷、稀）}} \text{邻二醇（顺式）}$$ 紫色 $\text{KMnO}_4$ 褪色 → 可用于鉴别不饱和键（Baeyer 试剂）
  • 热、浓、酸性条件：氧化断键 → 产生酮 / 羧酸 / $\text{CO}_2$
$\text{KMnO}_4$ 氧化断键规则
双键碳类型氧化产物
$\text{R}_2\text{C}=$（两个 R 取代）酮 $\text{R}_2\text{C}=\text{O}$
$\text{RCH}=$（一个 R 取代）羧酸 $\text{RCOOH}$
$\text{CH}_2=$（无 R 取代）$\text{CO}_2 + \text{H}_2\text{O}$
低碳酸盐氧化（环氧化 → 邻二醇）

烯烃先环氧化，再在酸性或碱性条件下开环水解，可得到反式邻二醇（anti addition）：

$$\text{烯烃} \xrightarrow{\text{RCO}_3\text{H}} \text{环氧化物} \xrightarrow{\text{H}_3\text{O}^+\text{ 或 OH}^-} \text{反式邻二醇}$$

对比 $\text{OsO}_4$：$\text{OsO}_4$ 给顺式邻二醇，环氧化开环给反式邻二醇

(十一) 臭氧化反应

臭氧化-还原水解

用于确定烯烃双键的位置：

$$\text{烯烃} \xrightarrow{\text{O}_3} \text{臭氧化物} \xrightarrow{\text{Zn}/\text{H}_2\text{O}\;\text{或}\;\text{Me}_2\text{S}} \text{醛/酮}$$

机理：

  1. $\text{O}_3$ 进行 [3+2] 环加成 → 初级臭氧化物（molozonide）
  2. 重排 → 臭氧化物（ozonide）
  3. 还原水解（$\text{Zn}/\text{H}_2\text{O}$ 或 $\text{(CH}_3\text{)}_2\text{S}$） → 两个羰基化合物（醛和/或酮）

产物判断规则：双键断裂处各变为 $\text{C}=\text{O}$

双键碳类型臭氧化产物
$\text{R}_2\text{C}=$酮 $\text{R}_2\text{C}=\text{O}$
$\text{RCH}=$醛 $\text{RCHO}$
$\text{CH}_2=$甲醛 $\text{HCHO}$
应用：反推双键位置

给出臭氧化产物 → 反推原来烯烃的结构：

将两个羰基化合物的 $\text{C}=\text{O}$ 去掉氧，拼成 $\text{C}=\text{C}$ → 即可得原来的烯烃

臭氧化与 $\text{KMnO}_4$ 氧化的区别：
  • 臭氧化（$\text{O}_3$ + $\text{Zn}$/$\text{H}_2\text{O}$）：$\text{RCH}=$ 得到醛 $\text{RCHO}$（不被继续氧化）
  • $\text{KMnO}_4$（热、浓）：$\text{RCH}=$ 得到羧酸 $\text{RCOOH}$（醛被进一步氧化）
  • $\text{R}_2\text{C}=$ 两者均得酮
  • $\text{CH}_2=$ 分别得到 $\text{HCHO}$ 或 $\text{CO}_2 + \text{H}_2\text{O}$
氧化裂解对比总结表
试剂/条件$\text{R}_2\text{C}=$$\text{RCH}=$$\text{CH}_2=$
$\text{O}_3$ + $\text{Zn}$/$\text{H}_2\text{O}$酮醛$\text{HCHO}$
$\text{KMnO}_4$（热、浓、酸性）酮$\text{RCOOH}$$\text{CO}_2 + \text{H}_2\text{O}$
$\text{OsO}_4$ 或 $\text{KMnO}_4$（冷、稀）邻二醇（不断键）
第 7 页

(十二) 聚合反应

定义

由低相对分子质量的化合物通过自相加成或互相加成反应转变为高相对分子质量化合物的反应。

自由基聚合反应

高压聚乙烯
$$n\text{CH}_2\text{=CH}_2 \xrightarrow[\text{100-300}\,°\text{C},\;\text{高压}]{\text{自由基引发剂}} \text{[-CH}_2\text{-CH}_2\text{-]}_n$$

高压法（自由基引发）得到的产品：支链较多、支链度较高、结晶度低，分子量约 20,000-50,000。

低压聚乙烯（Ziegler-Natta 催化聚合）

催化剂：$\text{TiCl}_4$ / $\text{Al(C}_2\text{H}_5\text{)}_3$（Ziegler-Natta 催化剂）

$$n\text{CH}_2\text{=CH}_2 \xrightarrow[\sim50\,°\text{C},\;2.0\;\text{MPa}]{\text{TiCl}_4\text{-Al(C}_2\text{H}_5\text{)}_3} \text{[-CH}_2\text{-CH}_2\text{-]}_n$$

低压法产品：高密度、支链少、结晶度高、线型分子，分子量更高。

高压聚乙烯 vs 低压聚乙烯：
类型引发方式条件产物特征
高压聚乙烯自由基引发高温高压支链多、密度低、结晶度低
低压聚乙烯Ziegler-Natta 催化低温低压支链少、密度高、结晶度高

(十三) 烯烃的来源和制备

实验室和工业制备方法

1. 乙醇脱水

$$\text{CH}_3\text{CH}_2\text{OH} \xrightarrow{\text{H}_2\text{SO}_4,\;170\,°\text{C}} \text{CH}_2\text{=CH}_2 + \text{H}_2\text{O}$$

2. 卤代烃脱卤化氢

$$\text{CH}_3\text{CH}_2\text{CH}_2\text{Br} \xrightarrow{\text{KOH/EtOH}} \text{CH}_3\text{CH=CH}_2 + \text{KBr} + \text{H}_2\text{O}$$

3. 邻二卤代物脱卤素

$$\text{CH}_3\text{CHBrCHBrCH}_3 \xrightarrow{\text{Zn},\;\Delta} \text{CH}_3\text{CH=CHCH}_3 + \text{ZnBr}_2$$
应用：乙烯可用作植物催熟剂，但使用不当会造成危害。
芥子气 $[\text{(ClCH}_2\text{CH}_2\text{)}_2\text{S}]$ 是含氯代物的典型有毒化合物。
